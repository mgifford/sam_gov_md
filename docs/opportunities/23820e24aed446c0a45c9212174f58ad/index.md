# Repair Basketball Court B1700

- Agency: DEPT OF DEFENSE
- Type: Presolicitation
- Posted: 2026-03-18 14:49:12.498-04
- Solicitation Number: FA660626R0007

## Summary

Project YTPM 26-5001, Repair Basketball Court B1700 (Notice ID FA660626R0007) at Westover Air Reserve Base, Chicopee, MA. This acquisition is anticipated to be issued as a Request for Proposal (RFP), Firm-Fixed-Price contract. Performance period will be ninety (90) Calendar Days from issuance of the Notice to Proceed. This project is being solicited as a 100% Total Small Business Set-Aside, subject to the North American Industry Classification System (NAICS) Code 238330 (Flooring Contractors), with a size standard of $19.0 million. The solicitation will be issued as a Request for Proposal (RFP), expected early April 2026 with an estimated thirty (30) Calendar Day response period. This project consists of removing and storing bleachers, cove base, and other items on the floor. Strip the entire basketball court floor. Sand surface smooth and uniform with no grooving. Repair any defects, apply graphics and line markings, and seal surface with high durability non-slip basketball court polyurethane coating. Clean and reinstall court cove base molding and bleachers at Buidling 1700. The Contractor shall provide all labor, materials, and equipment to install, construct, and complete the work planned in accordance with the Statement of Work (SOW) and associated documents. There will be an organized Site Visit planned for this solicitation; the correct date, time, and location will be included in the solicitation. The estimated magnitude of construction for this project is between $35,000 and $150,000. There is no Government Furnished Property supplied for this project. The solicitation and all supporting documentation will be posted at this site for review and download. All contractors and subcontractors interested in this project must register at this site https://SAM.gov. It is the offeror(s) responsibility to check the website periodically for any amendments to the solicitation. There will be no other form of distribution for this solicitation other than downloading from this site.
