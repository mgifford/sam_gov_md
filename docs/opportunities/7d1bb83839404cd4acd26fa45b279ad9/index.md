# Data Migration RFI

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-04-24 09:06:57.881-04
- Solicitation Number: FA283426Q0001

## Summary

The United States Air Force, Air Force Life Cycle Management Center, AFLCMC/WVSS is seeking information from interested companies with the capability to deliver a data migration solution for the 3DEXPERIENCE (3DX) platform. Previous efforts configured the application code to serve as a collaboration and approval platform to support the Government's migration to a digital-based acquisition program. The application integrates data from other supporting applications including Cameo and Teamwork Cloud (TWC) and CATIA.
