## Summary

Award notice for the contract associated with RFQ N66604-26-Q-0059. This procurement was solicited as a sole source requirement to The Spiratex Co. (CAGE Code: 7U574). At this time, Spiratex is the only approved source of supply for thin line towed array hoses, as specified by Naval Sea Systems Command (NAVSEA). Qualification of new vendors would require a functioning towed array to be removed from the fleet, which would cause an unacceptable cost and schedule impact.
