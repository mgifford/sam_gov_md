# BTO Innovation Opportunity (BIO) Fair

- Agency: DEPT OF DEFENSE
- Type: Special Notice
- Posted: 2026-04-10 09:27:55.392-04
- Solicitation Number: DARPA-SN-26-04

## Summary

The Biological Technologies Office (BTO) of the Defense Advanced Research Projects Agency (DARPA) will host the BTO Innovation Opportunity Fair (BIO Fair) event for prospective proposers on November 12, 2025, in Arlington, Virginia. Advance registration is required for in-person and virtual event. There will be no on-site registration. The event will include overview presentations by DARPA personnel, including BTO Program Managers.
