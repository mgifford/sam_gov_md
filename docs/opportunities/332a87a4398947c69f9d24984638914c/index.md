# CSO: Automated Tool Control System for Delaware National Guard Army Aviation Support Facility

- Agency: DEPT OF DEFENSE
- Type: Special Notice
- Posted: 2026-04-23 15:55:29.961-04
- Solicitation Number: W912L5-26-S-C010

## Summary

See attached 4-page Commercial Solutions Opening (CSO) for an Automated Tool Control and Inventory Management System at Army Aviation Support Facility (AASF) for Delaware Army National Guard (DE ARNG). Tool storage with automated tool tracking and accountability reporting system. Real time 100% accurate inventory of all tools within the system while permitting efficient retrieval and return to storage. DE ARNG U.S. Property and Fiscal Office (USPFO) will select a vendor and issue one FAR-based purchase order using DFARS Subpart 212-70 Defense Commercial Solutions Opening (CSO) procedures. The primary purpose of the system is to mitigate risk of foreign object damage (FOD) in aircraft. Second, the system should increase tool accountability, decrease tool shrinkage, increase maintenance efficiency, and collect tool usage data. Submit solution briefs to USPFO Contracting Office points of contact listed below.
