# Ventilation System Components

- Agency: ENERGY, DEPARTMENT OF
- Type: Sources Sought
- Posted: 2026-03-11 09:56:26.609-04

## Summary

United Cleanup Oak Ridge (UCOR) requires information on ventilation system components with the following specifications for an approximately 475,200 cubic feet volume: Exhaust approximately 60,000 cubic feet per minute (cfm) 7 air changes per hour The system requires the use of HEPA filtration with the following requirements: Nuclear Grade 99.97% Filter Ability to Bag in and Bag Out Pre Filter 0.3 micron Please provide the following for available options: Louvers Physical Dimensions (Diameter or Length x Width x Height and Weight) Installation Means and Price (Frames or Hangers) Prefer gravity damper Fans Physical Dimensions (Diameter or Length x Width x Height and Weight) Installation Means and Price (Frames or Hangers) Electrical Power Requirements (Voltage and Phase) Ducting Installation Means and Price (Hangers) HEPA Filter Units Physical Dimensions (Length x Width x Height and Weight)
