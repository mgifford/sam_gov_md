# Tactical Electronics 2nd Line Elite EOD Tool Kits

- Agency: DEPT OF DEFENSE
- Type: Combined Synopsis/Solicitation
- Posted: 2026-04-24 14:43:31.592-04
- Solicitation Number: M6700126Q0062

## Summary

This is a solicitation for commercial products prepared in accordance with FAR Overhaul Part 12. The Government requires four (4) brand-name 2nd Line Elite EOD Tool Kits by Tactical Electronics. In accordance with FAR Overhaul 12.102(a), this requirement is restricted to the specified brand name only; "equal" products will not be considered. Please refer to the attached RFQ document and Brand Name Justification for complete technical specifications, evaluation criteria, and instructions for submitting a quotation.
