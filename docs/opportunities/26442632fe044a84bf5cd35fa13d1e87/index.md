# Siding Replacement

- Agency: INTERIOR, DEPARTMENT OF THE
- Type: Solicitation
- Posted: 2026-03-13 15:27:06.538-04
- Solicitation Number: 140P6426B0003

## Summary

SOLICITATION NOTICE: The National Park Service, Interior Regions 3/4/5, Ohio Major Acquisition Buying Office, intends to issue a Request for Proposal (RFP) for the Siding Replacement project for the Indiana Dunes National Park in Porter, IN. Work will consist of providing all labor, materials, tools and other equipment necessary for siding replacement, painting, roof replacement, and window replacement. SOLICITATION NUMBER: 140P6426B0003 PROJECT MAGNITUDE: Between $500,000 and $1,000,000 TYPE OF CONTRACT: This will be a firm-fixed price contract Construction Wage Rate Requirements for Porter County, IN will apply. EVALUATION FACTORS: See solicitation for details North American Industry Classification System (NAICS) CODE CLASSIFICATION and SET ASIDE: The NAICS Code is 236220, and the size standard is $45.0 million. This project is set aside 100% for Total Small Businesses. SITE VISIT: See solicitation for details The contracting office will not provide hardcopies or email copies of the solicitation. REGISTRATIONS: In order to be awarded a contract, an offeror must have an UEI number. Also, the offeror must be registered and active in the System for Award Management (SAM), website at http://www.sam.gov, and have completed their Online Representations and Certifications at the time of the solicitation close date once the solicitation is posted. POINT OF CONTACT: Jarrod Brown, Contracting Officer, jarrod_brown@nps.gov, 234-571-7515.
