## Summary

REQUEST FOR INFORMATION Radar Foam Repair THIS IS A SOURCES SOUGHT SYNOPSIS ONLY. THIS IS NOT A SOLICITATION, NOR REQUEST FOR PROPOSALS/QUOTATIONS. NO CONTRACT WILL BE AWARDED FROM THIS REQUEST FOR INFORMATION. The 162d Wing Contracting Office located in Tucson, AZ is seeking qualified sources for repair/overhaul of F16 Radar Absorption Panels using approved materials, procedures in accordance with approved drawing(s) and technical order(s). The purpose of this request for information is to gain knowledge of potentially qualified sources and their size classifications (i.e. Small/Large business, HUBzone, 8(a), etc.). No reimbursement will be made for any costs associated with providing information to this request for information, nor any subsequent exchanges. The North American Industrial Classification System (NAICS) Code is 336413 with an associated U.S. Small Business Administration size standard of 1,250 Employees. The Product Service Code (PSC) is J016. Contractors will be required to go through source approval process. This process is done via the F16 System Program Office/Air Force Life Cycle Management Center, at Hill AFB, UT.
