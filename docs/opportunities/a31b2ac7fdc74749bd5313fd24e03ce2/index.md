# Request for Information - F-15 Rudder Sensor Depot Activation Capabilities

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-04-17 12:15:07.454-04
- Solicitation Number: FA8634-22-F-0035-03

## Summary

This request for information focuses on the provision of the above-referenced Depot Activation capabilities for the F-15 EX’s current EX Rudder Sensor. The repair and maintenance of the Rudder Sensor requires not only the Rudder Sensor itself but all associated parts/spares and the rights in technical data necessary to maintain and ensure the software used in the displays functions correctly to support verification of repairs at designated ALC.
