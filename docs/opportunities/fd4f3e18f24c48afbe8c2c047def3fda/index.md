# MOF Williams River at Tea Creek Bridge Replacement

## Summary

THE SOLICITATION FOR THIS PROJECT WILL BE POSTED ON OR ABOUT April 15, 2026. PROJECT DESCRIPTION: MOF Williams River at Tea Creek Bridge Replacement. Work for this project consists of, but is not limited to: Furnishing and installing one two span box beam bridge system on FR 135 Tea Creek Campground Bridge Replacement. Construction survey and staking, E&S, clearing and grubbing, installation of cofferdams and dewatering the site, excavation for placement of riprap, removal of existing structure, installation of bridge structure, micro piling design and installation, roadway subbase and aggregate, asphalt cement, guardrail, and riprap armoring. Other construction activities to be expected are turbidity monitoring, in-channel Perimeter Control. Fish Relocation fish removal, candy darter monitoring and capturing, and stream rehabilitation. Contractor is responsible for ensuring all equipment meets State, Federal, and Local Laws. PROJECT LOCATION: This project is located in the Monongahela National Forest on the Marlinton Ranger District Ranger District. Contractors shall furnish all labor, supervision, equipment, materials, transportation, supplies (unless specified elsewhere as Government-furnished), and incidentals required to accomplish specified work in compliance with the terms, specifications, and provisions of the contract. Both performance and payment bonds will be required for this project as well as a bid guarantee. Estimated project magnitude is between $1,000,000 and $5,000,000. NAICS 237310 with size standard $45 mil.
