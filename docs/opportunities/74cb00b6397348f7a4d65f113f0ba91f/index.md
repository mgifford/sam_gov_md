# NOI - Service Agreement for the NHLBI Pulmonary Function Testing (PFT) Laboratory SentrySuite Server and Workstations

- Agency: HEALTH AND HUMAN SERVICES, DEPARTMENT OF
- Type: Presolicitation
- Posted: 2026-04-17 14:46:59.852-04
- Solicitation Number: 75N98026Q00222

## Summary

Pre-solicitation notice of intent to award a sole source contract for a one-year service agreement to provide maintenance, software updates, EMR interface support, and Windows 11 compatibility upgrades for the NHLBI Pulmonary Function Testing (PFT) Laboratory SentrySuite/Vmax server and workstations. This requirement supports ongoing clinical research operations and ensures system functionality, cybersecurity compliance, and data integrity. See attached notice for full details.
