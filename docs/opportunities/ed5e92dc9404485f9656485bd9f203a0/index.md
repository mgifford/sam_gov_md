# SECURITY ACCESS CONTROL MAINTENANCE

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-03-26 09:47:34.965-04
- Solicitation Number: N6883626QS030

## Summary

To provide Maintenance and Service for the currently installed, Lenel Controlled Access, electronic security systems (ESS) detailed in Technical Exhibit 1 (TE1). The equipment listed in TE1 are the major components that make up the security systems that operate Common Access Card (CAC) reader doors and gates, Closed Circuit Television (CCTV) video monitoring systems, and intercom systems installed throughout the buildings at Naval School Explosive Ordnance Disposal (NAVSCOLEOD). This list is not all inclusive. Maintenance and Service must be included for any hardware/software not listed on TE1 that supports any of these ESS to include operating system updates as needed to maintain system functionality.
