# Model PM8572A-1 - 50MHz Dual-Channel Pulse Waveform Generator

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-03-23 14:05:40.489-04
- Solicitation Number: FA8517-26-R-0383

## Summary

Automatic Test Equipment (ATE) Component to support Low Altitude Navigation and Targeting Infrared for Night Automated Test Set (LANTIRN ATS) AN/AAM-104. The purpose of this effort is to procure PN: PM8572A-1, from TABOR ELECTRONICS LLC, (CAGE: 9MSS4) as the Original Equipment Manufacturer (OEM). PN: PM8572A-1 is the COTS, F3I replacement for PN: WW2572A-1MEG-S-RACK, NSN: 6625-01-585-4346KV, from TABOR ELECTRONICS LTD, (CAGE: 9WD0A), Noun: Generator, Signal (Pulse Waveform Generator, provides high-frequency, independent pulse, square, and arbitrary waveforms)
