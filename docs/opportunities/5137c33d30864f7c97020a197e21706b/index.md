# Cloud Hosting, System Administration, and Website Support and Development for the Institute of Education Science

- Agency: EDUCATION, DEPARTMENT OF
- Type: Sources Sought
- Posted: 2026-03-27 13:14:51.087-04
- Solicitation Number: AP4050

## Summary

The Institute of Education Sciences (IES) plans to award a new task order for support of its web operations by providing cloud hosting, system administration, and website administration and critical feature development, and user, staff and developer support. IES currently maintains a virtual data center hosted in Amazon Web Services (AWS). The primary function of the data center is to host survey collection and dissemination websites in support of IES’ mission. This includes web servers, database servers, a vast array of websites and data driven web applications, a system management and web development environment, terminal servers housing analytical and statistical software, and other system support servers.
