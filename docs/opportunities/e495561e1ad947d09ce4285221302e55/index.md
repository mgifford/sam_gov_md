# Enterprise Cloud Modernization (ECM) Sole Source Justification

- Agency: COMMERCE, DEPARTMENT OF
- Type: Presolicitation
- Posted: 2026-04-23 14:13:59.836-04

## Summary

The USPTO intends to award a contract to GDIT for Enterprise Cloud Modernization (ECM) cloud usage and cloud professional services on a sole source basis. ECM is a professional services requirement to support the Infrastructure/Hosting - Cloud product, with public cloud enhancements for effective cost savings, streamlined efforts and a purposeful investment. See attached Sole Source Justification (SSJ) for details.
