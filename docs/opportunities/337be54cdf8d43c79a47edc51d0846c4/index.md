# F-16 Wheel Assembly

- Agency: DEPT OF DEFENSE
- Type: Presolicitation
- Posted: 2026-03-12 18:27:07.598-04
- Solicitation Number: FA8203_F16_Wheel_Assembly

## Summary

The Government anticipates issuing a new Firm Fixed Price (FFP) sole source contract to Jay-Em Aerospace Corp. (cage code 23233). The Government does not own the rights to the data for the F-16 Landing Gear Wheel Assembly (NSN 1630012982516 LE P/N 220A123-2) . Suppliers will need to have or obtain the technical data and drawings from the original equipment manufacturer Jay-Em Aerospace Corp. (cage code 23233). Attempts to reverse-engineer this part has been deemed not plausible or economically reasonable. In accordance with (IAW) FAR 5.207(c)(16)(ii), all responsible sources may submit a capability statement, proposal, or quotation, which shall be considered by the agency.
