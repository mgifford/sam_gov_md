# Deer Creek Lake Gearbox Rebuilds - W91237-26-Q-A034

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-03-17 12:39:30.939-04
- Solicitation Number: W9123726QA034

## Summary

This Requirement shall cover all work required to pick up, clean, inspect, rehabilitate, test, and deliver two (2) parallel shaft gearboxes and one (1) right angle gearbox to Deer Creek Lake. The existing parallel gearboxes are Foote Bros. Maxi- Power Parallel Shaft Reducer Gearbox, serial number (drive side) 2-H56243-A-1 4/66 and (non drive side) 4-H56243-A-1 4/66. The existing right angle gearbox is a Winsmith Speed Reducer Model 8CB, serial number A82-118A. The gearboxes must be rebuilt to original manufacturer?s recommendations. See attached drawing 52.3/1 for arrangement details reference only. Existing couplings shall be removed, preserved, and re-installed prior to shipping back to the project. The Contractor shall test each gearbox as described below in the presence of a representative of the Government. Gearbox deliveries must be within 16 weeks after contract award. Location of project is Deer Creek Lake, 21897 Deer Creek Road, Mt. Sterling, OH 43143-2230.
