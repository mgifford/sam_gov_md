# Airlift Loading Ramps

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-04-09 17:29:03.768-04
- Solicitation Number: W50S7F26SS0002

## Summary

The 148th Fighter Wing has a requirement for three (3) sets of aluminum ramps to load MJ-1/MHU-83s onto C-130, C-17, or C-5 aircraft. Details can be found below. Aluminum Ramps Quantity: 3 Sets Original Manufacturer: DAMAS P/N: Q6915, Q6708, and Q6916 --------------------------------------------------------------- Please respond to the following questions: 1. Do the loading ramps your company provides eliminate the need for any wood shoring, including the pedetal shoring? 2. What are the dimensions of each set of loading ramps? 3. What is the maximum single axle weight each set of loading ramps can handle? 4. Are all components certified by ATTLA #2012.02.03.01 Rev 1? 5. Please list your average delivery timeframe.
