# Multi Culture Program Coordinator

- Agency: DEPT OF DEFENSE
- Type: Presolicitation
- Posted: 2026-03-16 03:31:34.364-04
- Solicitation Number: FA560626QA018

## Summary

Spangdahlem AB is anticipating the solicitation of the below contract. Be advised, this is NOT a solicitation, nor does it guarantee a solicitation for this requirement to be posted in the future. This is an official pre-solicitation, FA560626QA018, this is anticipated to be a firm-fixed-price contract. This is only a pre-solicitation to notify prospective offerors. Feedback is encouraged. Quotes are not currently being requested. The purpose of this contract is to provide newcomers with an introduction to living in Germany. The contractor shall conduct multicultural activities for personnel at Spangdahlem AB, Germany, introductory language and culture classes, bi-weekly cultural tours to Trier, monthly German cooking classes. Mission requirements may dictate short notice changes in class scheduling/ location and/or course cancellation.
