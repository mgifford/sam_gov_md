# Hydraulic Pressure Unit, Subsea and Linear Actuator

- Agency: DEPT OF DEFENSE
- Type: Combined Synopsis/Solicitation
- Posted: 2026-03-25 12:15:06.578-04
- Solicitation Number: N6660426Q0256

## Summary

Please see attached RFQ and Specifications for all details about this opportunity. AMENDMENT 0001 - 15JAN2026 RFQ amended to answer questions from industry. QUESTION: Are these units replacing an existing system? ANSWER: No, this is a new prototype system currently in development. A control system does not yet exist. It would be ideal if the quoted product contains the necessary control electronics and, if possible, that interface to the control system be defined (data type, connector, power requirements, etc.). That being said, the government does have the capability to provide a bespoke control system for the products if required. QUESTION: Are we to quote to desired specifications (no deviations) or are calculations and analysis expected to be completed to support the design? ANSWER: The Specifications are the Government's Minimum Requirements. Minor deviations from the specifications are acceptable but the government will need to assess whether these deviations impact the products’ ability to meet mission need. Calculations and analysis are not required but if included would make a stronger case for the feasibility of success of a given product. QUESTION: Is it possible that the response dates will be extended? ANSWER: The Government will make that determination at a later date, if necessary. QUESTION: We are interested in your RFQ and would like to inquire if there is a technical contact we could discuss the details of this quote request with. ANSWER: All questions about all aspects of this solicitation must be sent to the Government Contracts POC at maria.d.diaz-sotomayor.civ@us.navy.mil. The end-user and the technical team will answer all questions through the Contracts POC and all questions and answers will be posted on the RFQ on SAM.gov. AMENDMENT 0002 - 22JAN2026 RFQ amended to answer questions from industry. QUESTION: We have received the following question from a supplier and would appreciate your guidance: we will need the end user to confirm the exact scope of supply and provide the full technical package supporting the request. In particular, we must receive the valve torque data for all positions involved, the completed LASD form(attached) for the linear applications and all relevant datasheets or project specifications (environmental conditions, materials, hydraulic supply details, compliance requirements, interface information, etc.). These details are essential for verifying feasibility and ensuring the proposed configuration meets the Navy’s expectations. Please provide this additional info and fill out the LASD sheet above as soon as possible as we are unable to quote accurately without it. ANSWER: We’re unable to align the recommended spreadsheet with our technical requirement. It is each offeror’s responsibility to come up with the technical approach to fulfill this requirement. QUESTION: As part of our review of RFQ N6660426Q0256, we kindly request the drawings and models for the Hydraulic Pressure Unit and Hydraulic Actuator referenced in the solicitation. These technical documents will help us evaluate the specifications in detail and prepare a compliant quotation. ANSWER: We did not specifically reference a hydraulic pressure unit and hydraulic actuator in the solicitation, and there are no documents to provide. The desired attributes listed in the specifications provided were derived from our mission requirements, and product capabilities we were able to find open source from potential suppliers. AMENDMENT 0003 - 05FEB2026 Solicitation amended to clarify minimum specifications and extend response time. Please see attached revised documents for all details about this opportunity. AMENDMENT 0004 - 25MAR2026 Solicitation amended to change the minimum requirements and the evaluation method.
