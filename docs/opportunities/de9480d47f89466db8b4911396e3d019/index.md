# Corpus Christi Fire Alarm Replacement Project

- Agency: GENERAL SERVICES ADMINISTRATION
- Type: Presolicitation
- Posted: 2026-03-13 09:50:46.981-04
- Solicitation Number: 32966

## Summary

The General Services Administration, Public Building Service is seeking a fire alarm contractor to furnish all labor, tools, materials, equipment, supplies and supervision for design build services for the purpose of full replacement of the existing fire alarm system with a new fire alarm, voice communication and evacuation system at the Corpus Christi Federal Courthouse. This facility is 4 stories and approximately 169,000 sq-ft. The Contractor must be a licensed fire alarm installation contractor in the State of Texas. SYSTEM DESCRIPTION Available Manufacturers: Manufacturers/Brand: The equipment to be installed must be able to be serviced by at lease three independent Fire Alarm contractors within 250 miles of the projects site as authorized by the manufacturer. All equipment and components shall be new and of the same manufacturer, brand and current model. The Following Manufacturers/brands are known to meet the above requirements. Siemens Building Technologies EST Notifier It is anticipated that this project will take approximately 365 days to complete. Please note that there is no solicitation at this time; the purpose of this announcement is to identify available sources only. Award of any resulting procurement will be based on a source selection employing a “Best Value Concept” methodology.
