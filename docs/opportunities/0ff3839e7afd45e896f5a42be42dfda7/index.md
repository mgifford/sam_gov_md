# REPAIR-LEVEL ROOF REPLACEMENT

- Agency: HEALTH AND HUMAN SERVICES, DEPARTMENT OF
- Type: Combined Synopsis/Solicitation
- Posted: 2026-03-04 15:56:08.57-05
- Solicitation Number: 75H71526R00004

## Summary

This project is intended as a repair-level roof replacement (re-roof) of two modular/manufactured clinic buildings. The work includes removal of the existing roofing membrane, wet or deteriorated insulation, and localized deteriorated sheathing as required, followed by installation of a new thermoplastic polyolefin (TPO) roofing system. This project is not intended to be a structural reconstruction or building upgrade.
