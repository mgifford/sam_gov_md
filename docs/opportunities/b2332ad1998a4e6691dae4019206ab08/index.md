# Pulmonary Function Test (PFT) System

- Agency: VETERANS AFFAIRS, DEPARTMENT OF
- Type: Presolicitation
- Posted: 2026-04-23 18:26:57.754-04
- Solicitation Number: 36C261-26-AP-2864

## Summary

Pulmonary Function Test (PFT) System machines perform body plethysmography, a noninvasive type of lung function testing known as a pulmonary function test, which can determine how much air is in lungs after inhalation and how much air is in lungs exhalation. The system also includes accessories, software, installation and training for clinical users.
