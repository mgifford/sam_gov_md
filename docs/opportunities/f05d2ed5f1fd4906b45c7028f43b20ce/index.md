# Transformer 1600 Watts

- Agency: DEPT OF DEFENSE
- Type: Presolicitation
- Posted: 2026-03-26 08:21:24.506-04
- Solicitation Number: F3N0526042A005

## Summary

200 Transformers -Must be 1600 Watt. -Transformer 220/240 Volt to 110 Volt. -Must be step down only. -All EU plugs must be dual polarity. -Steel sheet housing must be gray color. -Must have portable handle. -Single phase FFP-self cooling. -2-meter flexible cable, shuko plug type F. -Must have 2 each U.S. socket only compatible with plugs type A & B. -Must have 2 primary circuit breaker 4,5 AMP to react as overload and to be used as an on-off switch. -Location of circuit breaker/on-off switch must be in the front centered above socket.
