# Laser Range Finder

- Agency: DEPT OF DEFENSE
- Type: Combined Synopsis/Solicitation
- Posted: 2026-03-20 15:56:14.525-04
- Solicitation Number: FA448426Q0007

## Summary

The 621st Contingency Response Group (CRG) Joint Base McGuire-Dix-Lakehurst, New Jersey is procuring a new requirement of nine (9) compact, military-grade laser range finders to support weather operations at austere and contingency airfields. Devices will be used to establish accurate visibility reference markers and distances, day and night, to improve real-time weather observation and mission planning. The Laser Range Finders specifications must be in accordance with the attached Statement of Work (SOW).
