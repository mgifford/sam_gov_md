# West Point Spillway Gate Painting

## Summary

THIS IS A TOTAL SMALL BUSINESS SETASIDE. THIS PROJECT IS A STAND-ALONE ?C? TYPE, REQUEST FOR PROPOSAL (RFP), BEST VALUE SOURCE SELECTION SOLICITATION. Construction Order of Magnitude: Between $10,000,000 - $25,000,000 This project requires the Contractor to provide all labor, materials and equipment necessary to prep, prime and paint all metal components of the spillway gates, hoists and associated equipment. Painting includes blasting and painting of each structure, minor structural girder modifications, installing/replacing rubber seals and cathodic protection anodes on each gate, and performing weld repairs in accordance with the specifications. In addition, the contractor will be required to remove and replace the ?new? set of stoplogs seal assemblies prior to the gate painting effort. Finally, the contractor shall light clean and provide a maintenance overcoat paint on both sets of stoplogs ?new? and ?old?. The North American Industry Classification System (NAICS) code for this project is 237990, Other Heavy and Civil Engineering Construction.
