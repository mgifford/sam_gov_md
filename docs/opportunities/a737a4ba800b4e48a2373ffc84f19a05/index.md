# Mobile Tactical Cannon (MTC) Request for Prototype Proposal

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-03-06 13:23:42.445-05
- Solicitation Number: W912CH26RA017

## Summary

Army Contracting Command - Detroit Arsenal (ACC-DTA) is soliciting a Request for Prototype Proposal (RPP) for mature and available Mobile Tactical Cannon (MTC) solutions for prototype manufacturing, network integration, ammunition compatibility, safety and characterization testing, and issue to an Army unit for Soldier Evaluation. The Government intends to award a Fixed Price Other Transaction Authority (OTA) Agreement.
