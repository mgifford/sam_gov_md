# White River NF Supervisor's Office Fire Alarm Repair

- Agency: AGRICULTURE, DEPARTMENT OF
- Type: Combined Synopsis/Solicitation
- Posted: 2026-04-24 12:49:20.491-04
- Solicitation Number: 1240LP26Q0023

## Summary

Amended to include Attachment 8 Alarm Zones and Building Map Solicitation Due Date extended to 4/28/26 2:00 p.m. Mt Time to the inbox of rebecca.l.schowalter@usda.gov This is a combined synopsis/solicitation for commercial items prepared in accordance with the format in FAR Part 12. This announcement constitutes the only solicitation. Quotations are being requested and a separate written solicitation will not be issued. Solicitation 1240LP26Q0023 is issued as a Request for Quotation (RFQ) for a fire alarm repair. This acquisition is set-aside for small business concerns. The applicable North American Industry Classification Standard Code is 561621 The small business size standard is $25 Million. This acquisition is a Total Small Business Set-Aside. All responsible sources may submit a quotation which will be considered by the agency. The purpose of this procurement is to replace fire alarm system components and obtain incidental installation services required to repair the existing fire alarm system. The current fire alarm system contains discontinued components that must be replaced with compatible, listed, and manufacturer-approved parts to restore full functionality. The fire alarm system is connected to a monitoring system managed by Proguard Protective Services (Proguard). The connection that remotely synchronizes the alarm system to Proguard has been malfunctioning. A piece of the motherboard is broken. The system has been unable to synchronize with the monitoring company.
