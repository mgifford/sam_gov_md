# 733 CES Fire Alarm Panel Parts & Migration

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-04-17 17:54:26.875-04
- Solicitation Number: FA480026Q0051

## Summary

733 Civil Engineering Squadron is requesting the purchase and installation of a new code compliant Edwards fire alarm control panel, along with all required modules, power supplies, and interface components. This panel shall interface seamlessly with all existing devices and monitoring systems already installed. The vendor MUST have an Authority to Operate (ATO) system programming keys at Ft Eustis. Additionally, the existing fire alarm system needs to be migrated from EST3 to EST4. ONLY ONE AVAILABLE SOURCE IAW RFO 12.102(a) AND RFO 5.101(c)(4)(vii): The fire alarm panel must interface seamlessly with all existing initiating devices, notification appliances, auxiliary panels, and monitoring systems already installed on Ft Eustis. Due to original equipment manufacturer (OEM) licensing restrictions and proprietary programming keys, only the vendor who installed the system has the exclusive authorization and access to provide a compatible fire alarm panel and perform the EST3 to EST4 migration. The existing fire alarm system is Edwards’ brand, but was installed by Convergint Technologies, LLC, an authorized reseller and is designed to operate exclusively with components, software, and programming tools produced and supported by the OEM. As a result, only the OEM possesses the technical data and licensing rights needed to install, program, and commission a fully compatible and code compliant replacement panel. No other vendor can legally obtain the required software keys, configuration files or technically provide these capabilities without violating OEM licensing restrictions or compromising system performance. Our office reached out to Convergint Technologies, LLC. to verify the information above. We were able to obtain a signed letterhead memorandum, dated 25 March 2026, stating "Convergint holds the exclusive authorization and access to the proprietary programming keys, configuration software, and licensing required in this region. These tools are not made available to other distributors or service providers.
