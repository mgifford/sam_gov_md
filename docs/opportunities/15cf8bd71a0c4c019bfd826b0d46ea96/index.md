# ICAN

- Agency: HOMELAND SECURITY, DEPARTMENT OF
- Type: Justification
- Posted: 2026-03-24 10:12:42.229-04
- Solicitation Number: 70Z0G326POIPL0003

## Summary

USCG has a need to procure ICAN software Investment Protection (IP) for software applications Data Switch and Maestro to support the Nationwide Automatic Identification System (NAIS) project. In December 2009, Northrop Grumman Space and Mission Systems of Herndon, VA, was awarded Contract HSCG23-09-C-ADP001 to initiate the NAIS program. ICAN’s software was part of the software incorporated under the NAIS project. The USCG accepted the ICAN software at that time. For the past sixteen (16) years, USCG has maintained and upgraded the ICAN software as part of the NAIS project.
