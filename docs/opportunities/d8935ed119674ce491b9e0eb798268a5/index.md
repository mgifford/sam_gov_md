Solicitation number HC108426R0005 is issued as a request for proposal (RFP). This requirement is for proprietary, commercial off-the-shelf, brand-name Menlo Security, Inc.’s Cloud Browser product, as a Menlo Security, Inc. reseller-provided managed service, providing the Department of War’s remote browsing isolation solution for Cloud Based Internet Isolation (CBII) service.
