# EASEY Phase 2 Re-engineering of CBS/CSA/ECMPS2.0

- Agency: HEALTH AND HUMAN SERVICES, DEPARTMENT OF
- Type: Justification
- Posted: 2026-04-02 09:31:43.627-04
- Solicitation Number: 03272026_NIH_NITAAC

## Summary

The purpose of this posting is to publish an approved justification to exclude fair opportunity. See attached for exact details. This acquisition is to procure contract support to design, develop, and deploy the functions currently performed in the legacy CAMD Business System (CBS) and CAMD System Administrator (CSA) reporting applications into the existing Emissions and Allowance System for Energy (EASEY) framework. The CBS and CSA functions will be fully integrated into the EASEY environment which includes an Application Programming Interface or API based architecture hosted in cloud.gov. This work objective also includes the design, development and deployment of additional ECMPS2.0 functionality.
