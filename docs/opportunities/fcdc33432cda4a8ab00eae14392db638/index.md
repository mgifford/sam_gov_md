# Hardware Content Filtering Diode (HCFD) Solutions for All-purpose Structured EUROCONTROL Surveillance Information Exchange (ASTERIX) Sensor Data Transfer

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-03-17 11:38:08.411-04
- Solicitation Number: M67854-26-I-0090

## Summary

The purpose of this RFI is to identify sources capable of providing a Hardware Content Filtering Diode (HCFD) solution to support the secure transfer of All-purpose Structured EUROCONTROL Surveillance Information Exchange (ASTERIX) data from an unclassifed network to a SECRET network for the Common Aviation Command and Control System (CAC2S).
