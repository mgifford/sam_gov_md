THIS POSTING IS FOR TINKER AFB - MAC BOA HOLDERS ONLY Project Solicitation Number: FA813726R0008 Project Title: WWYK150083 - Repair Transformer Vault 6, B9001 Project Location: Tinker AFB, B9001 IAW FAR 36.204 the magnitude of this project is between $1,000,000 and $5,000,000 Repair to building 9001 transformer room #6 located at column X37 to Y38 on the roof. The work to be performed under the terms of this Order consists of furnishing all materials, labor, plant, tools, transportation, equipment, testing services, quality control and incidentals required to accomplish the repair by replacement of the two gas filled 2000KVA transformers, associated 480V switch gear labeled #6C and #6D, all 15KV switches, and the original 3000A tie bus which have exceeded their rated life expectancy.
