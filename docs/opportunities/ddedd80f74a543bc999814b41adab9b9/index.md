# Base Operating Support Services at Cooperative Security Location Comalapa, El Salvador

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-04-01 13:37:55.252-04
- Solicitation Number: N6945026R0036

## Summary

The outcomes to be achieved are Base Operating Support Services at Cooperative Security Location (CSL) Comalapa, EL Salvador. This acquisition will provide base support services for IT Support and Management, Force Protection, Airfield Facilities, Material Management, Facility Investment, Custodial, Pest Control, Integrated Solid Waste Management (ISWM), Grounds Maintenance and Landscaping, Base Support Vehicle and Equipment, and Environmental Services. The outcomes for this acquisition are consistent with the Federal Acquisition Regulation (FAR) 37.001 definition of service contracts.
