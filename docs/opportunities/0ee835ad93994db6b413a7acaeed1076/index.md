# OCLC Software Annual Subscription

- Agency: DEPT OF DEFENSE
- Type: Special Notice
- Posted: 2026-03-10 14:08:22.469-04

## Summary

The proposed action is a firm-fixed-price purchase order to OCLC, Inc. for proprietary software licenses, hosting, and support services. . The Dudley Knox Library requires continued access to a suite of integrated services for cataloging, discovery, proxy server, and interlibrary loan (ILL) functions. These services are critical for the library to support the research and educational needs of Naval Postgraduate School faculty, staff, and students List of OCLC Products and Services Cataloging and Metadata Subscription Wordshare ILL First Search ILLiad License ILLiad Hosted Server EZProxy Hosted EZProxy Analytics
