# 6680--Loch Raven Boiler Fuel Reserve Tank

- Agency: VETERANS AFFAIRS, DEPARTMENT OF

- Solicitation Number: 36C24526Q0377

## Summary

The purpose of this project is to purchase and install a 6,000-gallon reserve fuel tank to support the boiler at the Loch Raven VA property. The fuel tank will supplement the existing reserve fuel tank and be placed on the existing asphalt drive lane adjacent to the emergency generator. The project includes delivery, installation, fuel line connection, and securing the tank area.
