# Primary/Secondary Crash Network

- Agency: DEPT OF DEFENSE
- Type: Combined Synopsis/Solicitation
- Posted: 2026-04-01 13:58:17.355-04
- Solicitation Number: FA448426Q0010

## Summary

The 305th OSS Joint Base McGuire-Dix-Lakehurst, New Jersey is procuring a new requirement ) for a Primary and Secondary Crash Networks. The current system is FORUM. This upgrade system must be compatible with current base infrastructure and be compatible with current FORUM phones installed at receiving agencies. The system must contain a visual feature that identifies when each agencies activates their phone. The visual feature will also have the ability to isolate individual agencies during activation. The system must be usable for up to 20 parties at McGuire Field and up to 30 parties at Lakehurst Maxfield Field, that directly support the multiple flying missions. The Primary and Secondary Crash Network specifications must be in accordance with the attached Statement of Work (SOW).
