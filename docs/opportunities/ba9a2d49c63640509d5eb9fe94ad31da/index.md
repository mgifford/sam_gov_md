## Summary

The solicitation for FA448426R0009 – Washer and Dryer Services is now available. Please review the solicitation and all supporting documents for complete details and requirements. Clarification: Per the PWS it list 521 Washers, 130 Dryers and 252 Stackable Units. These units are multipled by 12 months to equal the total quanity on the CLINs. Example is CLIN 0003 Provide, install and maintain stackable dryers, quantity is 3024, divide 3024/12 (months). This equals 252 stackable per base and option year, which is what is outlined in the PWS.
