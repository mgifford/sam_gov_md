# Savannah ANG Tank Repairs

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-03-27 11:44:03.959-04
- Solicitation Number: W9128F26RA090

## Summary

This project includes repairs and maintenance of two vertical aboveground storage Tanks 964 and 967 at Savannah ANG, GA. All Work Complete for Tanks 964 and 967 Repairs (R). Bring all tank repairs into compliance with American Petroleum Institute (API) 653 and all applicable design criteria and standards. The DoD AST Standard AW 78-24-27 shall govern where it conflicts with API 650 or API 653. All tank deficiencies noted in previously performed API 653 inspection report shall be corrected in order to bring the tank into compliance. All Work Complete for Tanks 964 and 967 Maintenance (M) including repair tank coatings, provide tank strapping chart, and perform final API 653 inspection.
