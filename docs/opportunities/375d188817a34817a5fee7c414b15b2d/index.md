## Summary

VA is seeking to identify qualified contractors and identify brands of nursecall systems with a code blue communication system capable of integrating with existing Hill-Rom hospital beds without the use of middleware to capture all the required bed data as well as other patient/room data and provide that information in a visual dashboard accessed via the workstations located at the nurse stations.
