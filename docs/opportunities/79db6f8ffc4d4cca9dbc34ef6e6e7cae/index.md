## Summary

Item Name: Interface Box NSN: 7025-01-706-1912 Part Number: 13045871 Minimum Quantity: 203 each Maximum Quantity: 819 each End Item: Bradley Description: This Interface Box is an external training device used on routes RS 170 signals from the long-range advanced scout surveillance system (LRAS3) and multiple integrated laser engagement system (MILES) training equipment to the improved Bradley acquisition subsystem (IBAS) and commanders independent view (CIV).
