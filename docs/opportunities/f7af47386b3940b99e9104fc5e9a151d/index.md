- Agency: DEPT OF DEFENSE

## Summary

This is the formal release of the NAVSEA Team Submarine (TEAM SUB) follow-on for the Ship Acquisition Program Manager (SHAPM) requirement. This solicitation is for a follow-on task order that will be issued under the SeaPort-NxG Multiple Award Contract (MAC) --- ONLY COMPANIES THAT HAVE AN ACTIVE SEAPORT-NXG MAC AWARD MAY SUBMIT A PROPOSAL AS A PRIME CONTRACTOR FOR THIS REQUIREMENT. NOTE that the solicitation and its attachments are available via link to the PIEE module, as provided in the attachments section of this posting.
