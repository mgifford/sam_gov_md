# Wisconsin Model V-465D1 Engine Overhauls Naval Base Coronado (NAS North Island)

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-04-23 19:40:18.084-04
- Solicitation Number: N0024226PR00437

## Summary

The Contractor shall provide all personnel, equipment, tools, materials, supervision, quality control and other items and non-personal services necessary to perform repair service for 65 horsepower, gasoline fired internal combustion engines, Electronic ignition, Wisconsin Model V-465D1 as defined in this PWS. The Contractor shall perform to the standards in this contract and return engines to original operating specifications. Up to 5 engines repaired in the base year and 5 engines repaired in each subsequent option year. POP: base year and four, one-year options Please read the sources sought document and review all attachements before submitting your capability statements.
