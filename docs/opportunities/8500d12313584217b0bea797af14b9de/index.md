## Summary

The Subarray Transmit Orthogonality for Receive Multiplexing (STORM) program expands and exploits transmit multiplexing for SPY-6. STORM introduces two capabilities in the base program: Subarray-Level Independent Transmit (SLIT) and Directional Modulation (DM).
