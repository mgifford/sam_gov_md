# Ft. Campbell Rectifier Replacement

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-03-17 09:59:03.713-04
- Solicitation Number: W912QR-26-R-A017

## Summary

This requirement is for a complete, turnkey solution to replace obsolete DC power plant systems, including rectifiers and inverters, which are past their useful service life. The selected contractor will be responsible for providing all necessary personnel, equipment, tools, materials, supervision, and transportation to successfully execute the project.
