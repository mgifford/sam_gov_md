# Spofford Circuit

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-04-24 16:17:33.226-04
- Solicitation Number: FA309926Q0006

## Summary

Network connection services - Provide a 20 megabytes per second (Mbps), end-to-end network connection from 581 Liberty Dr, Laughlin AFB, TX 78843 to 14776 Hwy 131, Quemado, TX 78877. The connection will be considered a point-to-point circuit. No internet or ISP data is required. Termination of the circuit at 14776 Hwy 131, Quemado, TX 78877 will be in the government-owned pedestal. The point of demarcation at the delivery points will ultimately need to be a fiber optic connection as either an LC or SC type connector on single-mode fiber pair. The vendor will need to be able to provide a way to submit trouble tickets for outages to repair the circuit if an outage occurs.
