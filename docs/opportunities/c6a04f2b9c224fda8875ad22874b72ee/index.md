# 14816 PAO Chiller System Combined Synopsis/Solicitation

- Agency: DEPT OF DEFENSE
- Type: Combined Synopsis/Solicitation
- Posted: 2026-03-19 11:54:40.637-04
- Solicitation Number: 14816

## Summary

The purpose of this contract effort is to provide the 309 Software Engineering Group (SWEG)/521AB System Integration Lab (SIL) F-35 program a Polyalphaolefin (PAO) Chiller Skid System. The requirement consists of installing a new PAO chiller, pump, manifolds, distribution piping, controls, and instrumentation. The system will provide cooling for aircraft representative equipment.
