# PLC upgrades for Blast and Paint

- Agency: DEPT OF DEFENSE
- Type: Combined Synopsis/Solicitation
- Posted: 2026-04-23 15:43:19.006-04
- Solicitation Number: M6700426Q0026

## Summary

Marine Corps Logistics Command in Albany Georgia intends to award a sole source purchase order to Mcnaughton-McKay, Cage Code 0CFH4, for the purchase of replacement components for Allen Bradley Micrologix 1500 Programable Logic Controllers (PLC) with Allen Bradley Compact Logix 5380 series. Justification and approval for this award can be found in the attachments. McNaughton-McKay is the sole authorized distributor for Allen Bradley parts.
