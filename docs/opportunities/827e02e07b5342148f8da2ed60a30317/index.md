## Summary

This is a Sources Sought for the Government to make a determination of whether small businesses can support a consolidation of the herbicide and pesticide contracts for all USACE lakes within the Trinity Region of Texas. Please see attached the sources sought document and draft performance of work statements (PWS) for further details on this Sources Sought.
