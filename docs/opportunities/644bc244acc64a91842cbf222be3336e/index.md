# X1FZ--Apartment Rentals for Medical Residents

## Summary

The Department of Veterans Affairs will be soliciting for vendors that can provide Apartment Rentals for Medical Residents for the Asheville VA Medical Center located in Asheville, NC. This will be performed in accordance with the Statement of Work included in the solicitation. A Request for Quotation will be issued on or about 21 April 2026. The Period of Performance is for a Base and Four Option Years. This is a Total SDVOSB Set-aside. The applicable North American Industry Classification System (NAICS) Code is 531110 - Lessors of Residential Buildings and Dwellings the Size Standard is $34 Million, and the Product Service (PSC) Code is X1FZ - Lease/Rental of Other Residential Buildings. Contractors interested in providing this service should monitor Contract Opportunities for the solicitation # 36C24626Q0411. Point of Contact: Emily Donahue Emily.Donahue@va.gov
