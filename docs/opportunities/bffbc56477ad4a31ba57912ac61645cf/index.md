# Request for Quotation for Supply and Installation of Wireless Intrusion Detection & Security Alarm System for U.S. Embassy Phnom Penh

- Agency: STATE, DEPARTMENT OF
- Type: Solicitation
- Posted: 2026-04-02 00:51:52.539-04
- Solicitation Number: 19CB6026Q0005

## Summary

The U.S. Embassy Phnom Penh has a requirement for a vendor to supply and install a “Wireless Intrusion Detection & Security Alarm System” for U.S. Embassy Phnom Penh. The replacement solution provided by the vendor must be high quality and reliable, while meeting or exceeding the current power requirements of the existing system in both function and capacity. You are invited to submit a quotation. Please see attached Request for Quotation.
