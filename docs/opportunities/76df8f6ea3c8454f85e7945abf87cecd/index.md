# Rynglok Tool Kit

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-04-13 16:04:39.058-04
- Solicitation Number: FA301626Q0081

## Summary

The 12th Maintenance Squadron's Fabrication Flight (12MXS/MXMF) at Joint Base San Antonio-Randolph conducting Market Research for one (1) new RYNGLOK TOOL KIT, Part Number RTSK8-02-001 (NSN 5180-01-378-8873). This toolkit is essential for the installation and maintenance of Rynglok hydraulic tubing couplings on the T-7 RedHawk aircraft. As Eaton is the Original Equipment Manufacturer (OEM) with exclusive rights to the drawings, specifications, and manufacturing data for Rynglok systems, no other brand of tool kit can be substituted. The tool kit must have all the necessary tools to service the following tubing sizes: #3 (3/16"), #4 (1/4"), #5 (5/16"), #6 (3/8"), #8 (1/2"), #10 (5/8"), #12 (3/4"), #14 (7/8"), #16 (1"), #20 (1 1/4"), and #24 (1 1/2"). Only authorized sellers / distributors shall apply for this requirement. The NAICS Code for this is 332912.
