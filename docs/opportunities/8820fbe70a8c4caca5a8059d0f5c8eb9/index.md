# Program Manager Development Program

- Agency: HEALTH AND HUMAN SERVICES, DEPARTMENT OF
- Type: Sources Sought
- Posted: 2026-03-12 14:30:44.164-04
- Solicitation Number: 75N99226SS105

## Summary

The Advanced Research Projects Agency for Health (ARPA-H) is conducting market research in support of the Office of the Director (DIRO). ARPA-H has a requirement for a Program Manager Development Program to further the agency’s mission. This requirement supports ARPA-H in executing its mission by enhancing Program Manager (PM) development, and implementation capability through an ARPA-H Program Manager Development Framework. This requirement is intended to deploy and sustain a fully operational Program Manager Development Framework by: Providing a day 1 fully deployable ready-to-use Program Manager Development Framework—requiring no additional stand?up time—that is specifically designed to train and prepare Program Managers to operate effectively in an ARPA-like environment. The contractor shall demonstrate experience operating within an ARPA or similar high-risk, high-reward innovation culture to design, deliver, and refine the framework so that it rapidly builds and sustains program management, performer engagement, risk-mitigation, and transition-planning capabilities. Strengthening the capabilities of current ARPA-H PMs through structured mentorship, cross-PM learning, and targeted skill development that improves program design, metrics definition, risk management, and transition strategies. This Sources Sought Notice serves to determine the capabilities of the market to provide specialized and agile solutions. Please see the attachment PMDP_SSN_031226 for further information that is being requested and instructions for responding to this notice.
