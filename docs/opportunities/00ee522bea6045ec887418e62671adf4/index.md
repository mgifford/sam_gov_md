# New Road Closure Electric Lift Gate System for Dalton Cache Border Station LPOE located at Mile Post 42, Haines Alaska

- Agency: GENERAL SERVICES ADMINISTRATION
- Type: Special Notice
- Posted: 2026-03-12 03:24:12.59-04
- Solicitation Number: 47PD5526R0022

## Summary

This procurement is for: Small Business (total- set aside). Primary NAICS is 238210. PLACE: Dalton Cache Border Station/ LPOE LOCATED AT: Mile Post 42 Haines Hwy/ Hwy 3, Haines AK. Summary of project: To install a new electric vertical lift gate system at the border station. Lift gates will need to span entire width of Hwy 3. Two lift gates w/ 20 foot spans for a total of 40 feet of closure ability when both gates are down. System will also need to be administered from inside the border station as well as the ability to use a remote device for use at beginning and end of duty day. Project will need to be fully completed as soon as possible but no later than the end of Summer (late August- early Septmeber) Please refer to project documents for more information.
