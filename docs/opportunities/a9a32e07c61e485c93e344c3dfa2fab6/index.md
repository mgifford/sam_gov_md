# NOCA: Area X SFS Panel

- Agency: DEPT OF DEFENSE
- Type: Special Notice
- Posted: 2026-03-06 14:26:40.522-05
- Solicitation Number: 2026-PZI-1681

## Summary

This is a notice of the Government's intent to solicit, negotiate and award a sole source contract to Advantor Systems. The intent of this contract is to procure the delivery and installation of an additional alarm system in a secure location within building 1614 on Hanscom AFB. Advantor Systems shall provide all labor, equipment, and materials required to install the ACS/IDS in accordance with the Intelligence Community Directive Number 705, DoD Special Access Program Security Manual, UL 2050, Security Requirements for Cryptographic Modules, and the Technical Specification for SCIF Construction.
