# Line Powered Mobile Column Lift System

- Agency: DEPT OF DEFENSE
- Type: Combined Synopsis/Solicitation
- Posted: 2026-04-13 09:22:49.803-04
- Solicitation Number: W911S226U3000

## Summary

Amended to change NAICS from 423830 to 333998 Line Powered Mobile Column Lift System comprising four (4) individual columns, designed for lifting large vehicles such as semi-trucks and buses. It provides a total lifting capacity of 72,000 lbs(18,000 lbs per column). This system operates on 208-230/190V, 50-60 Hz, 3-Phasepower (low voltage option). It features intuitive push-button controls, multiple automatic safety locks, and is ALI/ETL Certified to meet ANSI/ALI ALCTV safety standards. Designed for portability and versatility in vehicle maintenance, it supports SINGLE, PAIR, or TEAM operation modes.
