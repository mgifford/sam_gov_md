# ECDIS-N and NCDS Engineering Support Services Seaport

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-03-25 09:40:29.987-04
- Solicitation Number: N6449826R3106

## Summary

This requirement is for services to be provided to The Naval Surface Warfare Center Philadelphia Division (NSWCPD) Technical Code 541, Ship Navigation System, NCDS and Navy ECDIS Controls Branch (Code 541), include the engineering, technical personnel and facilities required to maintain the navigation system operational readiness and integrate technological improvements for in-service and new construction navigation system configurations and equipment. NOTE: Type of Contract: This will be a Cost-Plus-Fixed-Fee (CPFF) Task Order that will be competed under the SeaPort-NXG Multiple Award Contract (MAC) amongst all MAC holders (unrestricted, full and open competition).
