## Summary

***2026.03.27 - RFI 002_Repair Airfield Pavement Crack Seal_signed has been uploaded to this site.*** ***2026.03.26 - Reminder: RFIs DUE TODAY, 26 MARCH 2026 COB -YTPM 25-3002_Site Visit Sign-In_2026.03.25 has been uploaded to this site.*** ***2026.03.20 (2) - RFI 001_Repair Airfield Pavement Crack Seal_signed has been uploaded to this site.*** ***2026.03.20 (1) - YTPM 25-3002_Site Visit Sign-In_2026.03.19 has been uploaded to this site.*** Project YTPM 25-3002, Repair Airfield Pavement Crack Seal (Notice ID F5N3FQ6022A001 and FA660626B0002) at Westover Air Reserve Base, Chicopee, MA. This acquisition is issued as an Invitation for Bid (IFB), Firm-Fixed-Price contract. Performance period will be sixty (60) Calendar Days from issuance of the Notice to Proceed. This project is being solicited as a 100% Total Small Business Set-Aside, subject to the North American Industry Classification System (NAICS) Code 237310 (Highway, Street, and Bridge Construction), with a size standard of $45.0 million. This solicitation is issued as an IFB with an estimated thirty-five (36) Calendar Day response period. There is an organized Site Visit planned for this solicitation; the correct date, time, and location is included in this written solicitation (SF 1442). The estimated magnitude of construction for this project is between $150,000 and $3500,000. There is no Government Furnished Property supplied for this project. This solicitation and all supporting documentation is posted herein for review and download. All contractors and subcontractors interested in this project must register at this site https://SAM.gov. It is the bidder(s) responsibility to check the website periodically for any amendments to this solicitation. There will be no other form of distribution for this solicitation other than downloading from this site. Prior to supplying a bid, interested bidders must have an active registration in the System for Award Management (SAM). Firms can register via the SAM internet website at https://sam.gov/entity-registration . If the bidder is a Joint Venture (JV), the JV entity must have a valid SAM registration in the SAM database representing the JV as one business/firm/entity. If, at the time of award, a bidder is not successfully and actively registered in the SAM database, the Government reserves the right to award to the next prospective bidder. Notice to Potential Bidder(s)/Supplier(s): The Government reserves the right to cancel this solicitation, either before or after the closing date. In the event the Government cancels this solicitation, the Government has no obligation to reimburse a bidder for any costs.
