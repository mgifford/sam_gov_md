# Multi-Source Correlator Tracker Software Warranties

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-03-31 04:39:32.487-04
- Solicitation Number: FA5641

## Summary

The USAFE-AFAFRICA A3ZJ Command and Control Management Branch as well as the Battlespace Command and Control Center (BC3) has a mission need to procure three (3) Cyber Security Long Term Support (LTS) Maintenance Warranties (12 month duration) for the Multi-Source Correlator Tracker (MSCT) software that is utilized within the Scalable Control and Reporting Agile Mission-Kit (SCRAM-K) system in use at multiple USAFE locations. The MSCT is a critical software component of the overall SCRAM-K system that is in operational use at multiple USAFE-AFAFRICA Main Operating Bases (MOBs). This system is the approved command and control/battle management system given an Authority to Operate (ATO) by USAFE-AFAFRICA Deputy Commander (DCOM), and these LTS maintenance licenses are required to maintain continued ATO approval status. These licenses provide the MSCT software with patching/updates/vulnerability scan capabilities to continuously report and ensure it is in compliance with Risk Management Framework (RMF) security protocols. Please respond directly to this sources sought if capable or have any questions/concerns by 7 April 2026. Lastly, please do not submit any quotes or pricing as we will not consider these factors in our acquisitions. This is explicitly only a sources sought for these license software warranties.
