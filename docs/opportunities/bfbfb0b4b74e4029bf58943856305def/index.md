# DA10--National Teleradiology Program Picture Archive and Communication System Upgrade (VA-26-00042676)

## Summary

This contract action is for a sole source, firm-fixed-price contract with Intelerad Medical Systems, Inc. (Intelerad), 800 De Maisonneuve Blvd East, 14th Floor, Montreal, Quebec. for the upgrade of the existing National Teleradiology Program (NTP) Picture Archive and Communication System (PACS) from IntelePACS version 4.9 to version 5.7 or the most current version available at the time of implementation, including migration of all NTPspecific applications. The contract will also provide ensuing system maintenance and technical support for the upgraded platform. The Statutory Authority for this effort falls under Federal Acquisition Regulation (FAR) Overhaul VA Deviation 6.103-1 entitled, Only One Responsible Source and No Other Supplies or Services Will Satisfy Agency Requirements. See attached redacted J&A. The Point of Contact for this action is: Sean McAvoy, Contract Specialist; e-mail: Sean.McAvoy1@va.gov. The North American Industry Classification Systems (NAICS) code for this action is 541511. The Product Service Code for this action is DA10.
