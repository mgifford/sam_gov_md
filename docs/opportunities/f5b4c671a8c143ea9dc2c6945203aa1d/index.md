# Misawa Air Base F-35 Engine Run Simulator

- Agency: DEPT OF DEFENSE
- Type: Combined Synopsis/Solicitation
- Posted: 2026-04-02 20:00:38.651-04
- Solicitation Number: FA520526QB048

## Summary

The 35th Contracting Squadron is seeking vendors able to source and deliver an iPerformX F-35 Engine Run Simulator or Equal that meets or exceeds the below specifications. The requirement requires direct delivery to Misawa Air Base Japan. Please see attached RFQ and SOO for additional details and requirements. Salient Characteristics Needed: 1. Unclassified System: The simulator, including all hardware, software, and data produced must be fully unclassified 2. Must meet or exceed all specifications of the iPerfromX Simulator IAW the attached Statement of Objective (SOO) 3. Provide a 5-year support package 4. Provides timely support response capability – w/in 24 hrs 5. Has analog button controls along with touch screen for realistic operations 6. Provides physical backlit switches and knobs for all critical systems 7. Provides for software update abilities 8. Provides for ease of use 9. Provides advanced instructor inject scenarios and control the simulator 10. Proves full cockpit system that provides enhanced instructor controls for injecting emergency scenarios and monitoring student responses The contractor shall provide all equipment, tools, parts, transportation, personnel, labor, supervision, and management to install the F-35 Engine Run Simulator Suite at Bldg. 651, Maintenance Training Section, Misawa Air Base, Japan. The contractor shall install the complete simulator suite and provide comprehensive training on the maintenance and use of the installed system.
