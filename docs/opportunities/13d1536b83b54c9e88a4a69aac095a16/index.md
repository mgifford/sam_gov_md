# Bldg 840 Chiller Replacement

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-04-21 11:34:45.467-04
- Solicitation Number: FA489726Q0011

## Summary

This post is for market research purposes only. THIS IS NOT A SOLICITATION FOR PROPOSALS AND NO CONTRACT WILL BE AWARDED FROM THIS SOURCES SOUGHT. No reimbursement will be made for any costs associated with providing information in response to this synopsis or any follow up information requests. MHAFB is gauging the interest of total small business set aside with further potential set aside into socioeconomic categorys. MHAFB has a requirement for design-build construction services to replace the existing Chiller and install a Buffer tank that requires a new concrete pad at Bldg 840. The following work shall be IAW with manufacture instructions in conjunction with the SOW. The following requirement will include, but not limited to, the following: 1. Dispose of the old unit and any waste materials in accordance with all environmental regulations. Base HVAC shall be given an opportunity to salvage components such as condenser motors and control boards if it does not cause a delay. 2. Prepare the existing concrete pad/structural base for the new chiller installation. 3. Install vibration isolation pads/springs as specified. 4. Install all required valves, strainers, flow switches, and instrumentation. 5. Install 5" flanged piping to tie in 500-gallon carbon steel buffer tank into chilled water system return line. Must have 2" urethane insulation with aluminum jacketing. 6. Provide BACnet integration with system chiller. (Chiller interface card provided with equipment) 7. Perform manufacturer-certified startup procedures and ensure functional system operation. All IAW the statement of work (SOW). The magnitude of this project is between $100,000 and $250,000. In order to reduce chances of missed emails, the Subject line shall be "Bldg 840 Chiller Replacement Sources Sought Response" Each company’s response to this Sources Sought shall be a maximum of 3 pages and shall include the following information: Company name CAGE code Address Point of Contact Phone Number E-mail address Company's Small business category (Small Business, Woman-Owned Small Business, Small Disadvantaged Business [SDB] to include the 8(a) Program, Historically Underutilized Business Zones [HUBZone], Service- Disabled Veteran Owned Small Business [SDVOSB]). A brief statement of the company's interest in bidding on the project as well as details of the company's capability to perform a contract of this type, magnitude, and complexity
