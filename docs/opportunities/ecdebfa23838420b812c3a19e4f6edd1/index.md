# 400G Optical Network Upgrade Installation and Commissioning

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-03-19 12:08:46.866-04
- Solicitation Number: FA480026SSJBLEON

## Summary

THIS IS A SOURCES SOUGHT NOTICE FOR INFORMATION AND PLANNING PURPOSES ONLY. This announcement closes 12:00 p.m. Eastern Time on Monday, 26 March 2026. The applicable North American Industry Classification System (NAICS) for this requirement is 541512, Electrical Contractors and Other Wiring Installation Contractors, with a corresponding size standard of $34M. Interested parties must be registered under NAICS 541512 in the System for Award Management (SAM) at www.sam.gov. The Government plans to consider all information submitted in response to this Sources Sought Notice. The contractor shall provide all project management, personnel, labor, equipment, tools, and technical expertise necessary to perform the installation, configuration, testing, and integration of government-furnished 400G optical networking equipment. The 633rd Communications Squadron (633 CS) at Langley Air Force Base, Virginia, has a need to upgrade its core optical transport network. This project requires professional services to install and commission Government-Furnished Equipment (GFE), specifically Cisco Network Convergence System 2006 Series (NCS2K) Dense Wavelength Division Multiplexing (DWDM) nodes and 400G Transponder Modules, into the existing network infrastructure to complete this critical modernization effort. Please see the attached notice for additional information.
