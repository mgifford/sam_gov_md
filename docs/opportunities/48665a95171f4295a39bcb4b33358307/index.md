## Summary

The Mission and Installation Contracting Command - Installation Contracting Office, West Point, NY (MICC-West Point) is issuing this SOLICITATION Brand Name Instron 1500HDX Universal Testing Machine(UTM) Delivery and Installation at the United States Military Academy (USMA). The solicitation is in electronic format only and available on the Internet at: https://www.SAM.gov/ It is the offeror’s responsibility to check www.SAM.gov provided as necessary for any posted changes. FOLLOW SOLICITATION INSTRUCTIONS Solicitations (Quotes) and additional supporting documentations (If required - See solicitation 52.212-2 Addendum) must be submitted via email to Leslie Duron, leslie.a.duron.civ@army.mil on or before the close date noted in the solicitation.
