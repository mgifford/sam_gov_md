# Sources Sought: Automated Logic Corporation WebCTRL for Facilities for Facility Related Control Systems (FRCS) programmable building controllers

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-03-12 16:48:13.938-04
- Solicitation Number: JA20

## Summary

The purpose of this sources sought notice is to determine if there is commercially available controls equipment with equivalent specifications of the Automated Logic Corporation WebCTRL for Facilities for Facility Related Control Systems (FRCS) programmable building controllers as detailed in Attachment 2 – Technical Information. The attachment describes the programmable building controller requirements for the Naval Base Ventura County California HVAC DDC system.
