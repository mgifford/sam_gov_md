# NATO Business Opportunity: M&S Proof of Concepts - Centralized Data & EXCON-Simulation Automation for MDO

## Summary

Headquarters, Supreme Allied Commander Transformation (HQ SACT) has issued a Request for Proposal (RFP) for Modelling & Simulation (M&S) Proof of Concepts: Centralized Data and Exercise Control (EXCON)-Simulation Automation for Multi-Domain Operations (MDO). The increasing complexity of MDO requires enhanced interoperability, automation, and data coherence across NATO Computer Assisted Exercises (CAX), Computer Assisted Wargames (CAW), and Computer Assisted Analysis (CAA). Current processes supporting exercise scripting, simulation execution, and data preparation are fragmented, simulation-specific, and heavily dependent on manual intervention. To address these issues, the RFP establishes two complementary but separable capability development efforts: 1) automation between EXCON toolsets and simulation systems and 2) development of a centralized, common agreed dataset for NATO simulation systems. While technically distinct and awardable separately, both efforts contribute to a common objective: enabling simulation-agnostic automation and standardized, reusable data foundations to support NATO MDO-driven events.
