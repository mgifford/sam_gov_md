# 7A20--RFI - Budget Tracking Tool (BTT) Replacement Tool

- Agency: VETERANS AFFAIRS, DEPARTMENT OF
- Type: Special Notice
- Posted: 2026-03-27 13:58:00.293-04

## Summary

The VA has determined an acquisition strategy for Budget Tracking Tool (BTT) Replacement Tool. Please see further information below. Please note that since the initial posting of the request for information in 2025, the title of the acquisition has changed from Budget Tracking Tool (BTT) Replacement Tool to Veteran Affairs (VA) Integrated Planning, Programming, Budgeting, and Execution (PPBE) and Procurement System (VIPPS). As of March 27, 2026, this action has been solicited as full and open through General Services Administration (GSA) Multiple Award Schedule 54151S - Information Technology Professional under Request for Quote (RFQ) Number 36C10B26Q0107.
