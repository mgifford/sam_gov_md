# Missile Segment Enhancement (MSE) Containerized Launcher (CL) and Remote Interceptor Guidance 360 Degrees (RIG-360) Containerized System (CS) Development and Demonstration

- Agency: DEPT OF DEFENSE
- Type: Presolicitation
- Posted: 2026-04-21 11:33:58.711-04
- Solicitation Number: W31P4Q-26-F-0071

## Summary

This requirement is for MSE Containerized Launcher and RIG-360 Containerized System Development and Demonstration. The effort activities establish the requirements to support the design, development, and test of one (1) MSE CL prototype and one (1) RIG-360 CS. This is a best effort rapid development and demonstration effort for a containerized launch system. The CL that is developed and built for this effort will serve as the demonstration unit to support virtual and live fire test demonstrations. The proposed source for this effort is Lockheed Martin Missiles and Fire Control (LMMFC), 1701 West Marshall Drive, Grand Prairie, Texas 75051, commercial and government entity (CAGE) code 64059. In accordance with FAR Part 6.302-1(a)(2)(iii)(A) and (B), LMMFC is the only source with knowledge, expertise, and full disclosure technical data package (TDP) to perform this effort. The Government does not have the necessary intellectual property rights to utilize the data for full and open competition.
