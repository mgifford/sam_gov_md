# Direct Digital Controllers (DDC) and Related Software

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-04-20 16:28:46.625-04
- Solicitation Number: FA940126Q0000001

## Summary

Kirtland Air Force Base (KAFB) seeks to implement a brand name only requirement to procure Direct Digital Controllers (DDC) and related software for projects that include replacement, upgrade, or new installation of the HVAC equipment under the Energy Management Control System (EMCS). This Class Brand-Name Only Justification would cover all KAFB DDC systems and will include plant controllers, as well as single zone controllers. This requirement is for FY26 through FY31.
