# FY26_Maxwell AFB_Jones Auditorium Wireless Microphone Refresh

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-04-02 13:19:46.943-04
- Solicitation Number: F2XHC36041A001

## Summary

Jones Auditorium supports senior-level professional military education and requires a reliable, secure, and user-friendly wireless microphone system suitable for frequent, high-visibility instructional and ceremonial use. The current wireless microphone system has reached end-of-life and must be replaced with a modern, digitally encrypted, Audio-over-IP-capable solution.
