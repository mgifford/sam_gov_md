There are telecommunication lines (fiber & copper) that run between these two areas along a concrete pedestrian trail used primarily by park service personnel. � This project will install approximately 1,000 linear feet of new telecommunication lines within the existing buried conduit between the visitor center and the headquarters area including new wires, new panels on both ends, connections to existing systems, and troubleshooting to ensure proper operation.
