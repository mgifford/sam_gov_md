## Summary

This is a solicitation for commercial items prepared in accordance with (IAW) the format in Federal Acquisition Regulation (FAR) subpart 12.201 as supplemented with additional information included in this notice. This announcement constitutes as the only solicitation; quotations are being requested and a written solicitation will not be issued. Solicitation number 70Z03826QL0000112 is issued as a Request for Quotation (RFQ). This solicitation document and incorporated provisions and clauses are those in effect as of November 28, 2025, IAW the Revolutionary FAR Overhaul (RFO). The applicable North American Industry Classification Standard Code is 488190. The small business size standard is $40 million. This is a restricted solicitation. All responsible sources may submit a quotation which shall be considered by the agency. It is anticipated that a firm-fixed price purchase order shall be awarded on a sole source basis to Orbit Communication Sytems, Inc. (CAGE Code: 34NS0) as a result of this solicitation for the following items: SCHEDULE OF SUPPLIES Line Item 1 Nomenclature: Audio Control Panel National Stock Number (NSN): 5895-01-HS2-1948 Part Number (P/N): AL23-3140-9MODVER3 (Manufacturer CAGE Code: S5496) Quantity: 6 Each Line Item 2 Nomenclature: Jack Panel (JP) National Stock Number (NSN): 5895-01-HS2-1949 Part Number (P/N): AL22-3330-31MODVER1 (Manufacturer CAGE Code: SH921) Quantity: 3 Each Line Item 3 Nomenclature: Audio Control Panel National Stock Number (NSN): 5895-01-HS2-1947 Part Number (P/N): AL24-3370-9MODVER4 (Manufacturer CAGE Code: S5496) Quantity: 4 Each Sources must be able to obtain the required technical expertise, engineering data, and Original Equipment Manufacturers (OEM), Orbit Communication System LTD (Cage Code S5496) and Orbit Communication LTD (Cage Code SH921), parts required to successfully perform the required repairs.
