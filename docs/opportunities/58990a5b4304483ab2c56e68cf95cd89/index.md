# 618-26-702 EHRM Install DDPO 2

- Agency: VETERANS AFFAIRS, DEPARTMENT OF
- Type: Presolicitation
- Posted: 2026-04-22 13:57:54.491-04
- Solicitation Number: 36C26326B0030

## Summary

The construction contractor shall supply all equipment, labor, materials, and supervision to complete project upgrades which includes the installation of Data Drops and Power Outlets, mounting of Integrated Outpatient Code Editor(IOCE), Tracking Boards and Wall mounts. The initiative is an implementation and the deployment of new Electronic Health Record (EHR) system across federal entities and install additional Data Drop and Power Outlets (DDPO’s) as identified during the CSR assessment. The period of performance for this work is ninety (90) calendar days from issuance of Notice to Proceed.
