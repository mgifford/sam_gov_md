# Dual-energy, high-energy X-ray linear accelerator

- Agency: DEPT OF DEFENSE
- Type: Special Notice
- Posted: 2026-03-12 09:38:12.25-04
- Solicitation Number: FA865126XRAY

## Summary

The government intends to procure 1 each dual-energy Varex Linatron M6A from VJ Technologies to include: one each internal collimator, one each low leackage x-ray head, one set of spare parts, one laser for alignment, one compact RF unit, one modulator/power distribution cabinet, one power line conditioner, one (set) of system cables and hoses, two each M-Series Operator Reference Guides, factory buy off, linatron connection guidance and commissioning assistance, external signal interface, control console, standard spare parts package, system verification summary and test summary included.
