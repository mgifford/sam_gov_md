# Modular Trailer Lease, Cannon AFB, NM

- Agency: DEPT OF DEFENSE
- Type: Justification
- Posted: 2026-04-10 10:38:04.006-04
- Solicitation Number: FA485526F0025

## Summary

The current Trailer lease will end 9 April 2026 after exercising all option years. The current contractor's leased facility is already in place, equipped with a working OSA, appropriate SIPR/NIPR protocols and required fire and security equipment tied into the base infrastructure. Based on leadership feedback, any disruption to the end user would result in unacceptable mission risk. IAW FAR 8.401 (Overhaul) and GSA 538.7104-3(b)(iii), the order is hereby issued on a sole source basis in the interest of economy and efficiency because it is a logical follow-on to an FSS order already issued on a competitive basis.
