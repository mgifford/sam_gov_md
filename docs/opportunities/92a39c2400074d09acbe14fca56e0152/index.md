# RFQ - 16-inch Endress+Hauser Promag W 500 Electromagnetic Flowmeter

- Agency: DEPT OF DEFENSE
- Type: Combined Synopsis/Solicitation
- Posted: 2026-03-30 16:13:25.141-04
- Solicitation Number: W911S226U2929

## Summary

Amended to include set aside code. An inline, remote version electromagnetic flowmeter for measuring industrial or municipal water and wastewater. It features a 4-line illuminated display with touch control, 4-20mA HART output, and a polyurethane liner. The process connection is a 16 inch CL.150 carbon steel flange (ASME B16.5) with 316L electrodes. Includes a weather protection cover.
