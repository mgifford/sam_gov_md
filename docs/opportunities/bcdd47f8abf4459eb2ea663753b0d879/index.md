# 99 CES PAPI Lights

- Agency: DEPT OF DEFENSE
- Type: Combined Synopsis/Solicitation
- Posted: 2026-03-10 10:24:12.938-04
- Solicitation Number: F3G3CE6042A001

## Summary

The requirement is for various lighting components for the Precision Approach Path Indicator (PAPI) system manufactured by ADB Safegate. These parts are critical for the repair and maintenance of the existing airfield lighting control system at Nellis Air Force Base. The specific components include lamps, filters, and other proprietary parts that are essential for the system's proper operation and FAA compliance.
