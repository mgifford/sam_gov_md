# Ordnance Handling Equipment

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-04-16 16:02:12.721-04
- Solicitation Number: N0017426R1011

## Summary

The Naval Surface Warfare Center, Indian Head Division Picatinny Detachment (Government) requires the manufacture and assembly of MK 3 MOD 2 Strong Back Lifting beam Ordnance Handling Equipment (OHE), other Undefined OHE as well as spare parts, which are required for the movement, transportation and storage of various naval storage containers. This handling equipment supports multiple Navy program weapons containers. Amendment 0001 posted 4-16-26.
