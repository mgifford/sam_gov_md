## Summary

The Base Operations Support Contract (BOSC) for Marine Corps Base Camp Pendleton, California, is a Firm-Fixed Price (FFP), Indefinite Delivery Indefinite Quantity (IDIQ) performance-based contract, set aside 100% for Small Businesses under NAICS code 561210 (Facilities Support Services) with a $47 Million size standard, and has a potential duration of up to 96 months.
