# T-7 Sunshade Shelters

- Agency: DEPT OF DEFENSE
- Type: Special Notice
- Posted: 2026-04-07 14:54:14.452-04
- Solicitation Number: FA301626R0026

## Summary

The 12th Maintenance Group is installing new sunshades to support the initial deployment of T-7A aircrafts. The project includes three new bays in total. One of these bays will be added to an existing row of sunshades that is already scheduled for installation. The other two bays will be constructed as a new, standalone row. Each bay is designed to shelter two T-7A aircrafts. This will be solicited as a Sole Source based on the awarding this to any other firm would be impractical, as it would require them to attach their structure to the end of the existing row, which would violate the warranty of the work currently underway and create unacceptable risk for the government.
