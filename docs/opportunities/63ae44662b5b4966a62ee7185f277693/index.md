# SENTINEL Secure Chat Servers

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-04-16 12:57:28.913-04
- Solicitation Number: N6523626QE138

## Summary

The Naval Information Warfare Command (NIWC) Atlantic, Code 22130, has a requirement for the procurement of SENTINEL Secure Chat Servers. This is a single source solicitation for commercial items. The Government intends to negotiate and award a Firm-Fixed Price purchase order on a noncompetitive basis to Trusted Solutions LLC (UEI NNPSXPPTLEH1, cage: 7GNG9, Small Business). The Government intends to negotiate solely with Trusted solutions under the authority of FAR 13.106-1(b)(1). See attachment 2 sole source justification. In accordance with FAR 6.302-1, all responsible sources may submit a capability statement, proposal, or quotation, which shall be considered by the agency. The NAICS Code is 541511 and the size standard is 34 million. See attached RFQ for specific details regarding this requirement.
