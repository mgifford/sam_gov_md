# Nuclear Surety Cross Check Analysis (NSCCA) for ASIC

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-04-21 11:01:07.242-04
- Solicitation Number: FA8219-26-NSCCA_ASIC

## Summary

The purpose of this Request for Information (RFI) is to gather market research information for understanding the Government’s options regarding Nuclear Surety Cross Check Analysis (NSCCA)/ Nuclear Safety Analysis and Technical Evaluation (NSATE)/ Independent Verification/Validation (IV&V) for the Application Specific Integrated Circuit (ASIC) hardware/ software integration for the Sentinel Weapon System. NSCCA/NSATE is defined in AFMAN 91-119/AFMAN 91-118; test, simulations, and other evaluation techniques must be used to prove the safety and surety of the weapon system and for a thorough software verification process. The Government is evaluating Independent Verification/Validation options for nuclear safety design certification for ASIC hardware/software integration for the Sentinel Weapon System.
