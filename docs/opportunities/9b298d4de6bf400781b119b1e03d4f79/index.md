# TRANSCOM Reference Data Management (TRDM)

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-04-20 15:03:02.261-04
- Solicitation Number: TRANSCOM26D003

## Summary

TRDM provides a single distribution source of enterprise reference data for enhancing C2, transportation, logistics, deployment, and financial systems’ reference data management and functional data collection. The scope of TRDM requires the contractor to perform current technical & Cybersecurity operations, maintenance, and distribution of the TRDM system and data while modernizing the architecture. These tasks include but are not limited to requirement analysis, data collection, data management, interface and architecture validation, and data distribution. Requests and requirements fielded by TRDM require contractor expertise on implementation, fielding, planning, scheduling, interoperability testing, documentation, and other key tasks defined in the software development lifecycle.
