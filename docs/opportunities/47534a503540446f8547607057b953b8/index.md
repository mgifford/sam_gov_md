# Request for Information - F-15 MITR-DVR Depot Activation Capabilities

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-04-17 12:19:56.432-04
- Solicitation Number: FA8634-22-F-0035-04

## Summary

This request for information focuses on the provision of the above-referenced Depot Activation capabilities for the F-15 EX’s current EX Modular Instrumentation Traffic Recorder (MITR) Digital Video Recorder (DVR). The repair and maintenance of the MITR-DVR requires not only the MITR-DVR itself but all associated parts/spares and the rights in technical data necessary to maintain and ensure the software used in the displays functions correctly to support verification of repairs at designated ALC.
