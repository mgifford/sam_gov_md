# MARINE CORPS ENTERPRISE NETWORKS (MCEN) EVENT MANAGEMENT SYSTEM (MEMS)

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-04-22 12:47:15.817-04
- Solicitation Number: M67854-26-I-4925

## Summary

Marine Corps Systems Command (MCSC), the predecessor to the current organization, Program Executive Officer Digital & Enterprise Services (PEO Digital), was tasked with deploying a solution to monitor all events on the Marine Corps Enterprise Network (MCEN) within a single solution that integrates all requirements outlined below. The solution must be capable of capturing telemetry that enables derivation of behavioral and attitudinal performance indicators. Exemplar insights enabled by these indicators include network traffic analysis, service reliability tracking, monitoring application performance, user experience and satisfaction scores, system usability scales, and/or net promoter-like scores. System shall enable enterprise network service monitoring and extend to the tactical edge. Solution must be capable of discovering, monitoring, and mitigating/remediating vulnerabilities. Solution must have autonomous syncing capability and be able to execute functions when disconnected from the network. Role based access shall be delegated to each administrative and tactical level. Shall support common and custom application programming interface and integrate with established situational awareness dashboards.
