# Request for Information (RFI) for Supplemental Type Certificate (STC) Updates for APX-119 Identification Friend or Foe (IFF) Transponder Retrofit on the E-11A Battlefield Airborne Communications Node (BACN) Fleet

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-04-10 09:33:13.874-04
- Solicitation Number: BACN-STC-APX199IFF-26

## Summary

This is a Request for Information (RFI) for market research and planning purposes only. Responding is not a prerequisite for any future solicitation. The purpose of this RFI is to identify potential contractors with experience in updating Supplemental Type Certificates for Bombardier Global 6000 and Global XRS aircraft along with planning, integration, and testing required to meet Federal Aviation Administration (FAA) and USAF airworthiness requirements. Further details can be found in the attached RFI letter.
