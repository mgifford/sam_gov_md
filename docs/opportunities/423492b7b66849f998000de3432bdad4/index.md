## Summary

***Amendment*** The purpose of this amendment is to hereby replace the Item Description in its entirety, update Clause 52.212-2 Evaluation and extend the closing date. Please complete Company Information (Name & Address) and CAGE Code in Block 17a. 2. Please wet sign, put the name and title of signer, and date signed in blocks 30a., 30b., and 30c. respectively. 3.
