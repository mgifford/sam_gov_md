# Web-Based News Streaming

## Summary

Spangdahlem AB is conducting market research for the below requirement. Be advised, this is NOT a solicitation, nor does it guarantee a solicitation for this requirement to be posted in the future. This is an official request for information (RFI), FA560626QA020, this is anticipated to be a firm-fixed-price contract. This is only a request for information to notify prospective offerors. Feedback is encouraged. Quotes are not currently being requested. The purpose of this requirement is to provide web-based news streaming services. The contractor shall provide web-based streaming services that are accessible through the Non-classified Internet Protocol Router Network (NIPR) for personnel at Spangdahlem AB, Germany.
