# Submarine Sail Systems Field Services Support

- Agency: DEPT OF DEFENSE
- Type: Presolicitation
- Posted: 2026-04-21 09:55:37.925-04
- Solicitation Number: N6449826R3108

## Summary

This is an advanced notice for N6449826R3108. This procurement is for specific On-Site Field Representative support services for the following Submarine HM&E systems: High Data Rate Antenna System, Towed Buoy Systems, Towed Antenna systems, Communications Antenna Systems, Imaging/Periscope Systems, Universal Modular Masts Systems, Radar/Navigation Systems, Snorkel Induction System, Electronic Support Measure Mast System, Multifunction Modular Mast system, Sail Closure Door Systems, and general submarine sail HM&E Systems. This requirement is being solicited only to Seaport MAC Holders. Any proposals submitted from Non-Seaport MAC Holders will not be considered for award. Any questions regarding this notice should be emailed to: jessica.m.kelley23.civ@us.navy.mil Phone calls will not be accepted.
