## Summary

Update: Amendment 0001 in effect 04 March 2026. Amendment to answer question from Industry. See Solicitation Amendment 0001 for details. Army Contracting Command - Rock Island (ACC-RI) has requirement for M82 Washer Assembly. This requirement is a Small Business Set Aside, non-commercial, simplified acquisition under FAR part 13 (Simplified Acquisition Procedures) under NAICS code 326299 - All Other Rubber Product Manufacturing .
