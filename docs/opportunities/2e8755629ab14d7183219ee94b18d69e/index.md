# Mower Timber Stand Improvement Project

- Posted: 2026-03-18 07:18:29.815-04

## Summary

Statement of Requirement Contractor shall furnish all labor, materials (except for government-furnished property identified in

4.0 Government Furnished), supervision, transportation, equipment, supplies, and perform all work necessary for timber stand improvement removal in accordance with the specifications, provisions, and clauses of this solicitation and any resultant contract. This work involves the felling of trees to pre-commercial thinning (PCT) thin dense timber stands to increase the growth of the remaining trees. All the cut trees will be left on site. This project will allow the remaining trees to grow faster and increase the wood value for timber production. Vendors are HIGHLY ENCOURAGED to attend the Site Visit.
