## Summary

Amendment 36C24626Q0439 (0002) Question (Black) / Response (Red) Additional questions for RFQ 36C24626Q0439: Regarding Removal/Disposal: Can you please advise if the existing furniture to be removed/disposed of is the SAME list that was provided for RFQ 36C24626Q0412? I would not want the VA to be charged twice for the removal/disposal if it is included in 36C24626Q0412. Response: It is not the same furniture. Behavioral Health furnishings will be within the patient rooms and lounges. This will feature beds, mattresses, wardrobes, and existing lounge chairs, desks, and etc. If not, can you please provide a list of existing furniture that is to be removed/disposed for THIS RFQ? Is it only the items shown on page 4 of the Domiciliary Project Binder document? Is POD D the only pod that has existing furniture? Response: The items will be a one-for-one swap for what is being bought on the contract. Page 4 shows an example of the items with the exclusion of the desk and file cabinet. Regarding Carpentry Services for Wardrobes: What exactly is required for the carpentry services? Are these to be anchored to wall? Is there caulking required? Please clarify. Response: VA recommends contractors to confirm installation of the wardrobes with the basis of design and/or their recommended product. The existing cabinetry is framed in and will have to be removed. That is the extent of the carpentry services.
