# Draft Commander 3000 Apparatus Pump Testing Unit

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-04-07 16:06:49.348-04
- Solicitation Number: FA480326Q0057

## Summary

The unit shall be a mobile fire pump testing and training system capable of performing true pump tests from draft, creating a suction lift requiring the use of a primer in accordance with NFPA 1911 (2017 Edition). The system must support pump testing up to 3,000 GPM and allow training operations using recycled water.
