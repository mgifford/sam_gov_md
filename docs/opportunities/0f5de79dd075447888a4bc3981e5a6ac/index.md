## Summary

This is a sources sought/market research survey for a comprehensive facility rebranding effort to remove legacy branding and install new branding across specified locations. This effort will be a small business set aside. Approximately eighty percent (80%) of the work will be completed at Aberdeen Proving Ground, Maryland.
