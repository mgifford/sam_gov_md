# Single-Award Task Order Contract, PSFB

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-03-11 11:36:20.673-04
- Solicitation Number: 45CONS

## Summary

The 45th Contracting Squadron (45 CONS) is publicizing this Sources Sought Notice to gather industry information and input regarding an anticipated Single Award Task Order Contract (SATOC). 45 CONS is considering a small business set-aside program to accomplish the SATOC for Patrick Space Force Base. The proposed primary North American Industry Classification System (NAICS) code for this requirement is 238220 - Plumbing, Heating, and Air-Conditioning Contractors, however, individual task orders issued under this IDIQ may utilize the following NAICS codes, stated below, depending on the specific work requirements: 236220 - Commercial and Institutional Building Construction 237110 - Water and Sewer Line and Related Structures Construction 238110 – Poured Concrete Foundation and Structure Contractor 238210 – Electrical Contractors and Other Wiring Installation Contractors *Any NAICS code not expressly listed above shall not be utilized as a NAICS code for any task order issued under this IDIQ. Please see attached Statement of Work and Sources Sought Notice for details.
