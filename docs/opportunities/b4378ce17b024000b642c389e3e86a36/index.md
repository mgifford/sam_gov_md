# 24Ft, Fully Enclosed, Bumper Pull Trailer (Salient characteristics provided)

- Agency: DEPT OF DEFENSE
- Type: Combined Synopsis/Solicitation
- Posted: 2026-04-08 09:34:28.572-04
- Solicitation Number: W911S226U2980

## Summary

White Sands Missile Range (WSMR) Meteorology Branch requires the purchase of a new, commercially off the shelf, 24ft, fully enclosed, bumper pull trailer. The unit will be used for mission support activities throughout the test range. This requirement is urgent and required to augment existing support capabilities for upcoming scheduled missions. WSMR Met Branch understands the preferred delivery is 30 days ARO, however, due to the urgency we are requesting a 14-day delivery timeframe ARO. The fully detailed salient characteristics can be found on Trailer Procurement Requirements document.
