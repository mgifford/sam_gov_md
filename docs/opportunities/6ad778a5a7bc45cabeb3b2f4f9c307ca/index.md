# RFI for LINK 22 radios

- Agency: DEPT OF DEFENSE
- Type: Special Notice
- Posted: 2026-03-12 19:06:30.262-04
- Solicitation Number: FA821726LINK22

## Summary

Amendment 01 - This effort is extended until 10 April 2026 due by 11 am MDT, all other information remains the same. This is a REQUEST FOR INFORMATION ONLY to obtain information for the LINK 22 radio, timing source, and transit case as described in the attachment. The scope is as follows: Per the DoD Joint Tactical Data Link Migration Plan, the legacy Link-11 tactical data link is being sunset, creating critical capabilities gap for the CRC. The CRC relies heavily on Beyond-Line-of-Sight (BLOS) data exchange, and this project will integrate a new Link-22 capability to enhance operational effectiveness and support Joint All-Domain Command and Control (JADC2) objectives for both contingency and homeland defense missions. To mitigate this gap, this project will integrate the Link-22 tactical data link into the CRC weapon system. The Government will furnish some core Link-22 components, including the System Network Controller (SNC), Signal Processing Controller (SPC), and COMSEC device in a separate transit case. This RFI is focused on gaining information for a needed solution of a Link-22 capable radio, ancillary equipment, timing source, and the design and fabrication of an adaptable transit case solution that houses the radio system and its supporting peripherals and interfaces with the core Link-22 case that contains the SNC, SPC, and COMSEC device. Responses per direction in the RFI attachment. Due date is 18 Mar 2026 by 11 am MDT.
