# Team Submarine SHAPM Follow-on

- Agency: DEPT OF DEFENSE
- Type: Presolicitation
- Posted: 2026-03-12 14:05:52.355-04

## Summary

This Advance Notice is for the Team Submarine follow-on for the Ship Acquisition Program Manager (SHAPM) requirement. See enclosed SHAPM Advance Notice Supplemental Details pertaining to this requirement. The solicitation is planned to be released within the next 2 weeks. An Industry Day for this requirement was held on December 17, 2025. See attached Industry Day Slides and Industry Day Q and A. The DRAFT Statement of Work, Personnel Qualifications, and Sections L and M are now available as attachments to this notice. Note that the SeaPort Portal will NOT be used to post the solicitation or for offerors to submit proposals. See attached DRAFT Instructions for Submission of Offers. For the time being, all questions must continue to be submitted within SeaPort. This Advance Notice and subsequent solicitation is for a task order that will be issued under the SeaPort-NxG Multiple Award Contract (MAC) --- ONLY COMPANIES THAT HAVE AN ACTIVE SEAPORT-NXG MAC AWARD MAY SUBMIT A PROPOSAL AS A PRIME CONTRACTOR FOR THIS REQUIREMENT.
