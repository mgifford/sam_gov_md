# H10 Engineering Technical Support

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-04-23 13:57:21.222-04
- Solicitation Number: N0017826R3013

## Summary

The Naval Surface Warfare Center Dahlgren Division (NSWCDD) has a requirement for technical and engineering support for asymmetric warfare projects for research, development, engineering and technical support services vehicle for the research, analysis, design, development, prototyping, test and evaluation, integration, project management, implementation, and documentation of tactical and non- tactical systems as well as various Command, Control, Communications, and Computer Intelligence Surveillance Reconnaissance (C4ISR) Radio Frequency (RF) sensor systems/subsystems and other Special Systems associated with Homeland Security, Anti-Terrorism, Force Protection, Unmanned Systems and other related Programs. To request access to the DD Form 254 (Department of Defense Contract Security Classification Specification) associated with this solicitation, proposers must submit a formal request via email to Jennifer Montrief at jennifer.e.montrief.civ@us.navy.mil. This request must include the firms Commercial and Government Entity (CAGE) code and Unique Entity Identifier (UEI). Due to the presence of Controlled Unclassified Information (CUI) within the DD254, these identifiers are required to verify the companies status and eligibility to handle sensitive mission data before access is granted.
