# 57MM Target Practice (TP) Cartridges

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-04-01 14:10:04.506-04
- Solicitation Number: N0017426R1009

## Summary

Request for Proposal to procure 57mm Target Practice (TP) Cartridges. The 57mm TP Cartridge is a 57mm, electrically-primed cartridge designed to function in the MK 110 Gun Mount (GM). The TP Cartridge consists of an inert projectile, a brass cartridge case, an electric primer, and propellant.
