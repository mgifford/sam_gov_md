# AI Effects at the Edge Assessment Event

- Agency: DEPT OF DEFENSE
- Type: Special Notice
- Posted: 2026-04-09 10:22:30.389-04
- Solicitation Number: W519TC-26-AE-AEEA

## Summary

The Cyber Fusion Innovation Center (CFIC), in collaboration with Army Cyber Command (ARCYBER) Technology and Innovation Center (ArCTIC) is in search of industry expertise for the development of an Artificial Intelligence (AI) / Machine Learning (ML) model embedded on a commercial platform. This platform will enable advanced capabilities for modular mission payload (MMP) development and seamless integration with connected sensors. This release documents ArCTIC’s methodology for pursuing, selecting, and assessing such solutions. Please see the attachment for more information.
