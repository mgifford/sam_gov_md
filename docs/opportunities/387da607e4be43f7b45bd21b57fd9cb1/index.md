# Intrusion Detection System (IDS)

- Agency: DEPT OF DEFENSE
- Type: Presolicitation
- Posted: 2026-04-16 13:59:40.535-04
- Solicitation Number: W50S9426QA001

## Summary

In accordance with FAR 5.101(a) this is a presolicitation notice posted to the GPE, for an upcoming anticipated sole source solicitation. THIS IS NOT A REQUEST FOR QUOTES. This is an informational post as required by regulation. Below is a quick synopsis of the requirement. The 282d Combat Communications Squadron (CBCS), North Smithfield, requires a security system installed in order to meet security standards required to maintain wartime readiness. Advantor is currently installed at the 282nd, any IDS equipment installed would have to tie into the current system and report back to BDOC.
