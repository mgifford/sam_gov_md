- Agency: AGRICULTURE, DEPARTMENT OF
- Type: Special Notice
- Posted: 2026-03-26 13:16:52.557-04

## Summary

This notice is for a BPA Call Order to be issued against an existing Land ManagementIntegrated Resources (LMIR) BlanketPurchase Agreement (BPA). To respond to this solicitation, contractors must have an existing LMIR BPA that includes Tree Planting Services for the Rio Grande National Forest. Please see the attached BPA Call Order Form, Price Quote-Schedule of Items and submittal instructions. Please send any questions to andrea.haines@usda.gov no later than 3/31/2026, 3:00 MT.
