# 68TM-5 Instron Test Stand System - Sole Source

- Agency: DEPT OF DEFENSE
- Type: Combined Synopsis/Solicitation
- Posted: 2026-03-09 12:35:40.105-04
- Solicitation Number: FA822726Q5576

## Summary

This is a combined synopsis/solicitation for commercial items prepared in accordance with Revolutionary FAR Overhaul (RFO) Part 12. This announcement constitutes the only solicitation; proposals are being requested and a written solicitation will not be issued. Solicitation number FA822726Q5576 is issued as a request for quote (RFQ). This solicitation incorporates provisions and clauses by reference. The full text of provisions and clauses may be accessed electronically at www.acquisition.gov. The Government anticipates this acquisition to be a Sole Source award to Instron, A Division of Illinois Tool Works, Inc. (CAGE 80160). See justification attached to this solicitation. This requirement is filed under NAICS code 334519 and Product Service Code 6625. This requirement will provide a 68TM-5 Instron Test Stand System for the 582nd Missile Maintenance Squadron at Hill Air Force Base, Utah. The product provided will be supplied in accordance with the specifications in the attached Purchase Description. **Please refer to document FA822726Q5576 Combined Synopsis Solicitation for the anticipated CLIN structure upon award** - All CLINs will be established on a firm fixed price basis. - Proposals must indicate if the vendor can achieve delivery within 90 calendar days ARO (after receipt of order). If this delivery is not practicable, vendor must propose an alternate delivery schedule. Selection of awardee will include comparative analysis in accordance with RFO 12.203, Evaluation of Quotations and Offers, giving value to: price; meeting specifications in the product description; and delivery within the specified timeframe. Any exceptions to the instructions specified in the Product Description must be identified in the quote. By submitting a quote, the contractor agrees to all stipulations in the purchase specification unless specified as an exception in the quote. **The due date for all proposals is Monday, 16 March 2026 at 3:00 p.m. Mountain Standard Time.** All questions regarding this acquisition must be submitted by Wednesday, 11 March 2026 at 3:00 p.m. MST to allow for the Government to assess the inquiry and provide a response. Quote Submission Information: Submit quote to Contract Specialist, Rachel Wright, via email to rachel.wright.9@us.af.mil and to Contract Officer, Melissa Huston, at melissa.huston.2@us.af.mil. Please submit questions via the email above. Please provide the Company CAGE Code or Unique Entity Identifier (UEI) when submitting a quote. Attachment: 1. Instron 68TM-5 Product Description 2. Single Source Justification Instron 68TM-5 FOB: Destination Delivery Schedule: 90 Days ARO Pricing Validity Date: 90 Days validity from proposal, if this validity date is not practicable, vendors must propose an alternate realistic date for evaluation/award processing within proposal. **Please refer to document FA822726Q5576 Combined Synopsis Solicitation for a list of anticipated clauses for this effort**
