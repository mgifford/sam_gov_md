# Leased Space Furniture (Brand Name JSI)

- Agency: DEPT OF DEFENSE
- Type: Combined Synopsis/Solicitation
- Posted: 2026-03-06 16:06:18.729-05

## Summary

The contractor shall procure, deliver, and install JSI brand furniture for the United States Transportation Command (USTRANSCOM)/TCCS-FM. The scope of this project includes all labor, equipment, and materials required to install new desks, lateral files, overheads, tack boards, chairs, and bridge components that match the existing furniture at a leased facility located at 1222 Spruce St, St Louis, MO. Please see RFQ, Attachment 1, Furniture Item List/Drawing and Attachment 2, Statement of Work for more information.
