# Runway Approach Lighting and Airfield Lighting Vault

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-04-06 09:01:07.99-04

## Summary

The work includes new airfield improvement consisting of thefollowing but not limited to: New SALS with REILs approach lighting systemto serve runway 26, new Runway Distance Remaining (RDR) signage, PrecisionApproach Path Indicator (PAPI) to serve runway 26, and a new airfieldlighting vault with emergency backup generator, including demolition ofexisting lighting vault.
