# ADVANCED NOTICE OF ON-RAMP

- Agency: DEPT OF DEFENSE
- Type: Special Notice
- Posted: 2026-03-27 12:10:01.381-04
- Solicitation Number: ADVANCED_NOTICE_OF_ON-RAMP

## Summary

See attached Advance Notice. The Naval Information Warfare Systems Command (NAVWAR) is planning to add more suppliers to its existing Blanket Purchase Agreement (BPA) for a tactical data link modernization program. The program requires ongoing production, spares, and engineering services. The primary goal of this on-ramp is to increase the number of qualified vendors, which will foster greater competition for future orders under the BPA. This is a qualification process, meaning any company that meets the standards will be added to the vendor pool. Access to Controlled Documents: In order to obtain access to the full Advanced Notice, all interested vendors must meet the following requirements (1) Possess a current and properly scoped DD 2345 on file with the Defense Logistics Information Services (DLIS). If the company does not have a current and properly scoped DD 2345 on file and cannot provide a Joint Certification Program (JCP) number, please visit DLIS at http://www.dlis.dla.mil/jcp/ for more information on how to complete and submit the DD 2345; and (2) Provide a need-to-know certification, your facility clearance, and a mailing address to the designated Point of Contact (POC), the Contracting Officer?s Representative (COR). E-mail: PMW-150-Contracts@us.navy.mil.
