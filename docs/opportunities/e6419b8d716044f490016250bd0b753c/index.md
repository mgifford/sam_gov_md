## Summary

This solicitation is for the competitive procurement of the VXP Engine Vibration Accessory Sets, P/N: 4086AS0215-1 and NSN: 4920-01-727-7637 in support of the KC-130J Program. It is anticipated that this solicitation will result in a single award, Firm Fixed-Price (FFP) type contract.
