# Parkrimp 2 Crimper System

- Agency: DEPT OF DEFENSE
- Type: Combined Synopsis/Solicitation
- Posted: 2026-04-13 09:38:00.189-04
- Solicitation Number: W911S226U3003

## Summary

Amended to change NAICS code from 423830 to 332322 An or equal item must meet the following salient characteristics to be considered acceptable, Must be a complete hydraulic hose crimper system including a stand-mounted crimper and a 230/460 VAC, 3-Phase electric power unit. The crimper system must be fully compatible with the Governments existing inventory of Parker-manufactured Parkrimp series dies without modification. Must include accessories equivalent to the Parker Adapter Bowl, Spacer Ring, and Spacer Plate to accommodate the full range of approved hose and die combinations.
