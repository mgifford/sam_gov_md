IS UNIQUELY QUALIFIED TO PROVIDE THIS SOFTWARE TO THE NATIONAL GEODETIC SURVEY BASED UPON THE FOLLOWING: THIS SOFTWARE AND LICENSE IS PROPRIETARY AND NO OTHER COMPANY HAS THE SOURCE CODE TO PROVIDE THIS PRODUCT, LICENSES, OR MAINTENANCE SERVICES. GEOCUE GROUP, INC. IS THE SOLE PROVIDER OF THE GEOCUE SOFTWARE. GEOCUE IS UNIQUE IN ITS ABILITY TO MANAGE WORKFLOWS, ALLOW FOR THE IMPORT OF LARGE LIDAR DATA SETS FOR VISUALIZATION, EDITING, AND ANALYSIS, AND, APPLY LIDAR STRIP AND CALIBRATION ERROR ADJUSTMENTS. THIS CONTRACT IS NECESSARY TO AVOID A BREAK IN SERVICE FOR THIS CRITICAL MISSION REQUIREMENT. GEOCUE GROUP POSSESSES THE UNIQUE CAPABILITY TO IMPORT LARGE LIDAR DATA SETS FOR VISUALIZATION, EDITING, AND ANALYSIS REQUIRED TO PERFORM THIS REQUIREMENT. GEOCUE HAS NO AUTHORIZED PARTNERS OR RESELLERS WHO CAN PROVIDE THE SOFTWARE AND LICENSES. ***THIS PRESOLICITATION NOTICE IS NOT A REQUEST FOR COMPETITIVE PROPOSALS AND NO SOLICITATION PACKAGE IS AVAILABLE.
