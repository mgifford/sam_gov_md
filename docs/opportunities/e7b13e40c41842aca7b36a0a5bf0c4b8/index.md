# Brand Name - National Resource Directory (NRD) Renewal

- Agency: DEPT OF DEFENSE
- Type: Justification
- Posted: 2026-03-09 17:09:14.122-04
- Solicitation Number: FY25-0121

## Summary

The purpose of this action is to procure brand name Amazon Web Services (AWS) to provide Federal Risk and Authorization Management Program (FedRAMP) compliant, cloud-based hosting of the National Resource Directory (NRD). Services include 24/7 publicly accessible internet access to the NRD, remote access for system administrators, and ongoing security monitoring to include detection of and response to incidents, outages, and exploits. The Contractor shall provide management support for all software servers (database, web servers, application enablers), operating systems, hypervisors, hardware, and network infrastructure that are included in the Contractor's boundary and will specifically describe the Consumer/Provider responsibilities for maintenance. The Contractor shall provide web hosting infrastructure and application integrations support. The Contractor shall provide Operations Management and operations of the environment to be inclusive of all the above components and their respective integration.
