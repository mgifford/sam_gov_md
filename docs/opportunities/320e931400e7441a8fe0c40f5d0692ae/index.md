# GPU Compute Servies for Network of Optimal Dynamic Energy Signatures

- Agency: DEPT OF DEFENSE
- Type: Combined Synopsis/Solicitation
- Posted: 2026-04-17 16:25:55.129-04
- Solicitation Number: HR001126QE003

## Summary

The Defense Advanced Research Projects Agency is seeking a dedicated, managed high-performance computing (HPC) solution that includes compute hardware, network-attached storage, high-speed networking, robust security controls, and comprehensive user support. The solution must meet the performance, availability, and security standards detailed herein to support the Network of Optimal Dynamic Energy Signatures (NODES) program's scientific objectives. The NODES program seeks to enable distributed, data-driven biological discovery by integrating experimental platforms, large-scale biological datasets, and advanced computational methods. NODES performers generate and analyze substantial volumes of heterogeneous biological data and rely on computational workflows to enable rapid hypothesis generation, model development, and validation across a networked ecosystem.
