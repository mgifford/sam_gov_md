# Maintenance plan and extended warranty for iCAP PRO XP Rad/Duo Spectrometer and CETAC ASXpress Plus system

## Summary

This is a combined synopsis/solicitation for commercial items prepared in accordance with the format in FAR Part 12. This announcement constitutes the only solicitation. Quotations are being requested and a separate written solicitation will not be issued. Solicitation number 1232SA26Q0308 is issued as a Request for Quotation (RFQ) for an extended warranty for our existing iCAP PRO XP Rad/Duo and DPQ-00809108, with CETAC ASXpress Plus. The requirement is for a base year and up to four option years. This acquisition is not set-aside for small business concerns. The applicable North American Industry Classification Standard Code is 811210. The small business size standard is $34 million. This acquisition is an unrestricted requirement. All responsible sources may submit a quotation which will be considered by the agency.
