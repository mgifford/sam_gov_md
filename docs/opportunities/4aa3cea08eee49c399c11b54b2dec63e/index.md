# ATS ELAT Salmon Tracking Tags - KBAO

- Agency: INTERIOR, DEPARTMENT OF THE
- Type: Combined Synopsis/Solicitation
- Posted: 2026-03-04 16:25:36.172-05
- Solicitation Number: 140R2026Q0029

## Summary

The purpose of this requirement is to acquire 4 activation boxes and 320 JSATS-compatible Eel Lamprey Acoustic Tags (ELAT) for tracking juvenile Chinook salmon migration and survival in the Klamath River.
