# FY26 Georgetown Harbor OM Navigation Dredging

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-04-21 14:37:46.573-04
- Solicitation Number: W912HP26BA004

## Summary

DESCRIPTION OF WORK: The work consists of maintenance dredging approximately 390,000 cubic yards on the BASE contract and 170,000 cubic yards of OPTION work of unclassified material from the Georgetown Harbor which shall be deposited in the Sampit Placement Area. The work is located in Georgetown County, South Carolina. The work includes complying with the requirements of the Endangered Species Act and Marine Mammals Protection Act. This solicitation is hereby posted as on 7 April 2026. The purpose of this amendment 0001 is to provide a volume table with volumes for the BASE and OPTION CLINs. Due to the large file size of the Sediment Sampling and Analysis Report, Appendices A-F, the documents will be distributed via DoDSafe. Please request access from Sonja Carter, Project Manager, at Sonja.l.carter@usace.army.mil. This Amendment is hereby posted as on 21 April 2026.
