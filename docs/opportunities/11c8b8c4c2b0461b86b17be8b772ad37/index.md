# 100 LRS Wheel and Tire Containers

- Agency: DEPT OF DEFENSE
- Type: Combined Synopsis/Solicitation
- Posted: 2026-04-23 06:06:27.143-04
- Solicitation Number: FA558726Q0017

## Summary

Solicitation for the purchase of C-130 Main and Nose Landing Gear Tire (MNLGT) and KC135 Main Landing Gear Tire (MLGT) Containers for RAF Mildenhall UK. These containers must be designed in accordance with MIL-STD-1791 and be compatible with the 463L pallet system.
