# Sources Sought Notice N61331-26-SN-Q19 For Advanced Synthetic Aperture Sonar Imaging eNgine (ASASIN) Software Modifications

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-03-04 16:36:26.255-05
- Solicitation Number: N6331-26-SN-Q19

## Summary

Sources Sought Notice N61331-26-SN-Q19For Advanced Synthetic Aperture Sonar Imaging eNgine (ASASIN) Software ModificationsThe purpose of this Sources Sought notice is to comply with the Defense Federal Acquisition Regulation Supplement (DFARS) 206.302-1(d) posting requirement. This is NOT a request for proposals.The Naval Surface Warfare Center Panama City Division (NSWC PCD) intends to award a sole source, firm-fixed-price contract to Pennsylvania State University Applied Research Laboratories (PSU ARL) (CAGE Code: 7A720) for modification to their proprietary Advanced Synthetic Aperture Sonar Imaging eNgine (ASASIN) software.
