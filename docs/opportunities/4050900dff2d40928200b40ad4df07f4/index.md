# Collaborative Public Safety Solution

- Agency: JUSTICE, DEPARTMENT OF
- Type: Sources Sought
- Posted: 2026-03-11 13:33:19.551-04
- Solicitation Number: CJIS31126

## Summary

The CJIS Collaborative Public Safety Solution RFI is intends to improve information sharing, operational coordination, and decision?support across federal, state, local, tribal, and territorial public safety partners. The solution aims to provide secure, CJIS?compliant data exchange, integrated workflows, and modernized capabilities that enhance situational awareness, case coordination, and mission effectiveness. The FBI is seeking industry input on technical approaches, integration capabilities, and cloud?based services that can support this multi?agency public safety environment. *Full copy of the RFI with instructions are attached to this posting.
