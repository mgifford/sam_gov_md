# Request for Quotation for Supply and Installation of a Fully Operational CCTV System for U.S. Embassy Phnom Penh

- Agency: STATE, DEPARTMENT OF
- Type: Combined Synopsis/Solicitation
- Posted: 2026-04-24 05:29:20.106-04
- Solicitation Number: 19CB6026Q0009

## Summary

U.S. Embassy Phnom Penh has a requirement for a vendor to supply and install a “Fully Operational CCTV System” for U.S. Embassy Phnom Penh. The replacement solution provided by the vendor must be high quality and reliable, while meeting or exceeding the current power requirements of the existing system in both function and capacity. Please see attached Request for Quotation.
