## Summary

The Contractor shall provide program engineering and IPT management services directly to the NIWC Pacific NEN program office and its procurement and installation efforts. The Contractor shall perform program planning, engineering and integration, project management (PM), and technical analysis. (1) Purpose of this Amendment: This amendment is issued to advise all interested parties that the original Notice ID for this Sources Sought, N66001-26-R-3503, was inadvertently duplicated. This Notice ID is no longer active for this requirement. (2) New Official Notice ID: This requirement has been re-issued under a new, official Notice ID: N66001-26-R-3506. (3) Action Required: All interested parties are hereby directed to search for and reference Notice ID N66001-26-R-3506 on SAM.gov (via PIEE) for all information, official documents, and submission instructions related to the Naval Enterprise Networks (NEN) IPT support requirement. (4) Documentation Status: The new notice, N66001-26-R-3506, contains the official Market Survey document and a consolidated Question and Answer (QA) attachment. All future updates and documents will be posted exclusively to the new notice. (5) Closing Date: The official closing date and time for responses is specified in the new notice, N66001-26-R-3506. (6) This notice (N66001-26-R-3503) is superseded and should no longer be used.
