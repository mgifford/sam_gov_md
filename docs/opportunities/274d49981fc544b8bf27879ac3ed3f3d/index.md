## Summary

This is a request for information ONLY. This is not a request for quote/proposal and there is no solicitation package available at this time. The 375 Contracting Squadron, Scott Air Force Base, IL is seeking sources to provide Spill Response Services in accordance with the draft Performance Work Statement (PWS). Service to include Emergency Oil and Hazardous Substance/Hazardous Waste (OHS) spill response, sampling and cleanup services for the installation and resulting from operations of SAFB that take place off installation within 25 miles.
