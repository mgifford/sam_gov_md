# Install Infantry Moving Targets, Range 206, Grafenwoehr Training Area

- Agency: DEPT OF DEFENSE
- Type: Presolicitation
- Posted: 2026-03-10 08:16:31.323-04
- Solicitation Number: W912PB26BA003

## Summary

1. The project includes the following work: Install prefabricated concrete elements for all new target pits. Create cable trenches with data, low voltage wiring, empty conduit system and grounding conductor. Install 5 ea new raised infantry firing positions. Install 20 ea new stationary target pits. Install protective wooden planks on prefabricated concrete parts. Create protective embankments for all newly installed target pits. Create new graved areas, up to 3 meters wide, behind the new target positions. Create 6 ea new lane markers in the range depth 600 m. Demolish 9 old stationary personnel target pits. Test and dispose of concrete- and asphalt demolished material. 2. The solicitation will result in a Firm Fixed Price contract. The magnitude of the project is between $500,000 - $1,000,000. 3. The performance period is 300 calendar days, whereby a commencement time of 20 calendar days is included. 4. The solicitation will be sent to you on or about 31 March 2026 with an anticipated bid opening date of approximately 28 April 2026. Paper copies will not be available. 5. All responsible sources may submit a bid, which will be considered by the RCO Bavaria. 6. Work will be performed in Germany and contractor must comply with German law (e.g. authorization to do business in Germany).
