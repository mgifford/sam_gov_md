# FY26 Studio Camera Upgrade

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-03-17 15:45:55.058-04
- Solicitation Number: FA3300ABWPA26

## Summary

The 42d Contracting Squadron, Maxwell AFB, AL, is requesting information from industry to procure brand name Sony ILME-FR7K, or equal, studio camera systems for the 42d Air Base Wing, Public Affairs (PA) office. This requirement involves unique characteristics necessary for compatibility with the end-user's existing equipment inventory. Information gathered from this notice will be used to finalize the acquisition strategy for the requirement detailed in the attached draft Statement of Requirements (SOR).
