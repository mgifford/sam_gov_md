# Dyess Lanes Walk-In Refrigerator

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-04-15 09:31:58.909-04
- Solicitation Number: FA466126Q0028

## Summary

The requirement is for the procurement and installation of a custom outdoor walk-in refrigeration system consisting of a combination freezer, cooler, and dry storage unit. This will include Refrigeration systems (multiple BTU-rated units), doors, partitions, lighting, rain canopy, and anchors. The installation process will consist of the removal/disposal of existing unit, full system install, refrigerant charging, startup, and calibration.
