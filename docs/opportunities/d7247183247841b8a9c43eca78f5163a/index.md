# Airshow 2026 Tents, Tables, Chairs Rental

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-03-30 16:11:12.038-04
- Solicitation Number: FA4897QC039

## Summary

Mountain Home Air Force Base has a requirement for the rental of tents, tables and chairs. Please see the attached Solicitation, Statement of Work, and Wage Determination. ATTENTION: Please review Amendment #1 to this solicitation, issued by the 366th Contracting Squadron, Mountain Home AFB. This amendment revises the required table size from (10' x 30") to (8' x 30"). As a result, the required quantity of (8' x 30") tables has increased from twelve (12) to eighty-seven (87). For details, please see the revised documents "COMBO - Tents Tables Chairs Rental - Amednment 1" and the Performance Work Statement, "PWS - Tents_Tables_Chairs Rental - 12 MAR 26 - Amendment 1," in the attachments.
