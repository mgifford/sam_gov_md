# Port Fourchon Navigation Channel, Maintenance Dredging FY26, Lafourche Parish, LA

- Agency: DEPT OF DEFENSE

- Posted: 2026-04-07 10:29:23.669-04

## Summary

PROJECT TITLE: Port Fourchon Navigation Channel Maintenance Dredging FY26 PROJECT LOCATION: Lafourche Parish, LA This solicitation is issued as unrestricted. The NAICS code for this procurement is 237990. The magnitude of construction is between $10,000,000 and $25,000,000. The work consists of the removal and satisfactory disposal of all material excavated from the Port Fourchon Navigation Channel, within the limits shown on the contract drawings; with disposal of dredged material on the gulf beach, west of the west jetty and/or east of the east jetty. A bid opening date is to be set by amendment.
