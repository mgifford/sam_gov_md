This is a Sources Sought for a brand name new Chlorinator Cell replacement for the water purification system at the Walla Walla VA Medical Center in Walla Walla, WA. The brand name parts required are as follows: Flow Cell Module DEPOLOX 5 C, non-pressuriized version .
