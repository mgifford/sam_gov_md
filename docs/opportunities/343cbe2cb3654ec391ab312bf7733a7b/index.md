# Fire Department Emergency Command Vehicle Upfitting

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-03-11 16:01:02.856-04
- Solicitation Number: W50S8926QA010

## Summary

Nebraska Air National Guard is requesting quotes to fully upfit a new 2025 Ford F-450 Dually into a mission-ready Fire Department Command Vehicle. The project includes installation of multi-color emergency lighting, siren and control systems, custom center console and computer mount, radio and antenna integration, exterior warning lighting, a retractable bed cover, gooseneck hitch, storage solutions, external charging port, reflective fire-department graphics, and other NFPA- and FAA-compliant components. See attached Requirements Description for full details
