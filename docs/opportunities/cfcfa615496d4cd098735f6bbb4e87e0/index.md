# MSS Deployable Shelter

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-03-10 15:46:49.987-04
- Solicitation Number: M67854-26-I-5108

## Summary

This Sources Sought/RFI notice is issued by the United States Marine Corps (USMC), Marine Corps Systems Command (MCSC), Program Manager Combat Support Systems (PM CSS), Product Manager Maintenance and Support Systems (PdM MSS). PdM MSS is currently exploring the market for a proven, 20-foot Convention for Safe Containers (CSC) certified, International Organization for Standardization (ISO) format structures capable of Temporary/Tactical Sensitive Compartmented Information Facility (T-SCIF) and Temporary/Tactical Special Access Program Facility (T-SAPF) accreditation (i.e. has proven instances of receiving temporary accreditation in accordance with Intelligence Community Directive- ICD-705 v1.5.1 and DONSAPCO 0459-23 compliant shelters in an operational and/or training environment).
