## Summary

See attachment for AMD 0001 that answers (1) question received. --------------------------------- Statement of Work Nintex Automation Migration Services

1.0 Background: Strategic Weapons Facility Pacific (SWFPAC) is migrating forms/workflow services from legacy Microsoft Access database products to the new product Nintex Automation (aka K2). SWFPAC currently supports a legacy Microsoft Access database identified as mission critical and is in need of immediate migration. SWFPAC requires professional assistance to accelerate the migration in order to meet the deadline while ensuring required data integrity, database access, and solution operability.

2.0 Scope: Rebuilding of SWFPAC's existing Microsoft Access solution in Nintex Automation consists of a series of lookup tables that feeds the Transactions, SL-3 Gains/Transfers and Form 10576 forms, which are the core functionality of the solution. SWFPAC will provide exported copies of all specifications and data structures for the forms and workflows from the existing solution. SWFPAC will submit the exports via uploading to the vendors secure file storage. The vendor will send back the completed products via the same secure file storage.

3.0 Deliverable: The contractor shall provide up to 223 hours of consultation/architect services to complete the migration of the solution, including the following solution items: Solution Item Qty. SmartObjects 36 SQL Stored Procedures 4 Lookup Table Forms 12 Lookup Table Approval Workflow 1 Additional Approval Workflows 11 SL-3 Gains / Transfers Form 1 SL-3 Gains / Transfers Approval Workflow 1 Transactions Form 1 Transactions Approval Workflow 1 10576 Form 1 10576 Form - Print PDF 1 N6471026Q0004 Page 4 of 37 Auditing for SQL tables 1 Solution Architecture Documentation 1 Knowledge Transfer 1

4.0 Place of Performance: Vendor will work remotely on their own equipment and with their own software.

5.0 Period of Performance: All work must be completed no later than June 5, 2026.

6.0 Security: All data exported from the SWFPAC environment will be scrubbed of any Controlled Unclassified Information (CUI) by SWFPAC personnel.
