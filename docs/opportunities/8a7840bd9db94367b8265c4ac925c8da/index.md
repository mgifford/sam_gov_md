# Interactive Slideshow Software with Formative Assessment

- Agency: DEPT OF DEFENSE
- Type: Award Notice
- Posted: 2026-04-21 13:36:04.315-04
- Solicitation Number: HE125426DE012
- Awardee: PEAR DECK, INC. El Segundo CA 90245 USA
- Award Amount: 529600.60

## Summary

DoWEA requires interactive slide software with formative assessment functionality that includes (1) an age-appropriate lesson bank or templates and (2) the ability to make customized lesson for use by K-12 students and teachers systemwide. As shown in the table above, the estimated number of K-12 student licenses needed is about 61,325. The number of school-level and above-school-level educators is approximately 12,000. DoWEA also requires related virtual product training for teachers and related technical support, maintenance, and upgrades.
