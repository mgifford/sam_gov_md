## Summary

Travis AFB is seeking 720A Kelvin Varley Voltage Divider. This item is replacement for an asset required on CA/CRL R14. The FLUKE 720A Kelvin Varley Voltage Divider was an initial special procurement in 2020 to be fielded to base level PMEL's to enable input voltages to be divided to within 0.1 part per million.
