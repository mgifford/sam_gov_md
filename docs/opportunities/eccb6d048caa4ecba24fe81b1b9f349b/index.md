# Project Manager Intelligence Systems & Analytics (PM IS&A) Army Intelligence Data Platform- Next (AIDP-Next) Request for Information #3

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-03-09 12:41:28.168-04
- Solicitation Number: ACCAPGAIDPRFI3

## Summary

Army Intelligence Data Platform - Next (AIDP-Next): Notice/Request for Information 3 Army Contracting Command - Aberdeen Proving Ground (ACC-APG) in support of the Program Manager (PM) Intelligence Systems and Analytics (IS&A) Aberdeen Proving Ground (APG), is conducting market research to shape the acquisition strategy and technical options for the delivery of the next iteration of AIDP-Next, to include capabilities to support echelon based solutions that include services, processing and data management capabilities. The army is considering multiple acquisition approaches, including commercial item procurements and other transaction authorities (OTA), and encourages industry feedback on the most effective strategy to deliver AIDP Cloud to Edge capabilities. One-on-One request response date: 13 March 2026 at 1700 EST Questions and/or feedback response date: 24 March 2026 at 1700 EST The purpose of this RFI is to: Post Industry Day slides and answers to questions Post Industry Partnering information Post instructions on how to request a one-on-one meeting with the Government – Vendors who have previously submitted a request will need to submit a new request following the instructions in this RFI. Post instructions on how to obtain the Draft Call for Solution (CfS).* Post an Excel document to be used by vendors in submitting questions. *NOTE 1: The Draft CfS contains CUI and must be requested using the instructions in the attached. Failure to follow the instructions will result in a request rejection. NOTE 2: The Government intends to use an accelerated award process. Industry is hereby notified that the final CfS will include a response deadline much closer to ten to fourteen days. FULL RFI IN ATTACHMENT SECTION.
