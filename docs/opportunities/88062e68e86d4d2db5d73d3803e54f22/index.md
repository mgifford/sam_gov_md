# Award Synopsis of the National Industrial Security Program Operating Manual (NISPOM) Compliance Upgrade Delivery Order under Raytheon Corporate BOA

- Agency: DEPT OF DEFENSE
- Type: Award Notice
- Posted: 2026-03-12 09:08:49.854-04
- Solicitation Number: N00019-25-RFPREQ-TPM265-0041
- Awardee: RAYTHEON COMPANY El Segundo CA 90245 USA
- Award Amount: 2589211.00

## Summary

The Naval Air Systems Command (NAVAIR) has awarded a Cost Plus Fixed Fee Delivery Order under Basic Ordering Agreement (BOA) N00019-20-G-0007 to The Raytheon Company. This task order is for the procurement of the tasks required to upgrade APG-79 radar test stations to be compliant with the latest National Industrial Security Program Operating Manuel (NISPOM) requirements. In order to remain NISPOM compliance, four test stations received upgrades to Windows 10. This contract was procured non-competitively pursuant to 10 U.S.C 2304(c)(1). Raytheon is the sole designer, developer, and integrator of the AN/APG-65/-65(V)/-73/-79(V)1/-79(V)4 Radar Systems. Raytheon, as the original equipment manufacturer (OEM) is the sole designer, developer, manufacturer, and supplier of the F/A-18 and EA-18G radar systems and is the only source with the knowledge, expertise, and on-site personnel necessary to accomplish this effort within the timeframe required. This synopsis is for informational purposes only.
