# Portable Toilets and Showers

- Agency: DEPT OF DEFENSE
- Type: Presolicitation
- Posted: 2026-03-23 08:22:52.018-04
- Solicitation Number: FA560626Q0005

## Summary

Spangdahlem AB is anticipating the solicitation of the below contract. Be advised, this is NOT a solicitation, nor does it guarantee a solicitation for this requirement to be posted in the future. This is an official pre-solicitation, this is anticipated to be a firm-fixed-price contract. This is only a pre-solicitation to notify prospective offerors. Feedback is encouraged. Quotes are not currently being requested. The purpose of this contract is to provide portable showers and latrines. The contractor shall provide portable showers, latrines, and all necessary maintenance and cleaning services to include gray water removal for personnel at Spangdahlem AB, Germany.
