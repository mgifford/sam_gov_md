# DWA Turbine Guide Bearing Pumps

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-03-18 13:46:18.83-04
- Solicitation Number: W912EF26RSS16

## Summary

The U.S. Army Corps of Engineers, Walla Walla District is seeking sources for a supply requirement for a base contract of a firm fixed price coming up for Dworshak Dam and Reservoir of Turbine Guide Pumps in Ahsahka, ID. The North American Industry Classification System (NAICS) code for this project is anticipated to be 333914 (Measuring, Dispensing, and Other Pumping Equipment Manufacturing) and the associated small business size standard is 750 number of employees. Please see attached Performance Work Statement for detailed description of requirements. THIS IS NOT A SOLICITATION. This source sought announcement is a tool to identify businesses with the capability to accomplish the work. Only those firms who respond to this announcement by submitting the following information will be used in determining whether to set aside this requirement. SUBMISSION REQUIREMENTS FOR RESPONSES TO THIS SOURCES SOUGHT SYNOPSIS: 1) A capability statement expressing interest in this requirement, describing your company and its capability to complete a project of this scope, magnitude, and complexity. Include a statement detailing any special qualifications or certifications, applicable to this project’s scope, held by your firm and/or in-house personnel. 2) Name of firm and point of contact. 3) State your business size (large or small business) under above NAICS Code. 4) UEI (Unique Entity Identifier) and CAGE Code from SAM.gov. 5) Can you meet the delivery date in the technical specs? If not, please suggest a realistic delivery date or if the technical specs need to be adjusted. Please do not respond to this notice unless you will be responding to the solicitation. Respond to this notice by emailing the following information to tonya.c.french@usace.army.mil.
