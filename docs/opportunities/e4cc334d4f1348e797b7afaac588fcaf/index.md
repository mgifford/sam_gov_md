Project Title: Tatanka Hotshots Base Fire Alarm Upgrades, Custer, South Dakota Description – This is a combined synopsis/solicitation for commercial items prepared in accordance with the format in FAR Part 12. This announcement constitutes the only solicitation; offers are being requested, and a separate written solicitation will not be issued. Solicitation 1240LT26Q0036 is issued as a Request for Quotation (RFQ) for supplying all design, labor, materials, equipment, miscellaneous items, freight and delivery for the removal and replacement of the fire alarm system including fire alarm panel, pull stations, detection devices, notification devices, and supervisory devices; provide associated ceiling tile removal and reinstallation, patching, painting, electrical, and communication cabling. Work also includes providing for on-going fire alarm monitoring. The Tatanka Hotshots Base hosts two buildings which were both constructed in 2002: The Tatanka Hotshots Quarters is a 6,300 gross square foot (gsf) single level building.
