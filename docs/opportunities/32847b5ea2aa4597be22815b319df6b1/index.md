# Car Wash Rack System Cannon AFB

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-03-30 12:00:22.358-04
- Solicitation Number: FA4855Reag0001

## Summary

This sources sought is being extended for the opportunity to gain more information from industry. I have attached pictures of the current car wash and equipment room due to many requests. This sources sought will most likely be updated again in the following weeks with some answers to questions received. Cannon AFB, NM requires Installation of a new Government-Owned Vehicle (GOV) wash rack system at Building 438. There is already a standing system in place that needs to be replaced. The system must include all necessary pumps, equipment, and controls to service three (3) interior wash bays and one (1) exterior sprayer for oversized vehicles. The project includes removal of the old, deteriorating system and full installation and testing of the new equipment. This sources sought is for market research purposes only and does not constitute the government to enter into a contractual agreement.
