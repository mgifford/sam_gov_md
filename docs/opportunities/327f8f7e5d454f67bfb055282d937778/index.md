# Offutt AFB FY26 Air Show: TBM Avenger for Static Display

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-04-09 11:01:40.93-04

## Summary

Description of Requirement: The 55th Contracting Squadron has an ongoing requirement to acquire a TBM Avenger aircraft for static display at the 2026 Offutt AFB "Defenders of Freedom" Air and Space Show.
