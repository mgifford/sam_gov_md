# 66--Xylem SonTek RS5 and Surfbee Flow Scout flow measurement system

- Solicitation Number: 140R2026Q0070

## Summary

THIS IS NOT A REQUEST FOR QUOTE (RFQ) AND NO SOLICITATION IS AVAILABLE The US Bureau of Reclamation (USBR), California - Great Basin Region, intends to negotiate a purchase order on a sole source basis with Xylem, Inc. 4828 Parkway Plaza BLVD # 200, Charlotte, NC 28217-1957 for a Xylem SonTek RS5 and Surfbee Flow Scout flow measurement system under the authority of FAR 13.106-1. Surfbee Flow Seeker for Sontek RS5 with lemo plug. Including inflatable hull, pump, patch kit, thrusters, handheld remote tablet, and ASV controller. Including multiport hardware connections to ADCP, Sontek M9 mount, adapter and multiport cable for Sontek RS5, mast with external GNSS, and FPV camera. RS5 portable five-beam

3.0 MHz acoustic Doppler current profiling/discharge measurement system intended for use from moving boats and other platforms in shallow channels. System is provided in a 2 inch (5 cm) diameter Delrin housing and consists of

3.0 MHz velocity measurement transducers in a 4-beam Janus configuration,

3.0 MHz vertical acoustic beam for depth measurement, compass/2-axis tilt sensor, temperature sensor, internal Li-Ion rechargeable battery and Bluetooth telemetry link. Features bottom tracking, RSQ Windows software for real-time display of current profiles, water depth, and computed discharge measurements. Includes 3-ft (91-cm) radio antenna cable, USB Bluetooth radio for connecting to customer-supplied computer, (2) Li-Ion rechargeable batteries and a USB battery charger.
