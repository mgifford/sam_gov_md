# Umatilla National Forest - North Zone Spring Tree Planting

- Agency: AGRICULTURE, DEPARTMENT OF
- Type: Award Notice
- Posted: 2026-04-15 18:00:21.902-04
- Solicitation Number: 1240BE26P0038
- Awardee: THREE STRIPES CONTRACTING LLC Medford OR 97501 USA
- Award Amount: 28864.00

## Summary

This contract solicitation is to secure services for coniferous tree planting of Government furnished tree seedlings by hand tool (hoedad) method on 451 acres of National Forest Lands. Conifer tree species to be planted include Western Larch (LAOC) and Ponderosa Pine (PIPO). There is only one stock type that will be planted under this contract; 160-plugs. Planting will occur across an elevational gradient of approximately 1,000ft – 1,500ft. The Contractor shall furnish all supplies and materials (except those designated as Governmentfurnished), equipment, tools, labor, supervision, transportation, and incidentals, to perform all work specified herein. LOCATION: Umatilla National Forest, Pomeroy and Walla Walla Ranger Districts. Project area is located in the Blue Mountains with some activity units located in Asotin County, Washington, and some activity units located in Umatilla and Union County, Oregon. See the attached project vicinity map and unit maps that are included as part of this contract.
