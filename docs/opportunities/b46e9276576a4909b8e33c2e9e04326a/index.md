Background and Requirement: El Paso VA Health Care System Cardiology Service requires an Endoscopy PillCam system to provide this alternate procedure which is non-invasive and is the viable alternative to full Endoscopy with anesthesia needs. This miniature camera is embedded in a capsule that can be swallowed and images saved and retrieved by data communication to an external sensor unit. This system must be capable of Full Interface with the VA architecture for the Electronic Medical record imaging section (CPRS/VISTA). The PillCams proposed data kit must be able to capture images as it moves through the digestive tract for retrieval and viewing. Must have a workstation in the clinic for uploading data and subsequent viewing by the GI Specialist to view and decipher any pathology from the detailed images. Workstation must allow the specialist to be able to select and embed chosen images into a formatted final report PDF that can be uploaded into the VA EMR. System must be patient friendly so that they are able to easily swallow the pill camera and understand/follow instructions given at time of clinic visit by the specialized nursing staff. System must be capable of saving and maintaining raw data images for duration of five years. This equipment must be able to perform the following functions for all the SPECIFIC INSTRUMENT REQUIRED: El Paso VA Medical Health Care System, Endoscopy PillCam system with the following: Endoscopy PillCam system.
