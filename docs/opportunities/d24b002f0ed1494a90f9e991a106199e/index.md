# CCSFS High Factory Assembled UL Level 3 Ballistic Rated Welded Steel Booth

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-03-30 09:55:55.405-04
- Solicitation Number: 2026-PZI-1718

## Summary

Request for Information/Sources Sought for Contractors that would be able to fulfill the requirement below. Three (3) 4’ x 6’ x 8’ High Factory Assembled UL Level 3 Ballistic Rated Welded Steel Booths mounted to three (3) trailers. Standard size hitch. Three (3) Ballistic Sliding Windows (one for each booth) Standard Size. One (1) Optional Stamped Drawing (the specifications of the trailer). The contractor will pre-wire the receptacles, lights, switch, HVAC (air condition/heat), and supply load center attached to the building. The tie-in and Data/Communications will be handled by SFS; not contractor. IAW SOW.
