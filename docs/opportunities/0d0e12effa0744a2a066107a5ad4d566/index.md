# SOURCES SOUGHT: Removal of Underground Storage Tank at Launch Facility E-10 (FE Warren AFB, WY)

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-03-06 17:07:24.117-05

## Summary

This is for Sources Sought Only for interested Vendors regarding the removal of an underground storage tank (TK 101) at Launch Facility Location Echo-10. This location is in Laramie County in Wyoming. Offeror shall be registered in SAM and has a basic NIST SP 800-171 Assessment or CMMC Level 2 at a minimum in order to receive Controlled Unclassified Nuclear Information. See attached Statement of Work dated 17 Feb 2026.
