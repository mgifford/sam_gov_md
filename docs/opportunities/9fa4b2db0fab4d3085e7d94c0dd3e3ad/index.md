# Rip Saw for DDCN - SP3300-26-Q-0079

- Agency: DEPT OF DEFENSE
- Type: Combined Synopsis/Solicitation
- Posted: 2026-03-19 12:28:47.486-04
- Solicitation Number: SP3300-26-Q-0079

## Summary

This Request for Quotation (RFQ) is being issued to establish a firm-fixed-price (FFP) purchase order for a rip saw at DLA Distribution Cherry Point, NC (DDCN). The Government intends to award one (1) contract as a result of this RFQ and will be issued on a Standard Form (SF) 1449. This acquisition is not a Defense Priorities and Allocations System (DPAS) rated procurement. This acquisition is being solicited as a Small Business Set-Aside. The North American Industry Classification System (NAICS) code for this project is 333243 and the size standard is 550 employees. The Product or Service Code (PSC) for this acquisition is 3220. Please see attached CSS for full details.
