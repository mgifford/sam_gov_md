# Boat Motor & Rigging

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-04-20 15:16:08.943-04
- Solicitation Number: FA252126QB033

## Summary

The objective is to procure and install one (1) new 250 horsepower (HP) outboard engine with a 25-inch shaft on a 2019 23' Lake Assault Aluminum Patrol boat. This includes the engine, all associated rigging components/electrical, and the labor to de-rig the old engine and install the new one on a government-provided vessel.
