# Car Wash Rack System Cannon AFB

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-03-23 10:38:42.568-04
- Solicitation Number: FA4855Reag0001

## Summary

Cannon AFB, NM requires Installation of a new Government-Owned Vehicle (GOV) wash rack system at Building 438. There is already a standing system in place that needs to be replaced. The system must include all necessary pumps, equipment, and controls to service three (3) interior wash bays and one (1) exterior sprayer for oversized vehicles. The project includes removal of the old, deteriorating system and full installation and testing of the new equipment. This sources sought is for market research purposes only and does not constitute the government to enter into a contractual agreement.
