# Costworks

- Agency: DEPT OF DEFENSE
- Type: Combined Synopsis/Solicitation
- Posted: 2026-04-15 16:02:26.067-04

## Summary

The Anniston Army Depot is requesting the renewal of license support for Costworks software that Anniston Army Depot (ANAD) uses in order for Directorate of Public Works (DPW) to formulate costs for certain projects; i.e. electrical, plumbing, road construction, and mechanical. The contractor shall provide seven (7) Network Users for the thirteen (13) software renewals, which includes access to the RSMeans Costworks 2026.This procurement is sole sourced to: R.S.
