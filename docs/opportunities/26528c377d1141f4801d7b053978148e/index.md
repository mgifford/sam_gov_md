# Security Systems Maintenance at Fort Bliss, Texas

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-04-20 16:27:23.88-04
- Solicitation Number: W56ZLW-26-Q-0009

## Summary

Questions due by 0900 CDT Thursday, 23 April 2026. This is a non-personal services contract to provide comprehensive maintenance and repair for the existing security cameras (Avigilon) and access control systems (DAQ Electronics) at two locations on Fort Bliss, TX. Key requirements include scheduled preventive maintenance, unscheduled on-call repairs, and a 24/7 emergency response capability with a mandatory 4-hour on-site response time for critical failures. Please see attachments for full details.
