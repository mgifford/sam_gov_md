# CNC Milling Machine

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-03-25 07:50:47.418-04
- Solicitation Number: FA9101CNC_Milling_Machine

## Summary

This is a Sources Sought notice for a new, fully functional CNC Milling Machine for Arnold Engineering Development Complex (AEDC) at Arnold Air Force Base, TN. The machine must be factory assembled, fully enclosed, and include an automatic tool changer, 4th Axis, and compatibility with Inventor software. Delivery and on-site installation at AEDC are required. For full technical specifications, please refer to the attached Technical Specification Sheet. The NAICS code is 333517 and the PSC is 3417. This is for market research only and not a solicitation.
