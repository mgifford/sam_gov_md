# New Building IT Design and Build

- Agency: NATIONAL SCIENCE FOUNDATION
- Type: Sources Sought
- Posted: 2026-03-12 09:52:00.322-04
- Solicitation Number: 49100426K0003

## Summary

This is a combined Sources Sought/Request for Information. NSF is reducing the scope of its on-premises enterprise IT infrastructure to allow, combined with Building Automation System, IT Asset Management, and Configuration Management Database (CMDB) tools, to provide visibility into all computer spaces operated by the NSF. Existing IT Infrastructure topography includes a Co-location Data Center, managed by Equinix located in Ashburn, VA and an Enhanced Communication Equipment Room, located in the NSF Headquarters in Alexandria, VA. The key component of this topography is the Amazon Web Services, Direct Connect, tied to the Equinix data center supporting NSF’s multi-cloud environment.
