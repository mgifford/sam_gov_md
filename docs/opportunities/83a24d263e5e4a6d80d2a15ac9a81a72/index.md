# DDNV Terex Repair - CSS SP3300-26-Q-0076

- Agency: DEPT OF DEFENSE
- Type: Combined Synopsis/Solicitation
- Posted: 2026-03-11 16:21:11.535-04
- Solicitation Number: SP3300-26-Q-0076

## Summary

This Request for Quotation (RFQ) is being issued to establish a firm-fixed-price (FFP) purchase order for the repair of a Terex reach stacker at DLA Distribution Norfolk, VA (DDNV). The Government intends to award one (1) contract as a result of this RFQ and will be issued on a Standard Form (SF) 1449. This acquisition is not a Defense Priorities and Allocations System (DPAS) rated procurement. Please see attached CSS for full details. Amendment 0001 has been posted revising FAR 52.212-1 & FAR 52.212-2.
