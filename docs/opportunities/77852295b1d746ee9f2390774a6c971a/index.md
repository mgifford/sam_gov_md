## Summary

*DESTINATION FOB – FORT BRAGG, NORTH CAROLINA * Test Instrumentation sensors for data recording that measure impact shock and vibration forces exerted during Aerial Delivery (AD). This is a combined synopsis/solicitation for commercial supplies. Please see Purchase Description for additional items and quantity. New Items/Equipment ONLY. Subject line in email MUST contain following format: SHOCK RECORDS - MICC-FH-043246 - (COMPANY NAME). Please provide each item priced individually and total cost. Delivery of items will be invoiced separately, please provide separate line item for delivery (if necessary). Military Installation Access is required. Must be able to provide proper paperwork and identification to gain access.
