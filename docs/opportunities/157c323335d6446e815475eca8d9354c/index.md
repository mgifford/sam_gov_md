# RFI0001-26 CDL Training Program

- Agency: JUSTICE, DEPARTMENT OF
- Type: Sources Sought
- Posted: 2026-03-04 13:20:49.724-05
- Solicitation Number: RFI0001-26

## Summary

This is a request for information for sources that are able to provide CDL training and certifcaitons for inmate population within UNICOR, Federal Prison Industries. All information in this request is considered confidential and shall not be used by the Respondent for any reason other than responding to this request. This is a Request for Information (RFI) and your response is NOT an offer. The results of this Request for Information (RFI) will be used to develop an Acquisitions Strategy, AND THIS RFI WILL NOT RESULT IN A CONTRACT OF ANY TYPE. This request does not obligate UNICOR OFG to pay any cost that any Respondent incurs in the preparation of its response. All cost associated with the preparation of a response to this request will be borne solely by the Respondent. All submitted responses shall become the property of UNICOR OFG and will not be returned. UNICOR has multiple locations throughout the US that would be in need of this requirement. UNICOR is looking for sources that can provide associated work within the attached statement of work. Interested vendors need to provide the information provided in the attached document. Deadline for sources sought response is March 17, 2026 at 10:00 AM EST.
