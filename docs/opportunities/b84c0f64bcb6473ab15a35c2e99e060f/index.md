# Notice of Contract Action: Mobility Air Force Tactical Data Link (MAF-TDL)

- Agency: DEPT OF DEFENSE
- Type: Special Notice
- Posted: 2026-04-16 17:01:40.671-04

## Summary

The Command, Control, Communications Network (C3N) located at Hanscom Air Force Base, MA has a requirement for the continued sustainment of the current Mobility Air Force, Tactical Data Link (MAF-TDL) system. This requirement is for Operational Support (OS) including Program Management, Depot Support, Field Support, and Cyber Security.
