## Summary

The purpose of this contract is to support NSWCPD Code 333 with the requisite EMC and EMI related EQT activities under its cognizance. The Contractor shall support NSWCPD Code 333 in execution of laboratory testing, test procedure delivery, test report delivery, and test fixture fabrication in contractor facilities in support of ship hardening efforts.
