# CFCD STK Annual License

- Agency: DEPT OF DEFENSE
- Type: Special Notice
- Posted: 2026-03-26 11:03:25.957-04

## Summary

STK provides capabilities to modify satellite mission requirements, design and analyze trajectories, conduct space operations, and perform trade studies using advanced optimization algorithms. It supports integration of real?time data from on?orbit constellations, enables simulation of electro?optical and infrared sensor systems, and facilitates planning and evaluation of multidomain concepts of operation across space, air, and terrestrial assets within a unified mission environment.
