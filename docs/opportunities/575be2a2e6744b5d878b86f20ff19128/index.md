## Summary

This is a construction requirement for the VAVs in building 1122 which have restricted air flow due to clogged coils and airway passages. The coils shall be cleaned to achieve original design airflow specifications. Period of Performance: Not to exceed 60 Calendar Days after Notice to Proceed issuance. Competition: This requirement is a 100% Small Business Set Aside for North American Industry Classification System (NAICS) code 238220.
