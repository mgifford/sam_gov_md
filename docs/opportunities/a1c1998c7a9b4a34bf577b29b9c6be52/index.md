# Flags, Guidons, Poles and Stands

- Agency: DEPT OF DEFENSE
- Type: Combined Synopsis/Solicitation
- Posted: 2026-03-19 16:05:49.276-04
- Solicitation Number: W911S226U2768

## Summary

White Sands Missile Range requires the following: US Flags, territorial flags, branch flags, guidon pole sets and flag stands. Flags are all intended for indoor use with a size 3 x 5. Attached is full product descriptions and quantities. They are all Nylon printed flags 100% made in the USA. All flags are single reverse except the Presidential flag which is double sided. Single Reverse: this is standard for all flags made in the USA. Front side reads correctly and the back side reads in reverse. Double sided: two flags sewn back-to-back with a liner. Both front and back reads correctly. Guidon pole with round bottom shall be wood. Need by date: Soon as possible from date of award, no longer than 30 day ARO.The amendment 0001 is to add one (1) presidential flag line item 014 and provide salient characteristics to line 015. In addition, the product description has been updated to reflect all changes, and the response date has been pushed forward.
