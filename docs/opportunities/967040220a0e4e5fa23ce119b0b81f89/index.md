# BRSE Production and Integration of Hardware - RFI

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-03-09 12:48:30.892-04
- Solicitation Number: NAVWAR_HQCA_2025_A_022

## Summary

The Naval Information Warfare Systems Command (NAVWAR), on behalf of Program Executive Office Command, Control, Communications, Computers and Intelligence (PEO C4I), Communications and Global Positioning System Navigation Program Office (PMW/A 170) seeks information from interested vendors for the production and integration of Battle Force Tactical Network (BFTN) Resilient Command, Control and Communications System Enhancement (BRSE) hardware starting in Fiscal Year 2027 (FY27). Request for Information (RFI) AMEDNMENT 0001 issued to extend the RFI response date and provide submission instructions. Please refer to RFI Amend 0001 attached. RFI UPDATE 0002 issued to publish responses to RFI questions received. Please refer to NAVWAR_HQCA_2026_A_022_BRSE RFI QandA_24Feb2026 attached. RFI UPDATE 0003 issued to publish response to an RFI question received. Please refer to NAVWAR_HQCA_2026_A_022_BRSE RFI QandA_2_09Mar2026 attached.
