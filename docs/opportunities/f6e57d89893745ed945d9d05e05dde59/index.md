# Mess Attendant (KP) Services

## Summary

This is a combined synopsis/solicitation is for Mess Attendant (KP) services required by the 140th Force Support Squadron at Buckley Space Force Base in Colorado. Offerors must review all attached documents and adhere to all instructions. Failure to do so will result in the quote being deemed non-responsive. This is a combined synopsis/solicitation for the 140th Force Support Squadron at Buckley Space Force Base in Colorado. They are requesting KP services (kitchen and janitorial services) for the base dining facility. Solicitation number W50S6V26R0002 is being issued as a REQUEST FOR PROPOSAL (RFP). This announcement constitutes the only Solicitation notice; proposals are being requested and the written solicitation is incorporated below. The North American Industry Classification System (NAICS) code is 561720 - Janitorial Services and the corresponding small business size standard is $22 Million. The FSC/PSC associated with it is S201 – Housekeeping-Custodial Janitorial. The Government intends to issue this as a firm fixed price (FFP), base plus four option years contract for KP Services as set forth in the attached Performance Work Statement. Please reference solicitation W50S6V26R0002 on all correspondence. The Government will utilize simplified acquisition procedures in accordance with FAR Part 13. There will be a site visit scheduled and available for vendors, this will be the only site visit completed. In order to attend please follow the instructions provided and return all documents by the requested deadline.
