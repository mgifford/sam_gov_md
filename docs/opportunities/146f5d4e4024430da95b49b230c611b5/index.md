# EV32, NIROP Fridley, Site 5, O M

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-04-01 10:32:44.601-04
- Solicitation Number: N4008526R0103

## Summary

The goal of this Contract Task Order (CTO) is to allow for the completion of tasks necessary to the Navy Environmental Restoration Program (NERP) at former Naval Industrial Reserve Ordnance Plant (NIROP) Groundwater Treatment Facility (GWTF) located at 54 Northern Stacks Drive in Fridley, Minnesota (the former location of NIROP Fridley). These tasks are ongoing and support the management, implementation, and execution of the NERP at NIROP Fridley. This is a 8(a) set aside for MACC Holders Only
