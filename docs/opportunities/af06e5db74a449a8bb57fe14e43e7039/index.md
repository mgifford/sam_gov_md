## Summary

THIS IS A SOLE SOURCE REQUIREMENT. Questions deadline: Please submit all questions regarding this requirement via email to audrey.lee.3@us.af.mil before June 2nd, 1700 MT, 2025. Please CC Christa.Phillips.1@us.af.mil. RFQ Submission deadline: : Please submit quote via email to audrey.lee.3@us.af.mil before June 19th, 1700 MT, 2025. Please CC Christa.Phillips.1@us.af.mil. This requirement is for the procurement of the CEESIM-VPX, 2 channel 4 port 2-40 GHz 10U chassis emitter generation rack used with in the F-16 Block 50 Software Integration Lab (SIL) in their efforts to support Operational Flight Program (OFP) testing for new Electronic Warfare (EW) systems. The CEESIM-VPX provides the radio frequency (RF) simulation of multiple, simultaneous emitters linked to static/dynamic platforms required to simulate true combat conditions. Please view the solicitation documents for more details.
