## Summary

Fleet Logistics Center Puget Sound (FLCPS) intends to solicit for direct replacement OEM Renk Bearing Shell Sets. This requirement is sole sourced to SS/Brand Name Entity due to COMSCINST

4490.1 MSC N7 guidance on Main Diesel Engine Generators.
