# Helena-Lewis and Clark National Forest Gipsy Lake Day Use Improvements

- Agency: AGRICULTURE, DEPARTMENT OF
- Type: Combined Synopsis/Solicitation
- Posted: 2026-03-06 10:57:49.157-05
- Solicitation Number: 1240LU26Q0028

## Summary

Amendment 01 Questions and Answers document added and extend quotes due date to 03/13/2026. Description: This project consists of constructing new parking and living areas at the Gipsy Lake Day Use site on the Townsend Ranger District. Construction of each new site will require excavation and grading work that will balance cut and fill for a natural fit into the landscape. Each site will include all new furnishings as shown in the drawings. BASE BID – SITE 1 Construct a double wide parking area with two living areas. Remove 4 large trees. Lop and scatter as shown on drawings for erosion control. OPTION 01 – SITE 2 Construct a single wide parking area with one living area. Install 2 new culverts. Construct a new ditch. OPTION 02 – SITE 3 Construct a single wide parking area with one living area. Install barrier rocks (government furnished).
