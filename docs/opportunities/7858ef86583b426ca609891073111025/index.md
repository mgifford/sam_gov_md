## Summary

This Request for Information (RFI) is issued solely for information and market research purposes. It does not constitute a Request for Proposal (RFP) or a promise to issue a RFP in the future. The USG seeks industry input to refine the attached Appendix A for Shuttle Transportation of Soldiers and address any industry concerns. Interested vendors: please provide a Rough Order of Magnitude to help gauge potential costs. Please see attached Appendix A for details. This request is for multiple CONUS locations: Gordon - Benning - Bragg - Meade - Carson - Irwin - Riley - Stewart - Huachuca - Lee/JBLE - Hood - Campbell - Drum - Bliss - PoM - West Point.
