# Gravel Purchase, Hiawatha NF

- Agency: AGRICULTURE, DEPARTMENT OF
- Type: Combined Synopsis/Solicitation
- Posted: 2026-04-13 17:26:40.061-04
- Solicitation Number: 12444626Q0026

## Summary

4/13 - Amendment 0002 has been issued which provides the attached QAs document. 4/9 - Amendment 0001 has been issued which replaces the original Attachment D Submission Package with Attachment D Submission Package 2. Two occurrences of the word "janitorial" have been removed. The solicitation is not extended. Description - This project consists of furnishing all material, equipment & trucking, labor including wages, supervision, transportation, weight slips, MDOT 23A gravel, 3” gravel and misc. materials necessary to complete aggregate hauling and chain spreading/unloading and unloading/pile placement as stated in the specifications and attachments. This is a combined synopsis/solicitation for commercial items prepared in accordance with the format in FAR Part 12. This announcement constitutes the only solicitation. Quotations are being requested and a separate written solicitation will not be issued. Solicitation number 12444626Q0026 is issued as a Request for Quotation (RFQ). This acquisition is set-aside for small business concerns. The applicable North American Industry Classification Standard Code is 212321 The small business size standard is 500 employees. This acquisition is a Total Small Business Set-Aside. All responsible sources may submit a quotation which will be considered by the agency.
