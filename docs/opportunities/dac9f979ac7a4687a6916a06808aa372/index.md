## Summary

The Presolicitation for this Request for Proposal was published under 36C24826Q0351. The number has been changed to 36C24826R0085. PROJECT TITLE: Design Build (DB) Install CT Scanner (LDCT), Project No. 673-CSI-131 PROJECT SUMMARY: The contractor shall provide all labor, tools, and materials necessary for the following work: Design-Build project to support the installation of a new GE Revolution Ascend (CT LDCT) system. This project is classified as a new equipment installation and will require both design and construction services to bring the IR (LDCT) suite into compliance with current VA CFM Technical Information Library (TIL) standards. Project is located at the James A. Haley Veteran’s Hospital, 13000 Bruce B. Downs Boulevard, Tampa, FL 33612. Period of performance is 281 days. This is a 2-phase Design Build procurement. There will be NO site visit for Phase 1. Phase 1 RFIs are due 03/11/26 at 10:00 AM Phase 1 Proposals are due 03/23/26 at 10:00 AM
