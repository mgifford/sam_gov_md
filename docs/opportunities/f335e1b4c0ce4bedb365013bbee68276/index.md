# RFI CHPP Industrial Hygiene

- Agency: DEPT OF DEFENSE
- Type: Special Notice
- Posted: 2026-03-17 14:31:17.271-04

## Summary

This Request for Information (RFI) is for information gathering and planning purposes only and does not consitute a solicitation or request for proposal. Eielson AFB is seeking industry input on the attached Performance Work Statement (PWS) for Industrial Hygiene for a coal powered Centreal Heating Power Plant. This is an industrial setting requiring work in high areas that may require fall protection. Be mindful that the work will be completed within a coal plant that contains heavy build up. This RFI will close on 24 March 2026 at 12:00pm AKT.
