## Summary

This is a Pre Solicitation Notice only. This is NOT a Request for Proposal, Request for Quote or Invitation for Bid. DLA Troop Support is informing the public that a RFP to supply NPR, DRAWBAR, TELESCO NSN 1740-01-662-1432, which is made IAW Military Drawings. DLA Troop Support is researching the possibility of a solicitation for the referenced items under a Firm Fixed type contract. Support of this item will be based on DLA Direct Deliveries, where the material would be shipped to the New Cumberland, PA. DLA stock distribution location. A quantity 120 each will be procured at this time. Interested contractors should paid attention to the DIBBS website for the RFP which should be released on or before March 24, 2026. This Pre Solicitation Notice is issued for the purpose of information only and the Government is in no way obligated to purchase any material as a result of this inquiry. The Government will not pay for any information submitted by respondents in response to this Pre Solicitation Notice. If any solicitation or solicitations would result, it/they would by synopsized on the FedBizOpps website or DLA Internet Bid Board System (DIBBS) at https://www.dibbs.bsm.dla.mil/ and it would be the offeror’s responsibility to monitor these websites for any such notices.
