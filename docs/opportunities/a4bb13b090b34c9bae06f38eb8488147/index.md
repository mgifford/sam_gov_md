# GYM FLOORING

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-03-06 13:31:17.179-05
- Solicitation Number: N0040626QS228

## Summary

THIS IS A FIRM FIXED-PRICE SUPPLY TYPE CONTRACT NAVSUP Fleet Logistics Center Puget Sound is negotiating a SOLE SOURCE requirement for the procurement of OEM SOF SURFACES INC. duraSOUND rubber acoustic tile gym flooring. This is a noncompetitive contract action lacking competition as SOF SURFACES INC. is the only source available per FAR 12.102(a). An offeror and its product must meet a qualification requirement in order to be eligible for award. The office from which the qualification requirement may be obtained is Commander, Naval Sea Systems Command (SEA 05P3). SOF SURFACES INC. is the only source for the duraSOUND rubber acoustic tile gym flooring product which meets fire performance criteria set forth in MIL-STD-1623 and is suitable gym flooring for this requirement. All responsible sources may submit a quotation which will be considered by the agency. The anticipated award date is March 18, 2026.? See the attached solicitation for requirements. Closes on 11 March 2026 at 11:00 AM PST. All questions regarding this solicitation should be submitted to Roger Bowers, roger.s.bowers.civ@us.navy.mil.
