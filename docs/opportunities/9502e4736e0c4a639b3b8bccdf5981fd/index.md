# AG Davis Positioning Tables

- Agency: DEPT OF DEFENSE
- Type: Special Notice
- Posted: 2026-04-16 11:39:17.942-04
- Solicitation Number: PANRSA-26-P-0000-034754

## Summary

The U.S. Army TMDE Activity (USATA) has a requirement to implement a brand name purchase of eight (8) AG Davis Positioning Tables for angle measurements. The index table will replace the use of sine plates and gage blocks for angle generation providing higher accuracy, improved efficiency, and lower turnaround time. The improvement in accuracy will eliminate the need to evacuate equipment to the Army Primary Standards Laboratory (APSL) as well. The index tables shall include the following specifications or equivalent (if not brand name):

1.0 Requirements: Model: CD12-ALMS-720 Index Tables Qty: 8

2.0 Delivery Requirements Shipping Address: U.S. Army TMDE Activity, Attn: John Means, Building 5435, Redstone Arsenal, AL 35898
