# FY26 Laundry Fire Suppression System (FCI Safford)

- Agency: JUSTICE, DEPARTMENT OF
- Type: Presolicitation
- Posted: 2026-03-11 14:35:24.02-04
- Solicitation Number: 15B60926Q00000001

## Summary

This is a Presolicitation Notice issued in accordance with (IAW) Revolutionary FAR Overhaul (RFO) FAR 5.1 and 12.201-1(c)(2). THIS NOTICE IS NOT A REQUEST FOR COMPETITIVE PROPOSALS NOR QUOTATIONS. The Government intends to solicit this requirement directly from contractors IAW RFO FAR 12.201-1(c)(2) for firm-fixed price contract for the scope of this requirement which includes the design, construction, and installation of wet fire sprinkler suppression system fully compatible with the existing Edwards EST3 Fire Alarm System. Additionally, it will include on-site training session for designated personnel and warranty. Anticipated award date is around April 20, 2026 (subject to change). Agency: Department of Justice/Federal Bureau of Prisons Office: 1529 West Highway 366, Safford, AZ, 85546 Set-aside: N/A NAICS Code: 238220 Description: A single award contract to a qualified and licensed contractor capable of providing all the necessary equipment, labor, materials, and incidentals to Install Fire Suppression System compatible with the existing Edwards EST3 System at Institutional Laundry Building located in Federal Correctional Institution (FCI) Safford, Arizona. Note: This project will be performed in its entirety in the city of Safford, AZ. The successful offeror must have valid licenses to perform the work in the state of Arizona. All responsible sources may submit a quotation, bid, or proposal, as appropriate, which will be considered by the agency.
