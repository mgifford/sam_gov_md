# Z--THOMAS STONE NHS - INSTALL INTERIOR STORM WINDOWS

- Agency: INTERIOR, DEPARTMENT OF THE

## Summary

The National Park Service, Contracting Operations East, New England Major Acquisition Buying Office, issues this Request for Proposal (RFP) for a construction requirement at Thomas Stone National Historic Site in Port Tobacco, Maryland. Work will consist of furnishing all labor, supervision, materials, tools, supplies, transportation, license, certificates, permits, personal protection equipment, professional services, and equipment as required to provide and install magnetic interior storm windows as detailed in the Scope of Work, included with the solicitation documents. The estimated magnitude of the complete requirement is between $25,000 and $100,000.
