# Nuclear Surety Cross Check Analysis (NSCCA)

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-03-18 12:20:14.358-04
- Solicitation Number: FA8219-26-NSCCA

## Summary

The purpose of this Request for Information (RFI) is to gather market research information for understanding the Government’s options regarding Nuclear Surety Cross Check Analysis (NSCCA)/ Nuclear Safety Analysis and Technical Evaluation (NSATE)/ Independent Verification/Validation (IV&V) for the software for the Sentinel Weapon System. NSCCA/ NSATE is defined in AFMAN 91-119; test, simulations, and other evaluation techniques must be used to prove the safety and surety of the weapon system and for a thorough software verification process. The Government is evaluating Independent Verification/Validation options for Nuclear Surety for software for the Sentinel Weapon System.
