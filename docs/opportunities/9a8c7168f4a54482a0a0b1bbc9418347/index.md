# Ultra Lightweight Camouflage Netting System Increment 1

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-03-11 09:51:05.612-04
- Solicitation Number: M67854-26-I-5406

## Summary

This is a Sources Sought Notice / Request for Information (RFI) for the Program Manager, Combat Service Systems (PM CSS), Marine Corps Systems Command in Quantico, Virginia, for the Ultra-Lightweight Camouflage Netting System Increment 1 (ULCANS INC 1). The purpose of this sources sought is to gain knowledge of the interest, capabilities, and qualifications of various members of industry (NAICS code 315990 and PSC code 1080 apply). PM CSS is conducting Market Research regarding the industry's interest and capability to manufacture the eight variants of the next-generation ULCANS INC 1 screen system with support apparatus. Please see the attachment for additional information. *Note - On 11 March 2026, the due date for responses was extended from 17 March 2026 to 27 March 2026 to allow the government additional time to respond to questions received. The attachment was also amended to show the updated response due date.
