## Summary

ITEM: Switch Box NSN: 5930-01-705-8315 MFR PART NO: DBH672-4S MINIMUM QTY: 125 MAXIMUM QTY: 356 FOB: DESTINATION KIND OF CONTRACT: FIRM FIXED PRICE SOLE SOURCE: The subject solicitation is Sole Source to Curtiss-Wright (CAGE 1P423). The TDP will only be available to the Sole Source Manufacturer. The solicitation should be released to the web on or about 25 March 2026. See the solicitation for the actual closing date. Once available, the solicitation can be downloaded from SAM (www.sam.gov). The solicitation contains DLAD provision 52.215-9023, entitled 'Reverse Auction,' which allows for the use of reverse auction as a pricing technique. After the solicitation closes, the contracting officer will determine if a reverse auction will be held. If a reverse auction is held, contractors must be registered in the reverse auction system in order to participate in the auction. Additionally, the contractor representative participating in the auction must have taken the on-line reverse auction training prior to the auction. Registration, on-line training, and other information regarding the reverse auction can be found at https://dla.procurexinc.com. Any questions related to the reverse auction system or process can be directed to the Contract Specialist.
