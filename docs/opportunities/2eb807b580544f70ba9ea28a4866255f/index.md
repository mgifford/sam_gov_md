## Summary

This is a solicitation for commercial items prepared in accordance with (IAW) the format in Federal Acquisition Regulation (FAR) subpart 12.201 as supplemented with additional information included in this notice. This announcement constitutes as the only solicitation; quotations are being requested and a written solicitation will not be issued. Solicitation number 70Z03826QL0000113 is issued as a Request for Quotation (RFQ). This solicitation document and incorporated provisions and clauses are those in effect as of November 28, 2025, IAW the Revolutionary FAR Overhaul (RFO). The applicable North American Industry Classification Standard Code is 488190. The small business size standard is $40 million. This is a restricted solicitation. (CAGE Code: 1L5F5) as a result of this solicitation for the repair of the following item(s): SCHEDULE OF SUPPLIES Line Item 1 Nomenclature: Cockpit Tactical Display National Stock Number (NSN): 1560-01-HS3-0673 Part Number (P/N): PM-10.4-MFD (Manufacturer CAGE Code: 1L5F5) Quantity: 9 Each Sources must be able to obtain the required technical expertise, engineering data, and Original Equipment Manufacturers (OEM), Neuro Logic Systems, Inc.
