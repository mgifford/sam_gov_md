# Airspace Common Operating Picture

- Agency: DEPT OF DEFENSE
- Type: Presolicitation
- Posted: 2026-03-23 11:36:16.315-04
- Solicitation Number: W900KK-26-S-ACOP

## Summary

Request for Information (RFI) Airspace Common Operating Picture The U.S. Army Contracting Command - Orlando (ACC-Orlando), intends to issue a Call for Solution to solicit innovative commercial products and technologies for the Airspace Common Operating Picture (A-COP) system. This solicitation will be conducted in accordance with the Army Open Solicitation procedures detailed within Solicitation W9128Z-25-S-A002, which was posted to SAM.gov on 01 May 2025. The government anticipates awarding a single, fixed-price contract as a result of this Call for Solutions. The Government is seeking your feedback on the attached Draft Call for Solutions. Interested offerors should submit a concise response (maximum 2-page white paper or 5 slides) within 10 calendar days of this draft’s posting. Please address only the following five questions: Question 1: If this is posted as an actual Call for Solutions, will you plan to respond and submit a solution? If so, please provide your business name, size, and point of contact. Question 2: What updates do you recommend to the draft Call for Solutions to assist you in deciding to respond and compete for the contract award? Question 3: Will this requirement enable you to provide an innovative solution? If not, what changes would you recommend to ensure that an innovative solution is provided? Question 4: Please confirm whether PSC code 7H20 is appropriate for this requirement. If not, please recommend a more suitable PSC. Question 5: Please confirm whether NAICS code 541512 is appropriate for this requirement. If not, please recommend a more suitable NAICS code. Submit RFI responses via e-mail not Later than the closing date (See Response Date) of this RFI to the following POCs: Roberto J.
