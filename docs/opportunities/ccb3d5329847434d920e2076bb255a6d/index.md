# 125 STS Human Performance Program

- Agency: DEPT OF DEFENSE
- Type: Presolicitation
- Posted: 2026-04-22 17:23:18.429-04
- Solicitation Number: W50S8Y26RA010

## Summary

The Oregon Air National Guard requires the services for Human Performance Program (HPP) supporting the 125 Special Tactics Squadron (STS). Contractors shall ensure operators are physically and mentally optimized for mission readiness and sustained peak performance, while minimizing injury and maximizing return-to-duty efficiency. This includes the following services: Physical Therapy/Injury Management, Athletic Training, Strength and Conditioning, and Cognitive Performance.A formal solicitation shall be posted at a later time to include evaluation criteria.
