# Request for Information - F-15 Flow Control Valve Depot Activation Capabilities

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-04-17 12:08:02.543-04
- Solicitation Number: FA8634-22-F-0035-02

## Summary

This request for information focuses on the provision of the above-referenced Depot Activation capabilities for the F-15 EX’s current EX FLOW CONTROL VALVE. The repair and maintenance of the FLOW CONTROL VALVE requires not only the FLOW CONTROL VALVE itself but all associated parts/spares and the rights in technical data necessary to maintain and ensure the software used in the displays functions correctly to support verification of repairs at designated ALC.
