# Security Maintenance

- Agency: DEPT OF DEFENSE
- Type: Presolicitation
- Posted: 2026-03-31 12:32:10.227-04
- Solicitation Number: N6883626QS046

## Summary

The Naval Supply (NAVSUP) Systems Command, Fleet Logistics Center Jacksonville (FLCJ), FL intends to issue an award off FAR part 12 procedures for a 100% small business set-aside. The anticipated award date will be 10 April 2026. The requirement is for security equipment maintenance in Jacksonville, FL. The applicable North American Industrial Classification System Code is 811310 with a size standard of $12.5 million.This notice does not constitute a request for proposals, request for quote or an invitation to bid. All sources eligible to provide the requirement must respond in writing. Responses must be supported with clear and convincing evidence to clearly articulate the ability to provide the requirement as outlines above. A request for documentation will not be considered as an affirmative response.
