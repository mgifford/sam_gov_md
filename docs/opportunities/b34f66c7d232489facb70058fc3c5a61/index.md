## Summary

A contract is awarded without Competitition. This commercial service acquisition is for a follow-on contract for an annual enterprise license and specialized technical support services to BYDS, LLC (formerly One Network Enterprises, Inc). The customized enterprise license covers access to BYDS's commercial off the shelf (COTS) software product, their Department of Defense Core (DoDC) platform, and FLITES Software Development Kit (SDK). This includes software maintenance updates for policy, compliance, security, and enabling functionality of FLITES requirements. The Technical Services are for unique and highly specialized services specifically related to intricacies and system configuration requirements of the BYDS product, DoDC and FLITES SDK. This critical service is required to support the FLITES Integrated Program office (IPO) with the configuration, development, integration, testing, and deployment of the FLITES application, and to operate and sustain FLITES in a Government approved cloud environment. The FLITES solution is developed by and with BYDS's product software as the underlying platform. The Government is the System Integrator (SI) and is responsible for the configuration and development of the FLITES application using FLITES SDK with support from BYDS. The technical service will perform all different types of services to include maintenance, repair/bug fixes, enhancements, development work, integration, testing, and problem solving to include within the FLITES SDK. The Techniclal support services develops solutions, repairs, updates, bug fixes and development in BYDS product and DoDC as necessary to enable seamless communication and integration to support the FLITES SDK. This is a three (3)- layer system to include BYDS product, DoDC, and FLITES SDK application. Only BYDS has expertise in all three layers of FLITES to know which layer the maintenance, repair, or development is necessary to meet Government requirements. In order for FLITES to be fully functional, it requires integration and interfaces between the FLITES SDK with the BYDS product and DoDC. While customization can be done at the application layer using the FLITES SDK, certain functions, corrections, or fixes that must be performed on the BYDS product or DoDC that enable the FLITES application functionality.
