# INSTALL BACK UP GENERATOR AND ATS FOR NMCI ROOM, BLDG 127, DAM NECK ANNEX

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-03-17 13:48:07.387-04
- Solicitation Number: N4008525R0072

## Summary

This project will provide backup power to the NMCI equipment room IT server racks and HVAC in Room S145 of B127 Taylor Hall, NAS Oceana Dam Neck Annex via a diesel stand-by generator and automatic transfer switch (ATS). To minimize the scale of the HVAC infrastructure requiring backup power, a computer room air conditioning (CRAC) unit will be installed to cool the NMCI equipment room and powered via the backup power system. ONLY CONTRACTORS ON THIS MACC GROUP CAN SUBMIT A PROPOSAL.
