# Repair MAF Roofs

- Agency: DEPT OF DEFENSE
- Type: Award Notice
- Posted: 2026-04-21 17:30:01.771-04
- Solicitation Number: FA4626-26-C-0004
- Awardee: JAXON CONSTRUCTION INC Bixby OK 74008 USA
- Award Amount: 489322.38

## Summary

Work consists of the demolition of the existing roofing system including asphalt shingles, underlayment, ice and water shield, gutters, soffit and fascia, and all other flashings trim and fasteners associated with the existing system. Existing dormers will also be demolished with roof decking repair. Installation of the new roofing system includes new shingle system as specified, new soffit and fascia, and new gutters and downspouts.
