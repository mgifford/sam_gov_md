## Summary

This is a combined synopsis/solicitation for commercial products prepared in accordance with the format in FAR Subpart 12.6 as supplemented with additional information included in this notice. This announcement constitutes the only solicitation; quotes are being requested and a written solicitation will not be issued. The solicitation number is W50S77-26-Q-0009.
