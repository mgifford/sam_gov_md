# Construction of Wheeler Sack Army Airfield (WSAAF) Lighting Upgrade Fort Drum, NY

## Summary

W912DS25RA003 - Construction of Wheeler Sack Army Airfield (WSAAF) Lighting Upgrade Fort Drum, NY Summary of Work: This construction project will install a major lighting infrastructure upgrade at the Wheeler Sack Army Airfield and will also repave a section of the main runway. The work includes construction of an airfield lighting vault, runway lighting fixtures, electrical infrastructure, concrete duct bank, utilities, HVAC, airfield paving, runway lighting and controls, site excavation, and restoration. All construction trades required. The project will need to be done in phases as mission operations must continue throughout the construction. A strong site superintendent will be required. The contract duration will be 852 calendar days for the entire project with phasing work involved. The North American Industry Classification System (NAICS) code is for general construction; 237310 for Highway, Street and Bridge Construction. This project is subject to the Project Labor Agreement (PLA) requirements of Executive Order 14063. See FAR provision 52.222-33, Alt 1, Notice of Requirement for Project Labor Agreement, and clause 52.222-34, Project Labor Agreement, for definitions and requirements pertaining to the use of a PLA for this contract. Please see the attached solicitation documents for further information.
