# REQUEST FOR INFORMATION (RFI) - Full Rate Production and Sustainment - MIBS program

- Agency: DEPT OF DEFENSE
- Type: Sources Sought

- Solicitation Number: NAVWAR_HQCA-2026-A-017-MIBS

## Summary

----3/6/2026: UPDATE: See Questions and Answers posted in the Attachment section below. ---- The MIBS program is interested in learning more about the technical solutions of Industry to support the Government?s technical objectives described in this RFI attachment. Please submit information regarding existing capabilities, Commercial Off the Shelf (COTS) or Non-Developmental Item (NDI) solutions, services, experience, and recommendations related to the delivery of a partial or full solution. The MIBS program provides resilient Integrated Broadcast Service (IBS) data feed to the Warfighter using its Joint Tactical Terminal - Maritime Evolved (JTT-ME) system. This system provides over-the-air (OTA), near-real-time (NRT) lethal threat intelligence, enabling the Warfighter to receive global, national, and theater threat and intelligence data, including Indications and Warnings of potential threats. It delivers long-term infrastructure capability to support the flexibility and delivery of future broadcast waveforms and cyber-secured IBS data feed. The JTT-ME system is comprised of a Universal Computer and Network Switch and two key components, including the KSR-4A and MX-12605/USQ. These key components are fielded, operational non-developmental items (NDI) that support Navy platforms. The RFI describes the current operational parameters for the key components. See attachment for more details and requested information. Thank you for your time and effort in advance.
