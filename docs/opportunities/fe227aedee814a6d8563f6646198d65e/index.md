# Program Depot Management (PDM)

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-04-14 11:36:05.669-04
- Solicitation Number: FA8575_F15_PDM

## Summary

The purpose of this effort is to perform scheduled Program Depot Maintenance (PDM) and related inspections as per “Work Specification (WS) for The Royal Saudi Air Force (RSAF) F-15SA PDM Model, hereinafter referred to as F-15SA PDM WS, for F-15SA conversion jets with tail numbers. The draft Performance Work Statement (PWS) (attached) describes the services required to conduct F-15SA Program Depot Maintenance (PDM) for the Royal Saudi Airforce (RSAF). These services include the tasking for the contractor to inspect provided aircraft as per the F[1]15SA PDM WS. The services to accomplish these work specifications include but are not limited to 1) Perform nondestructive inspections of assets both on and off the aircraft, 2) install assets that pass the inspections back onto the aircraft , 3) install Government Furnished Property (GFP), 4) maintain GFP calibration dates and, 5) maintain GFP serviceability.
