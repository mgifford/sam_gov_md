## Summary

9 April 2026 This RFI has been updated. See attached document "UDLM Sources Sought 26 March 2026-Revised" for updated details. To request access to view controlled documents via sam.gov. You must have a role assigned in sam.gov. This is not something Government personnel can assist you with. If you are unable to view the controlled documents, you will need to contact sam.gov for questions. After you have a role assigned in sam.gov, request to view the documents and ensure you put your company’s cage code in the request. If you do not provide the cage code, this will delay your request for access. Access will be granted to the controlled documents if an active DD2345 is on file for your company via the JCP website, https://www.dla.mil/Logistics-Operations/Services/JCP/. If a DD2345 is not found or is inactive, this will need to be updated and active before access will be granted to view controlled documents. ----------------------------------------------- This Sources Sought Synopsis (SSS) is in support of Market Research being conducted by the United States Air Force (USAF) to identify potential sources to perform KC-135R/T Depot Level Maintenance, Unscheduled Depot Level Maintenance, Modifications and Organizational Level/Intermediate Level Maintenance to supplement Organic Depot activities. This is not a Request for Proposal (RFP) or solicitation. Please see attachment for details.
