# Gravel Purchase, Hiawatha NF

- Type: Combined Synopsis/Solicitation
- Posted: 2026-04-03 13:35:36.396-04

## Summary

Description - This project consists of furnishing all material, equipment & trucking, labor including wages, supervision, transportation, weight slips, MDOT 23A gravel, 3” gravel and misc. materials necessary to complete aggregate hauling and chain spreading/unloading and unloading/pile placement as stated in the specifications and attachments. This is a combined synopsis/solicitation for commercial items prepared in accordance with the format in FAR Part 12. This announcement constitutes the only solicitation. Quotations are being requested and a separate written solicitation will not be issued. Solicitation number 12444626Q0026 is issued as a Request for Quotation (RFQ). This acquisition is set-aside for small business concerns. The applicable North American Industry Classification Standard Code is 212321 The small business size standard is 500 employees. This acquisition is a Total Small Business Set-Aside. All responsible sources may submit a quotation which will be considered by the agency.
