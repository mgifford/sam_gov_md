# Buckley SFB Rectifier Inverter System Replacement

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-03-20 13:03:58.159-04
- Solicitation Number: FA2543Rectifier

## Summary

The purpose of this Sources Sought is for market research purposes only to obtain a list of capable vendors. Please see attached Statement of Need for all requirements. Please include 2d Lt Maurice Epps email: maurice.epps@spaceforce.mil or TSgt Matthew Kennedy email: matthew.kennedy.24@us.af.mil on all communications. The replacement rectifier must meet or exceed the current amp load of the existing system of 2,800 amps (200amps x 14 units). The contractor shall install batteries equivalence or better than the existing batteries (C&D Technologies MS Endur II). The quantity of existing batteries are 3 strings with 48 jars per string. Each string shall include battery disconnects equivalent or better than the existing units (Peco II, -48v 1800 amp) The system must feed new power distribution to the D/C breaker panel to include load A,B,C,D from commercial power, battery power and generator power sources. The power distribution must meet or exceed the existing unit (Lorain Power System Model 1231v1) Power must feed into the A,B,C,D,E and F sectors of the DC breaker panel. The inverter system must accommodate the existing DC breaker panels and A/C circuits within the node facility.
