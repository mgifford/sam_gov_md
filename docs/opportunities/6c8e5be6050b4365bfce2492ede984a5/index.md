## Summary

Electrical system repair, Switchgear replacement. Supervisor's Office. Acquisition Description This is a combined synopsis/solicitation for commercial items prepared in accordance with the format in FAR Part 12. This announcement constitutes the only solicitation; offers are being requested, and a separate written solicitation will not be issued. Solicitation number 127EAW26Q0025 is issued as a Request for Quotation (RFQ) for the Electrical system repair, Switchgear replacement - Supervisor's Office project at San Bernardino National Forest, California.
