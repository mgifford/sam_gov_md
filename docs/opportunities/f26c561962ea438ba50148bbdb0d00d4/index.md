# USDA STRATUS GUS B2G Support Services

- Agency: AGRICULTURE, DEPARTMENT OF
- Type: Special Notice
- Posted: 2026-03-12 16:06:07.813-04

## Summary

Rural Development Technology Office (RD TO) Cybersecurity Division (CSD) will have administrative, technical and operational oversight of the API SME to ensure mission and business objectives are achieved while keeping security in the forefront. The API SME will be responsible for the end-to-end lifecycle of API oversight, security, ensuring robust, scalable, and secure interfaces for residential mortgage-related applications. This acquisition is to provide the Cybersecurity Division (CSD) with administrative, technical and operational oversight of the API SME but not limited to: Reviewing and validating API architectures for residential mortgage-related applications. Implementing security best practices for API authentication, encryption, and threat protection. Ensuring seamless integration with mortgage user’s interfaces and financial systems. Conducting API performance tuning, monitoring, and optimization. Collaborating with cross-functional teams, including developers, cybersecurity, and business analysts. Maintaining compliance with regulatory requirements such as NIST, FISMA, PCI, DSS, etc. Set-aside Status: Small business set-aside competition via USDA STRATUS Pool 2 BOA
