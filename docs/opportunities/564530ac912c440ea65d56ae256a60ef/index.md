# PACAF LMR Refresh

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-04-22 21:00:10.259-04
- Solicitation Number: FA521526P0008

## Summary

The is for a comprehensive tactical communications package. The core of the purchase consists of 93 units of the AN/PRC-163 Multi-channel Handheld Radio System. This is supplemented by critical software waveforms for secure and interoperable communications, including the Phantom Wraith Waveform Bundle, SATURN NATO ED 4, Wren TSM, and ARROW WF. The package also includes essential accessories for operational use, such as dismounted adapters, dual audio/EUD cables, USB cables, rechargeable batteries, and six-bay battery chargers. Or equal products will not be accepted for this requirement. Please see attachement for part number, manuffacturer, part description, and quantity needed. Questions can be directed to SrA Bristow at kyle.bristow@us.af.mil
