# UPS battery backup

- Agency: AGRICULTURE, DEPARTMENT OF
- Type: Solicitation
- Posted: 2026-03-06 16:19:12.867-05
- Solicitation Number: 12639526Q0042

## Summary

Work includes adding a UPS system to a server. Installation includes a 25kW UPS unit on one of the current 60-amp circuits for the main server rack. Equipment includes a Galaxy VS UPS 25kW 208V, 2 internal 9Ah smart modular battery strings, expandable to 4, Start-up 5x8, or equal, and Galaxy VS Maintenance Bypass cabinet, UL, Single-unit 10-40kW 208V, 20-80kW 480V, or equal. Also required is an Emergency Power Off button. Solicitation and attachment are posted below. Photos of room and equipment posted below. RFI: Would you be able to tell me what the battery runtime requirement is for this opportunity? Response: Less than 5 min, main purpose is to run between power outage and when the generator will kick on, which should be less than 30 secs.
