# Repair of I/O Discrete Fail Logic Card

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-03-26 14:01:03.712-04
- Solicitation Number: FA825126Q0002

## Summary

The U.S. Air Force requires a one-time, performance-based contract to repair the F-16 Input Output (I/O) Circuit Card (P/N 7001644-49). This card is a subassembly of the Digital Flight Controller within the F-16 Block 40/42/50/52 Flight Control System. It processes pilot inputs to send commands to the Integrated Servo Actuator amplifier and provides an interface for discrete signal processing. The contractor must provide all necessary expertise, labor, tools, and parts to repair the I/O card to a serviceable A condition within the timeframe specified in the contract. The contractor shall also maintain comprehensive test, repair, and cost accounting records for all serviced items. Requirement is restricted to qualified source(s). This is a sole source contract to Innovative Aerosystems Inc. CAGE Code 0EUW0.
