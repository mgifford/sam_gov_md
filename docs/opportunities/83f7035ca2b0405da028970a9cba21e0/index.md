# AirX Miniaturization and Prototype Development

- Agency: DEPT OF DEFENSE
- Type: Special Notice
- Posted: 2026-03-19 13:09:46.127-04
- Solicitation Number: N3239826Q1466

## Summary

Modification of the Dräger AviAir in a smaller version such that it maintains or improves upon current sensing capabilities while improving performance and incorporating additional sensing capabilities. These changes are to be made to address the outcomes of StF and airworthiness testing and to prepare the AviAir V2 for inclusion in the Navy Operator State Monitoring (OSM) system as well as the Air Force Integrated Cockpit Sensing (ICS) system.
