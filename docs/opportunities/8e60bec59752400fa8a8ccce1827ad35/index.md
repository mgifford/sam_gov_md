# Alberta Idealized Nasal Inlet (AINI), or equivalent instrument, with an Alberta Nasal Leak (or equivalent) test kit

- Agency: HEALTH AND HUMAN SERVICES, DEPARTMENT OF
- Type: Sources Sought
- Posted: 2026-04-01 15:52:00.434-04
- Solicitation Number: 75F40126SSN133298

## Summary

See attached Sources Sought Notice (SSN) for the FDA's requirement for an Alberta Idealized Nasal Inlet (AINI), or equivalent instrument, with an Alberta Nasal Leak (or equivalent) test kit. The AINI will be installed on Copley Scientific Next Generation Impactor (NGI) and Fast Screening Impactor (FSI) testing devices to be used for nasal deposition testing of spray dried powders for intranasal drug delivery.
