## Summary

PROJECT TITLE: Design Build (DB) Install CT Scanner (LDCT), Project No. 673-CSI-131 CONTRACT TYPE: Design Build (DB), Firm Fixed Price PROJECT SUMMARY: The contractor shall provide all labor, tools, and materials necessary for the following work: Design-Build project to support the installation of a new GE Revolution Ascend (CT LDCT) system. This project is classified as a new equipment installation and will require both design and construction services to bring the IR (LDCT) suite into compliance with current VA CFM Technical Information Library (TIL) standards. Project is located at the James A. Haley Veteran s Hospital, 13000 Bruce B. Downs Boulevard, Tampa, FL 33612. Period of performance is 365 days. MAGNITUDE OF CONSTRUCTION: Between $1,000,000 and $2,000,000 OFFICE ISSUING THIS REQUEST: Department of Veterans Affairs Network Contracting Office (NCO) 8 Point of Contact: Stacy Figlinski, Contracting Officer Email: Stacy.Figlinski@va.gov The Solicitation for this project will be issued on or about March 4, 2026. This project is 100% Set-Aside for Service-Disabled Veteran-Owned Small Businesses (SDVOSB). NOTE: The solicitation for this synopsis will be posted under 36C24826R0085 on 03/06/2026.
