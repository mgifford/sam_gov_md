# California Abandoned Mine Lands Agency Group (CAMLAG) Database Application Phase 3

- Agency: DEPT OF DEFENSE
- Type: Special Notice
- Posted: 2026-03-10 17:21:09.187-04

## Summary

Subject: Update to Sources Sought Notice No. W9123826S0005 - AMLDB Web Application Update This notice is to provide an update to the Sources Sought Notice No. W9123826S0005 for the project involving the update of the AMLDB Web Application and related AML GIS system components. Please be advised that the solicitation, originally anticipated for release in January 2025, has been delayed. The U.S. Army Corps of Engineers (USACE) is currently discussing the path forward for this requirement. There is no definitive date of solicitation in the near future. We appreciate the interest and responses received for this market research effort. Please continue to monitor SAM.gov for updates.
