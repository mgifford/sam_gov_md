# Sole Source Notification Security Camera Installation JBLM, WA

- Agency: DEPT OF DEFENSE
- Type: Special Notice
- Posted: 2026-04-03 11:37:23.273-04
- Solicitation Number: W50S9E26QRD02

## Summary

This is a pre-award notice of a sole source acquistion to Evergreen Fire and Alarm for the installation of security equipement, to include new cameras, in one of our facilities located on Joint Base Lewis McChord. This is an under SAT procurement using FAR part 12 proceedures. Justification for the use of the sole source is attached to this notification. This sole source is based upon the lack of competing companies certified within the local area to work on the Integrated Commercial Intrusion Detection System V used on JBLM.
