# Towed Array Integrated Product Team (TAIPT) Telemetry and Engineering Services Follow On

- Agency: DEPT OF DEFENSE

- Posted: 2026-04-20 14:56:13.699-04
- Solicitation Number: N6660425R0381

## Summary

Solicitation Number: N6660425R0381 is released as a sole source SBIR Phase III requirement to L3 Technologies, Inc. and is attached as part of this notice. The Naval Undersea Warfare Center, Division Newport (NUWCDIVNPT) has a requirement for the Towed Array Integrated Product Team (TAIPT) Telemetry and Engineering Services which supports the Naval Array Technical Support Center (known as NATSC), the towed array system In-Service Engineering Agent and other towed array production and support activities for the daily operations associated with the upgrade, production, and maintenance of the surveillance, surface, and submarine towed array components, as specified in the solicitation. This solicitation posting is related to Synopsis Notice ID N6660425R0130. Amendment 0001 was posted on 4/28/2025. The purpose was to delete two clauses that were not applicable. Amendment 0002 was posted on 6/4/2025. The purpose was to add one clause and extend the solicitation response due date. Amendment 0003 was posted on 8/1/2025. The purpose was to update Exhibit A CDRLs, Exhibit B CDRLs and Attachment 2 Performance Requirements Summary and correct a typographical error in Section L.
