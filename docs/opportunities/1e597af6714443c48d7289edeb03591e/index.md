# Gavins Point Vegetative Spraying

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-03-25 16:50:15.198-04
- Solicitation Number: W9128F26QA010

## Summary

This solicitation will be 100% set-aside for small business. Project Description: This is a non-personal services contract to provide vegetative control services at the Gavins Point Project. Services include chemical pesticide applications by various means by a licensed commercial pesticide applicator(s) to control weeds and other target vegetation on project lands for vegetative control within the designated work units, as specified in this contract.
