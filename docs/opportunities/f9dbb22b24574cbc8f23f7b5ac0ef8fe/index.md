# IT Operational Support Services IDIQ

- Agency: LIBRARY OF CONGRESS
- Type: Solicitation
- Posted: 2026-04-09 19:08:26.044-04
- Solicitation Number: 2026-CIO-0044

## Summary

Amendment 0001: The purpose of this amendment is to extend the submission deadline for Phase One to 01/05/2026 at 12:00 PM EST. Amendment 0002: The purpose of this amendment is to provide answers to questions received in response to the solicitation and update sections J, L, M. Amendment 0003: The purpose of this amendment is to update the submission date for Phase II questions and proposals, update the font size for Phase II proposals, and update Section J to provide a template to use for submitting questions. Amendment 0004: The purpose of this amendment is to update the tentative Phase II submission date and update Section L.2.6.B.1 and L.2.6.E.1. Advisory recommendation letters and Phase II Q&A are planned to be distributed by 03/06/2026. Amendment 0005: The purpose of this amendment is to update Sections J, L and M to add an initial task order for Operational Support, update Attachment J4 - Data Network Support and Attachment J5 - Initial Order Pricing Sheet, update the submission date for Phase II proposals, add a question due date for the Operational Support task order, and provide a response to Phase II questions. Amendment 0006: The purpose of this amendment is to extend the submission deadline for Phase II proposals to April 07, 2026 at 12:00 PM EDT. Responses to the Operational Support order will be posted prior to the Phase II submission deadline. Amendment 0007: The purpose of this amendment is to extend the submisison deadline for Phase II proposals to April 13, 2026 at 12:00 PM EDT. Responses to the Operational Support order is expected to be posted by April 07, 2026. Amendment 0008: The purpose of this amendment is to update the POP of the IDIQ and initial orders, update Attachment J7, extend the submission date for Phase II proposals to April 16, 2026, update the page limit for Volume V in Section L, and provide answers to questions for the Operational Support order. Amendment 0009: The purpose of this amendment is to correct a typo in the POP in Attachment J7. No other changes have been made. The Library of Congress (Library) requires contract support for the operation and maintenance of IT systems, keeping IT systems viable with supported vendor releases or off-the-shelf applications software upgrades. Operations and maintenance on IT systems will include all network, software and infrastructure associated with on-premises and cloud deployed systems and applications to include client/server, database, and web-based applications. Please review the attached solicitation (030ADV26R0008) for additional details on the requirements, how to submit a proposal, and what to include in your proposal. Please note the deadlines in Section L of the solicitation for submitting questions and proposals.
