# 149th R-11 Engine

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-04-09 13:44:13.589-04
- Solicitation Number: W50S78-26-SS-KR03

## Summary

The 149th Fighter Wing is seeking capabilities for vendors to provide an R-11 engine replacement and associated labor for installation. The place of performance will be the vendors shop. The vendors shop needs to be located within 15 miles near Lackland AFB. Vendor must provide towing capability from JBSA Lackland to vendor shop. The following specifications are required: • Make/Model: International 7400 • Manufacturer: Navistar International • Date of MFG: September 2009 • VIN: 1HTWGADT9AJ233826 • Engine Family: DT530 • 8.7-liter inline-six turbo-diesel engine • Displacement: 531 in³ • Label Number: 1887135C1 • Torque: 800 lb-ft This is a sources sought notice and not a formal request for quotes.
