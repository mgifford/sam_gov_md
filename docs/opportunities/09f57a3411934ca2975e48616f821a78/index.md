## Summary

REQUEST FOR INFORMATION (RFI) FOR CAD MODEL CONVERSION AND COMMENT ADJUDICATION SERVICES THIS IS A REQUEST FOR INFORMATION (RFI) ONLY. This RFI is issued solely for information and planning purposes – it does not constitute a Request for Proposal (RFP) or a promise to issue an RFP in the future.
