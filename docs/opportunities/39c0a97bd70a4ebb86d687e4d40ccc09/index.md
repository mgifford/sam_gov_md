# RFI Flatbed Printer (Brand Name)

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-04-16 13:52:28.429-04

## Summary

Virginia Contracting Activity (VACA) anticipates establishing a contract under North American Industrial Classification System (NAICS) Code 811310, "Heavy machinery and equipment repair and maintenance services" with a Small Business standard of $12,500,000. The NAICS Code 811310 is currently a preliminary classification. Our agency is seeking input from the industry to identify a more precise code for a Mimaki UV Flatbed Printer. VACA is issuing this notice in order to solicit capability statements from qualified sources to ensure sufficient competition exists to meet the Government's requirements.
