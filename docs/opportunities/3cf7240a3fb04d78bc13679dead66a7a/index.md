## Summary

Notice of Intent to Sole Source Requirement The Department of Veterans Affairs, Network Contracting Office (NCO) 4, intends to negotiate a sole source contract under the procedures outlined in FAR Subpart 13.5, which authorizes the use of simplified acquisition procedures for commercial products and services up to $9 Million. Specifically, FAR 13.501(a)(2) speaks to the requirement for a sole source justification. The associated NAICS code for this is 339112, Product Service Code is 6515. The vendor will be Martin AIJS 751 Lynnes CV Wadesworth, OH 44281 Description of equipment requirements: This procurement is for NuTrace R+ Scanners for the Pittsburgh VA Medical Center. A firm fixed price contract with no option years is anticipated and shall be awarded. It is the Government's belief that only the proposed candidate can meet this requirement as this is a brand name only requirement. It is the Government's intention to solicit and negotiate with only this source. This notice of intent is not a request for competitive proposals. It is NOT a solicitation or request for offers. No solicitation package is available. Any response to this notice must show clear and convincing evidence that competition would be advantageous to the Government. Responses must be received no later than 4:00 p.m. on Wednesday 4/15/2026. Olivia McDonald Contract Specialist Olivia.McDonald@va.gov See attached justification document.
