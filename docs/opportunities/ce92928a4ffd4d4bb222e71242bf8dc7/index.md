# DIL Sluice Gate Brake Replacement

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-04-24 14:36:16.775-04
- Solicitation Number: W9123726QA028

## Summary

THIS WORK CONSISTS OF FURNISHING ALL LABOR, EQUIPMENT, ACCESS, AND MATERIALS FOR REMOVING THE EXISTING SLUICE GATE BRAKE MECHANISMS (TOTAL 3) AND REPLACING WITH A NEW SYSTEM AND ONE SPARE THRUSTER SET UP MEETING THE REQUIREMENTS OF THIS SPECIFICATION AT A US ARMY CORPS OF ENGINEERS LAKE PROJECT. THE PROJECT WILL ALSO REQUIRE THE CONTRACTOR TO PROVIDE ONE (1) CRITICAL SPARE THRUSTER. THE EXISTING TWIST STYLE BRAKES ARE TO BE REPLACED WITH THRUSTER RELEASED BRAKES WITH A RIGHT-HAND DESIGN. THE EXISTING BRAKE ENCLOSURE HOUSING AND DRUMS SHALL BE REPLACED TO FACILITATE THE INSTALLATION OF THE NEW BRAKE MECHANISM.
