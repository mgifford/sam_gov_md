# GMTK Metric Tool Kits

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-04-09 13:03:29.189-04
- Solicitation Number: PAN26P3390

## Summary

This office has a requirement for six (6) metric General Mechanic Tool Kits (GMTK) to support the Army's fleet of UH72 Lakota helicopters. The 6 Metric Tool Kits requested will have a sufficient amount of tools to allow mechanics to operate on multiple aircraft at the same time and in multiple locations. These kits also contain tools that will allow the mechanics to complete phase level maintenance for future operations, especially when there is high OPTEMPO during border missions. The attached component list for a GMTK is imperial measurements. The requested purchase is for the METRIC version of these tool kits. Responses to this notice should include a capabilities statement to fulfill the requirement.
