# Antisubmarine Warfare (ASW) Target

- Agency: DEPT OF DEFENSE
- Type: Sources Sought
- Posted: 2026-03-27 16:22:38.532-04
- Solicitation Number: 85-0212

## Summary

This is a Request for Information (RFI) by the Naval Undersea Warfare Center Division Newport (NUWCDIVNPT). It does not constitute a Request for Proposal (RFP), or a commitment by the U.S. Government, nor should it be construed as such. Background: The Naval Undersea Warfare Center Division, Newport (NUWCDIVNPT), Code 85, is seeking information on a recoverable Antisubmarine Warfare (ASW) Training Target. ASW training targets provide the fleet with realistic, ASW target for open ocean and instrumented range use. This ASW Target is expected to be a MK 30 replacement. Requested Information: The Government is interested in receiving responses relating to the information attached to this announcement. Due to CUI, the information must be requested through the SAM.gov website. ATTACHMENT CONTAINS FULL REQUST FOR INFORMATION (SOURCES SOUGHT) POSTING.
