# Code 45 Lab Services SeaPort Task Order Solicitation

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-04-09 17:15:49.825-04
- Solicitation Number: N6660426R3101

## Summary

The Naval Undersea Warfare Center Division, Newport (NUWCDIVNPT), USW Platforms and Payload Integration Department (Code 45) has a requirement for engineering and technical services in support of the C45 Laboratory Complex and for the related programs which require the use the C45 Laboratory Complex. This Solicitation is for a task order under the SeaPort-NxG Multiple Award Contract. Only SeaPort-NxG MAC holders may submit a proposal.
