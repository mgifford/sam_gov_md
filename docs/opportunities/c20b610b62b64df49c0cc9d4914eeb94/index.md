## Summary

SOURCES SOUGHT NOTICE - PANNGB-26-P-0000 034832 Snow Blower Auger Components Notice Type Sources Sought Response Deadline 26 April 2026, 15:00 (Central Standard Time) NAICS Code [Insert Applicable NAICS Code] --- 1. General Scope This Sources Sought Notice (Request for Information, RFI) is issued to identify potential sources capable of providing **auger attachment components** for the Oshkosh H293BCZ-GAFC Snow Blower (SN 132460, 388143731, 3837186). The replacement parts must meet the compatibility, fit, and function of parts manufactured by **M-B Companies**. Only parts are required; the Government will perform self-installation. This is not a Request for Quote (RFQ) or solicitation for proposals, and no contract will be awarded directly as a result of this notice. --- 2. Detailed Requirements The Government requires **BRAND NAME SPECIFIC or EQUAL** components to the components manufactured by **M-B Companies** for compatibility with the Oshkosh snow blower, as annotated below (M-B Companies or Equal).
