# DDSP Material Handling Service Lift

- Agency: DEPT OF DEFENSE

- Posted: 2026-04-01 13:09:14.795-04
- Solicitation Number: SP330026Q0094

## Summary

Amendment 0002 posted adding two clauses, this must be signed and returned to be considered. Amendment 0001 posted clarifying the part number requested. This Request for Quotation (RFQ) is being issued to establish a firm-fixed-price (FFP) purchase to provide a fork truck/material handling service lift IAW the salient characteristics listed on page 3 for DLA Distribution Susquehanna (DDSP). The Government intends to award one (1) purchase order as a result of this RFQ and will be issued on a Standard Form (SF) 1449. This acquisition is not a Defense Priorities and Allocations System (DPAS) rated procurement. This acquisition is being solicited as a Small Business Set-Aside (SBSA). The North American IndustryClassification System (NAICS) code for this project is 333921 and the size standard is 1,000 employees. TheProduct or Service Code (PSC) for this acquisition is 4910.
