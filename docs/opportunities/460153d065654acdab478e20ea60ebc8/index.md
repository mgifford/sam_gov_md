# Creaform 3D Scanner Controller

- Agency: DEPT OF DEFENSE
- Type: Combined Synopsis/Solicitation
- Posted: 2026-03-23 07:57:11.757-04
- Solicitation Number: M6700426Q0019

## Summary

Marine Depot Maintenance Command has a requirement for the Creaform 3D Scanner Controller. This specific Scanner controller is required in support of MDMC's Creaform 3D scanning efforts in the Advanced Manufacturing Center of Excellence. This controller will allow for maximum optimization of MDMC's 3D scanning equipment and will improve the performance of all 3D scanning systems. This upgraded platinum controller will allow for larger scanning sessions with 48GB GPU, manage multiple scanning systems with expanded 4 port connectivity, increase processing speed with liquid cooled highflow radiator, and maintain large data sets with 4TB Primary and 16TB Backup capacity. See attached document "D1_Request for Quote" for requirment details.
