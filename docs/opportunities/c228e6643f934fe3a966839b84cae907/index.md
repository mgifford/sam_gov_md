# Joint Interoperability of Tactical Command and Control Systems (JINTACCS) Software Development and Sustainment (SDS)

## Summary

This is a SOURCES SOUGHT NOTICE to determine the availability and technical capability of small businesses (including the following subsets: Small Disadvantaged Businesses, Certified 8(a), Service-Disabled Veteran-Owned Small Businesses, HUBZone Small Businesses and Woman-Owned Small Businesses) to provide the required products and/or services. The DISA / Emerging Technologies organization is seeking information from potential sources for software development and sustainment of two systems summarized below. Tactical Data Link Configuration Management Tool (TCMT) TCMT is a Configuration Management (CM) application that was developed under a previous contract and to date can generate two documents 1) MIL-STD-6017 Variable Message Format (VMF) and 2) Joint Multi-Tactical Data Link (TDL) Operating Procedures (JMTOP) CJCSM 6120.01. It was developed to perform CM of TDL MIL-STDs using a systematic approach that ensures consistency of processing and tracks the impacts to previously released products. The TCMT is a replacement for the legacy software TDL Document System (TDS). TCMT is composed of both Commercial-Off-The-Shelf (COTS) products and custom code. United States Message Text Format (USMTF) – Web Application (WA) The USMTF-WA is an application in sustainment that supports the CM of MIL-STD-6040. Specifically, the USMTF-WA is used to update the Message Text Format (MTF) message catalog. The catalog contains over 300 message formats. The messages are the official source for DoD entities to convey information in a non-real time format. USMTF messages are made of Sets and Fields. Sets and Fields may be used in any of the 300+ MTF messages. The anticipated Period of Performance is Sep 2026 – Sep 2030. The anticipated Place of Performance is both Fort George G. Meade (FGGM) and the contractor’s location (within 50 miles of FGGM). The attached Sources Sought Notice and draft Performance Work Statement (PWS) provide additional details for responding to this inquiry. Please be advised, the draft PWS is marked as Controlled Unclassified Information (CUI). Interested vendors must formally request access in order to download and review the document.
