# INTENT TO SOLE SOURCE - MTS-3060A SmartCan

- Agency: DEPT OF DEFENSE
- Type: Special Notice
- Posted: 2026-03-17 09:38:08.878-04

## Summary

Spanhdahlem Air Base (Germany) has a requirement for one (1) MTS-3060A SmartCan and intends to award this requirement on a sole source Basis to Marvin Test Solutions Inc. The SmartCan Armament Tester Kit is an advanced, high-performance, battery-operated, handheld test set that can be used to support a broad range of field test applications and support test and troubleshooting of electronic systems used in harsh environments. The SmartCan Armament Tester Kit is extremely rugged and field-qualified. This item supports AIM-9 & AIM-120 on LAU-129, LAU-118 Preload, HARM Preload, BRU-57, 1760 Pylon F-16 Gun, TER-9 preload, TER-9 Bombs/Rockets (one at a time), MAU-12, Sgt. Fletcher Fuel Pylon
