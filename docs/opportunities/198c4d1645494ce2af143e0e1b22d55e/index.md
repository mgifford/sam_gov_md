# DCNG Leadership Coins_Amendment

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-03-10 10:12:26.583-04
- Solicitation Number: W912R1-26-Q-A005

## Summary

RFQ is amended to remove requirements for MAC coin, NGB-LSO coin, and MP BN coin. The total number of coin types are reduced from nine (9) to six (6). The DCNG has a requirement for nine (9) types of Leadership Coins of varying quantities for each type as part of an awards program. See attached SOO for design concepts. Quotes must include design proofs of each coin type for Government review. See attached RFQ for additional instructions.
