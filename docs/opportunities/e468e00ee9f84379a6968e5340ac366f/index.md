# Intrusion Detection System (IDS) located at the Aberdeen Federal Building

- Agency: HOMELAND SECURITY, DEPARTMENT OF
- Type: Combined Synopsis/Solicitation
- Posted: 2026-04-06 16:21:40.782-04
- Solicitation Number: 70RFPW26QW8000006

## Summary

Request for Quotation (RFQ) 70RFPW26QW8000006 Installation of Intrusion Detection System (IDS), Aberdeen Federal Building in Aberdeen SD 57401. The U.S. Department of Homeland Security (DHS), Federal Protective Service (FPS) has a requirement for the purchase and installation of a new standalone, one (1) Alarm Panel, new backup power supply, forty (40) new magnetic door contacts on all entry pedestrian and overhead doors, twenty-one (21) new glass break sensors located to monitor all first floor and below-grade windows, two (2) keypads, one (1) wall mounted siren and appurtenances necessary for a fully functional Intrusion Detection System (IDS) located at the Aberdeen Federal Building, 115 4th Avenue SE, Aberdeen SD 57401. This acquisition is being solicited as a 100% small business set aside. Services shall be provided via a firm-fixed price contract. The North American Industry Classification System (NAICS) code is 561621 and the size standard is $25 million. The source selection process for this acquisition is best value. The submission requirements and evaluation criteria will be defined in the Request for Quotation (RFQ). The Period of Performance for the installation is no later than 90 business days after the receipt of the Notice to Proceed. The contractor is required to register with the System for Award Management (SAM) at https://www.sam.gov. No contract award shall be made to any contractor that does not have an active registration in SAM. See the attached file titled “RFQ 70RFPW26QW8000006” for the Request for Quote documentation.
