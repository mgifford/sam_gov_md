# 400G Optical Network Upgrade Installation and Commissioning

- Agency: DEPT OF DEFENSE
- Type: Solicitation
- Posted: 2026-04-17 11:35:12.107-04
- Solicitation Number: FA480026Q0052

## Summary

The 633rd Communications Squadron (633 CS) at Langley Air Force Base, Virginia, has a need to upgrade its core optical transport network. This project requires professional services to install and commission Government-Furnished Equipment (GFE), specifically Cisco Network Convergence System 2006 Series (NCS2K) Dense Wavelength Division Multiplexing (DWDM) nodes and 400G Transponder Modules, into the existing network infrastructure to complete this critical modernization effort. The scope includes all tasks from initial site preparation through final system testing and delivery of documentation. The outcome shall be a fully operational, secure, and integrated 400Gbps optical network capability across four (4) specified locations on Langley AFB, integrated into the existing eleven (11)-site network environment. The resulting network shall be agile and scalable to meet critical mission demands. Please ensure to follow posting for updates and/or amendments to the solicitation.
