# RFQ for On-site Secure Document Shredding Services

- Agency: SOCIAL SECURITY ADMINISTRATION
- Type: Solicitation
- Posted: 2026-04-23 08:25:44.378-04
- Solicitation Number: 28321326QR0200002

## Summary

Request for Quotation for on-site secure document shredding services for Social Security Administration (SSA) Field and Hearing offices in the Northeast Region: New York, New Jersey, Maine, New Hampshire, Vermont, Connecticut, Pennsylvania, Rhode Island, Massachusetts, Maryland, Virginia, Delaware, the District of Columbia, and West Virginia to include a Base and 4 Option Years. The purpose of this amendment is to extend the Request for Quotation submission due date until April 29, 2026, for on-site secure document shredding services for Social Security Administration (SSA).
