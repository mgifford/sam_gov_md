# Library Document Archive (Subscription)

- Agency: EXPORT-IMPORT BANK OF THE US
- Type: Combined Synopsis/Solicitation
- Posted: 2026-03-11 16:01:23.862-04
- Solicitation Number: 83310126Q0018

## Summary

The Export-Import Bank of the U.S. has a requirement for a subscription that replaces DB Textworks, (Lucidea) an archives catalog that is going out-of-support, and replaced by a new platform. An archives catalog is a structured database that stores descriptive records of archival materials. The requirement includes the new platform, along with migration from the current one that will no longer be supported. A base year period of one year, with four option periods (one year each, if exercised) are intended. The requirement is set aside for small businesses.
