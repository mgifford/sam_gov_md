## Summary

This Request for Quotation (RFQ) is being issued to establish a firm-fixed-price (FFP) purchase order for Vest Carriers at DLA Distribution Susquehanna, PA (DDSP). The Government intends to award one (1) contract as a result of this RFQ and will be issued on a Standard Form (SF) 1449. This acquisition is not a Defense Priorities and Allocations System (DPAS) rated procurement. This acquisition is being solicited as a 100% Small Business Set-Aside. The North American IndustryClassification System (NAICS) code for this project is 339113 and the size standard is 800employees. The Product or Service Code (PSC) for this acquisition is 8465. 04/21/2026: Amendment 0001 is hereby incorporated and includes all vendor sumitted questions and answers.
