## Summary

ASH CREEK PIPELINE The Department of Interior, Bureau of Land Management (BLM), Montana State Office is soliciting quotes for the construction of an 11,380¿foot stock water pipeline with four (4) hydrants and integration of a storage tank. This project has an existing water well and a BLM provided storage tank. The contractor will furnish all labor, equipment, supplies, and materials necessary to install the pipeline, connect it to the existing well, and the BLM storage tank. The project located in Prairie County, Montana. The work location is shown in the drawings.
