# Demolition Services for LaGuardia Airport Runway #22 Localizer Shelter, Flushing, New York

- Agency: TRANSPORTATION, DEPARTMENT OF
- Type: Sources Sought
- Posted: 2026-04-21 10:54:07.088-04
- Solicitation Number: FAA-AAQ560-00847

## Summary

The FAA is currently planning a requirement for demolition services for LaGuardia Airport, Runway #22 Localizer Shelter at Flushing, NY. This is a request for information (RFI)/sources sought only at this time. See two draft attachments (1- SOW/specifications; 2- Drawings) for more description of the requirement. The following is projected procurement timeline for major procurement events: - Solicitation package expected to be released/distributed between June 1 - July 31, 2026. - Evaluation/Source Selection followed by contract award expected between September 1 - October 31, 2026. - Target start date for work performance is between January 2 - February 28, 2027 with a sixty (60) calendar days period of performance (POP) from effective date of notice to proceed (NTP) memorandum to be coordinated and performed in phases. Interested contractors are to provide capability statement, relevant business information to include Unique Entity Identification (UEID) number and point of contact(s) information. The solicitation package will be sent directly to small business contractors who demonstrate interest and capability in timely response to this RFI/sources sought. Please limit response to 1-5 pages. Provide response by email to: Michael.Eadon@FAA.gov.
